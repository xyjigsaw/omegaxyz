你还在用XX浏览器吗~

ideas程序人生常识ideas
第一次写软文，哈哈哈。其实是分享几张图片吧。 在我大天朝，大多数人使用的IE浏览器（事业单位用得比较多）、谷歌浏览器（Chrome）、360安全（liu mang）浏览器、360极速浏览器（其实就是用得谷歌浏览器的内核）、Safari等，还有一部分人用的FireFox。

IE浏览器

为什么说IE浏览器是垃圾呢？ 对Web开发者来说，兼容IE就像眼中刺 web设计者一天的生活：你花费数个小时做出页面来，然后在Google Chrome、Firefox、Safari 甚至是 Opera 里测试，结果都非常完美，太棒了！ 现在你打开IE，发现自己的作品看起来像是扔到搅拌机搅拌完了一样。于是你一边骂娘一边试着调整页面使其在IE6和IE7里看起来不那么奇怪，而花费的时间将数倍于之前开发的总时间。 Geek 们使用 IE 纯粹是被逼无奈火狐“吃”IE 全世界的 Geek 都被逼着在工作时使用IE，尽管有很多更好的浏览器，因为企业应用而必须兼容IE，必须使得网站在IE下运行正常，而我们又没有办法劝服所有人都去换一个更好的浏览器。 总之就是一句话，对于一些网页代码必须要专门对IE浏览器进行优化，还要考虑IE浏览器的版本号问题和SSL证书，更重要的一点是IE浏览器太吃内存大大降低了电脑速度。 当然了，现在微软已经放弃了IE浏览器了，他们在win10系统上预装了最新的Microsoft Edge浏览器，不过图标有点像IE浏览器。

谷歌Chrome浏览器

世界上活跃用户最多的浏览器非谷歌浏览器莫属了（当然首先大多数人得先用IE浏览器下载谷歌浏览器） 为什么谷歌浏览器好呢，讲个冷笑话： 给妹子重装了系统，她装了360浏览器，很快地，我发现桌面上多了360安全卫士，360杀毒，360手机卫士，360安全导航，360...然后又重装了系统，这次明令禁止安360浏览器，于是她装了猎豹浏览器。很快地，我发现桌面上多了金山卫士，悟空杀毒，金山手机管家。。。。。然后又重装了系统，这次明令禁止安装360，猎豹，后来她装了QQ浏览器，我没发现多了什么东西，原来什么QQ管家，QQ音乐，QQ输入法早就被安装好了。。。后来又重装了系统，这次我让她只能用chrome，世界清静了。妹子从此再也没找过我。 哈哈哈这样就知道了，国产软件大多数都是流氓软件，360和腾讯尤为突出。如果把 Chrome 比作安卓，那国产的就是装了各种助手、管家、大师、安全桌面的安卓。
