利用JavaScript批量删除QQ空间说说（只需一个浏览器）

高级语言技术域JavaScript&HtmlJavaScript
每个人都有历史，记录说说是保存瞬间思想的最好方式，但时间久了，我们发现从前的灵光闪现是多么的好笑，于是我们开始删说说，可是说说那么多，哎算了重注册一个QQ号吧。 别这样，这里教你一种快速删说说的方法，无需下载任何软件（但还是推荐使用360、QQ或谷歌浏览器比较好），只需要在电脑上打开QQ空间。按下键盘上的F12键（注意是功能键哦！！！），浏览器会自动打开一个界面后转到Console（控制台）选项卡，粘贴代码按下Enter键进行删除说说。需要保持在这个页面不要切换到其他网页。 这样这一页的说说就自动删除了哦！！！ JavaScript删除说说代码： 
var delay = 1000;

function del() {
    document.querySelector('.app_canvas_frame').contentDocument.querySelector('.del_btn').click();
    setTimeout("yes()", delay);
}

function yes() {
    document.querySelector('.qz_dialog_layer_btn').click();
    setTimeout("del()", delay);
}

del();
