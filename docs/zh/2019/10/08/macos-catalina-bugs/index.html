macOS Catalina常见问题汇总

技术域操作系统mac
作为一个伪程序员，我来说说macOS Catalina的一些软件兼容性和注意点。 macOS Catalina 正式版无法使用的APP
  • 有道词典闪退
  • brew版本过低的加载错误
  • VScode头文件有波浪线需要重新配置
报错原因： xcrun: error: invalid active developer path (/Library/Developer/CommandLineTools), missing 更新Xcode命令行工具 在命令行输入下面语句即可 
xcode-select --install
之后会跳出下面的界面，升级即可
  • App Store的LaTeX帮手Daum Equation Editor 由于是32位则无法使用了
  • 同步推32位无法使用了
  • 系统自带的Python由于是2.7的，将在不支持未来的版本（官方diss）
  • 控制台Python3首次加载速度特别慢
  • 控制台交互式shell将从bash改为zsh
输入以下语句，将您的账户升级到zsh 
chsh -s /bin/zsh
zsh在一定程度上是与Bash兼容的。但是符合bash规范的脚本无法保证被Zsh解释器正确执行。 当然，如果你想回到bash，那么直接在偏好设置中修改Shell的打开方式为下图即可
