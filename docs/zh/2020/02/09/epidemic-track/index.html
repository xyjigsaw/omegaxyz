传染病扩散可视化搜集

技术域其它元胞自动机

已知信息

往年春节期间人口流动数据，包括（高铁、飞机、高速等）
今年人口流动数据（高速路况、航班等）
疫情数据（包括确诊，疑似病例的居住地、发病地、接触史）
API：
①https://lab.isaaclin.cn/nCoV/
②百度地图API

可用模型

  • 污染聚散模型，如空间分析的BW模型，分析传播区域的发展空间态势
  • 疫情聚散模型，采用Moran'I系数计算疫区空间相关关系，分析疫情的发展是呈现空间聚集还是扩散
  • 因子影响模型，SAR，MAR等空间结构回归模型，识别空间传播的影响因子
  • 元胞自动机+传染病空间动力模型
