Linux非root用户部署jdk等命令

高级语言技术域JAVA操作系统JAVALinux

步骤

1. 从Oracle官网下载jdk的Linux安装包至用户文件夹。可以使用命令下载也可ftp传输。 2. 将jdk压缩包解压到指定文件夹，并记住文件夹的路径。 
tar -zxvf jdk-linux-x64.tar.gz
3. 打开.bash_profile或者.profile文件 
vi .bash_profile
4. 在文件末尾添加环境变量 
JAVA_HOME="${HOME}/jdk1.8.0"
CLASSPATH=$JAVA_HOME/lib:$JAVA_HOME/jre/lib 
PATH=$PATH:$JAVA_HOME/bin:$JAVA_HOME/jre/bin 
export PATH CLASSPATH JAVA_HOME
其中${HOME}是你的用户文件夹，/后面为jdk解压后文件夹的相对路径。（只需要修改第一行） 5. source命令重新执行刚修改的初始化文件，使之立即生效 
source ~/.bash_profile
6. 重启终端
同样其他命令也使用类似的操作，如npm的node.js 下载后，添加环境变量 
export PATH="${PATH}:${HOME}/nodejs/bin"
即可。
