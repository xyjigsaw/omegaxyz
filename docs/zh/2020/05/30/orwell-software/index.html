奥威尔：老大哥在看着你-软件公司十诫

技术域ideas随笔程序人生软件工程ideas软件工程
奥威尔：老大哥在看着你 Orwell: Keeping an Eye On You是一款反乌托邦政府监控模拟器。其中的奥威尔（Orwell）是一项新的政府安全计划，有权对“国家”中每个人的在线状态进行调查。它可以监视所有个人通信并访问任何计算机。为了保护公民的隐私，人类研究人员检查了奥威尔发现的数据，并决定应将哪些信息传递给安全部队，以及将哪些信息拒绝。 其中罗森软件公司提到的十诫很有意思。全文如下：

十诫

罗森科技相信卓越的品质和绩效与明晰的规则密不可分。所以我们每一天要以十诫来指导自己的工作:

1.所有愿景都有价值。

你有没有想过未来应该是这样或那样?某个问题应该如何解决?你当然想过。不要浪费这些想法。它们如同纯金一样珍贵。大胆追求心中所想。

2.软件可以解决所有问题。

没错。我们生活在信息就是一切的时代里。我们可以将大量复杂的事实简化为比特和字节。我们运行着精心设计的算法，在问题迎刃而解的时候对算法结果赞叹不已。

3.拥抱客户的愿景。

我们认为尊贵的客户最了解自己的核心需求。因此，应该遵循他们的愿景来努力实现优秀设计。

4.开放的思想和透明度至关重要。

本原则与第三项原则密切相关，保持思想开放和所有流程透明的重要性不容小觑。切勿疏忽，切勿抗拒。切勿含糊其辞。就这么简单。

5.始终与客户保持联系。

谁说“无知就是力量”，谁就大错特错。不了解客户的需求就没有力量。即使问题多出2、3倍，也要大胆提问。

6.同步交流优于非同步交流。

不要用电子邮件之类的非同步交流来浪费客户和自己的时间来。直接联系他人必定更有效率、更令人感觉亲切，这也能给你直接讨论和获得反馈的机会。所以我们根本没有设置电子邮件地址。

7.思考自己的责任。

我们的工作不仅对客户和自己有重要意义，而且毫不夸张地说，还改变了整个世界。你们都要将这件事铭记于心。我们的力量如此巨大，切勿掉以轻心。

8.亡羊补牢，吸取教训。

错误也有健康之处:只要能发现错误、加以纠正，它们就能如身体净化一般将你的大错- - -除净。所以我们才会让专业的审核工作组审核每一段代码。

9.工作在当下。

人人都喜欢讨论可能或本应发生的事情。铭记过去固然重要，放眼未来当然更有意义，但当下才是最重要的时刻，因为这才是必须完成所有工作的阶段。

10.你可以做得更好。

无论你能做什么，你都可以做得更好。要不断学习和提升自己。永不停止工作，力求完美。 1984 更多内容访问 [omegaxyz.com](https://omegaxyz.com/) 网站所有代码采用Apache 2.0授权 网站文章采用知识共享许可协议BY-NC-SA4.0授权 © 2020 • OmegaXYZ-版权所有 转载请注明出处
