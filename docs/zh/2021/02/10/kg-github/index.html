知识图谱应用Github仓库（长期更新）

技术域机器学习知识图谱自然语言处理NLP知识图谱

知识图谱可视化Demo

Github: https://github.com/xyjigsaw/Knowledge-Graph-And-Visualization-Demo 本项目支持2D，3D知识图谱可视化与查询，主要为JavaScript代码。 详细描述：https://omegaxyz.com/2020/03/03/knowledge-graph-demo/

Aminer学术社交网络数据知识图谱构建

Github: https://github.com/xyjigsaw/Aminer2KG 本项目对Aminer学者社交网络数据集进行了提取，生成了实体关系csv文件，并通过Pytorch-BigGraph对知识图谱进行表示学习。 详细描述：https://omegaxyz.com/2020/07/12/aminer-academic-social-network/

基于知识图谱的问答系统Demo

Github: https://github.com/xyjigsaw/COVID19-KBQA-DEMO 基于知识图谱的问答系统，即KBQA。实现方法是根据用户输入的自然语言问句，转化为图数据库中的关系查询，最终将数据库中的实体及关系呈现给用户。 详细描述：https://omegaxyz.com/2020/04/28/kbqa-demo/

学术访达（超大项目，有问题私信）

Github: https://github.com/xyjigsaw/AcaFinder 基于知识图谱的学术社交网络数据查询与可视化系统。

新冠COVID-19患者轨迹事理图谱

Github: https://github.com/xyjigsaw/COVID-19-traces-Knowledge-Graph 数据来自腾讯新闻的Covid-19患者跟踪数据。该项目基于Scrapy，PhantomJS和HanLP。 HanLP用于识别实体（位置，时间，动作等）并提取事件。

PDF论文转知识图谱（开发中）

Github: https://github.com/xyjigsaw/paper2kg 该项目提取PDF论文中的基本信息和事理关系生成论文知识图谱。

Linux知识图谱

Github: https://github.com/xyjigsaw/Linux-Knowledge-Graph 该项目提取Linux的基本原语，生成Linux关系图。 详细描述：https://omegaxyz.com/2020/08/04/linux-knowledge-graph/

学科领域本体关系数据与可视化

Github: https://github.com/xyjigsaw/academia 整理并分类人类社会的一级学科及二级学科，生成学科可视化网络。 详细描述：https://omegaxyz.com/2020/07/13/domain_data_visualization/
