你是如何变平庸的？

ideas随笔ideas
当你醒来第一件事就是找手机时， 当你下床后就立刻打开电脑时， 当你感到无聊便用游戏打发时间时， 当你感到寂寞就在各大社交网站寻求刺激时， 当你一遍遍刷新知乎、微博、朋友圈时， 当你沉迷游戏、小说、漫画不能自控时， 当你承受不住压力就要朝别人宣泄时， 当你感到委屈就忍不住对别人发火时， 当你愤怒不能自控向别人恶语相加时， 当你每次受到不公就想放弃时， 当你遇到困难阻碍就想退缩时， 当你自己被自己感动的不行时， 当你还在认为自己与众不同时， 当你对自己的人生事业毫无规划时， 当你离开校园依然觉得船到桥头自然直时， 当你持宠而骄、持技而傲目中无人时， 当你跟着舆论推波助澜时， 当你接受信息不懂独立思考时， 当你只为争论而争论时， 当你故意抹黑拉低别人时， 当你不能接受别人和你不一样的观点时， 当你接受不了别人的缺点时， 当你接受不了不完整的自己时， 当你以为一切都是你觉得时， 当你觉得世界会按照你想的发展时， 当你觉得人定胜天时， 当你觉得全世界会为你让路时， 当你被心灵鸡汤洗脑不再务实时， 当你为了鸡毛蒜皮的小事忙到不可开交时， 当你为了迎合领导放弃自己的思想时， 当你为了讨好说着言不由衷的话时， 当你想要和别人一样时， 当你希望别人也和你一样时， 当你为了领导、工作身不由己时， 当你有了一点小成就就沾沾自喜时， 当你为了琐事乱发脾气和人争吵时， 当你得理不饶人且咄咄逼人时， 当你极度依赖别人时， 当你无理取闹时， 当你碍于面子不肯认错时， 当你出于自尊不肯放下偏见时， 当你盲目信奉权威时， 当你去追求别人光鲜亮丽的生活时， 当你去实现虚无缥缈的价值、爱情时， 当你为了事业弃家庭、爱人于不顾时， 当你衡量一个人用金钱、工作时， 当你觉得成功就是有很多钱时， 当你为自己所作所为不停的去找理由时， 当你为自己的错误不停的找借口时， 当你不能虚心像别人学习时， 当你见不得别人比你优秀时， 当你嫉妒别人、怨天尤人时， 当你不能真心为别人祝福时， 当你议论他人是非时······
