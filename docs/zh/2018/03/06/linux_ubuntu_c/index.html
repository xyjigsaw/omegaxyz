在Linux（Ubuntu）下编写编译C语言

高级语言C&C++技术域C语言Linux
大家都知道在Windows中运行C语言很简单，打开一个IDE（VS或者codeblocks）编写代码，点击一个按钮就能运行了。 在Linux中，大家不怎么习惯用IDE，更多的开发者选择手动编辑、编译、连接、运行。 下面介绍如何进行在Linux（Ubuntu）下运行C语言。 ①打开Ubuntu，在一个空文件夹内建立一个新的空文件，并改名为.c的文件 如图test.c ②打开文件编辑C语言代码并保存 ③返回到当前文件夹，打开终端 ④接下来需要在终端编译，连接 编译代码: gcc test.c -o test 格式: gcc [C文件名] -o [C文件名无后缀] 此时会在当前文件夹内出现一个编译后的文件 ⑤在当前文件夹下运行test文件（无后缀） 运行代码: ./test 格式: ./名称 这样我们就能看到输出了。 附1：非图形界面Linux全终端运行C程序：http://blog.csdn.net/qq_36328643/article/details/52794876 附2：利用Vim编辑器编写C程序：http://www.runoob.com/linux/linux-vim.html
