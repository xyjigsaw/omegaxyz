利用GithubPage搭建静态博客

技术域Web开发webHTMLGithub
gitHub是一个面向开源及私有软件项目的托管平台，因为只支持git 作为唯一的版本库格式进行托管，故名gitHub。 gitHub于2008年4月10日正式上线，除了git代码仓库托管及基本的 Web管理界面以外，还提供了订阅、讨论组、文本渲染、在线文件编辑器、协作图谱（报表）、代码片段分享（Gist）等功能。目前，其注册用户已经超过350万，托管版本数量也是非常之多，其中不乏知名开源项目 Ruby on Rails、jQuery、python 等。 githubpage是其一个功能 ①申请一个github的账号 https://github.com/ ②首先，我们需要新建一个Repositories,Repositories就相当于一个库，存放我们的项目文件。 ③进入设置页面，找到Github page选择master branch，设置域名为自己的解析域名如en.omegaxyz.com，记录值为你的github用户名.github.io 最后把自己的HTML文件及依赖项上传到这个库中，注意要有index.html文件 请看我的两个github page做的网页：en.omegaxyz.com和movie.omegaxyz.com
