Hello, Android

技术域APP开发appAndroid
Android Studio 是一个Android集成开发工具，基于IntelliJ IDEA. 类似 Eclipse ADT，Android Studio 提供了集成的 Android 开发工具用于开发和调试。 在IDEA的基础上，Android Studio 提供 ： ①基于Gradle的构建支持 ②Android 专属的重构和快速修复 ③提示工具以捕获性能、可用性、版本兼容性等问题 ④支持ProGuard 和应用签名 ⑤基于模板的向导来生成常用的 Android 应用设计和组件 ⑥功能强大的布局编辑器，可以让你拖拉 UI 控件并进行效果预览 如何用Android生成一个Hello，world程序呢？ ①点击New Project ，输入应用名称和公司域名点击next ②选择SDK，这里我选用17年的API24，Android7.0手机平板模式（虽然奥利奥出来了） ③选择Empty Activity ④输入活动名称HelloAndriodActivity ⑤代码自动生成 最后点击启动虚拟机即可看到如下图 图片版权所有转载请注明出处
