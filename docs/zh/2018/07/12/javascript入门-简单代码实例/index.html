JavaScript入门——简单代码实例

高级语言技术域Web开发JavaScript&HtmlwebJavaScript
JavaScript，也称ECMAScript，是一种基于对象和事件驱动并具有相对安全性并广泛用于客户端网页开发的脚本语言，同时也是一种广泛用于客户端Web开发的脚本语言。 JavaScript是世界上最流行的脚本语言，因为你在电脑、手机（Android、iOS）、平板上浏览的所有的网页，以及无数基于HTML5的手机App，交互逻辑都是由JavaScript驱动的。 简单地说，JavaScript是一种运行在浏览器中的解释型的编程语言。 更多简介参考： https://omegaxyz.com/2018/06/16/javascript-introduction/ 下面给出单独的JS代码及嵌入到HTML中的JS代码用做简单实例测试，可以进入HTML测试页面测试代码。 https://omegaxyz.com/onlinetools/web-running/

①生成简单文本及HTML标签、循环

以99乘法表为例 
document.write("<table border='1' bordercolor= 'red'>");
for(var i = 1; i <= 9; i++){
  document.write('<tr>');
  for(var j = 1; j <= i; j++){
    document.write('<td>');
    document.write(j+'*'+i+'='+i*j+'</td>');
  }
  document.write('</tr>');
}
document.write('</table>');

{
var today=new Date()
var h=today.getHours()
var m=today.getMinutes()
var s=today.getSeconds()
// add a zero in front of numbers<10
m=checkTime(m)
s=checkTime(s)
document.getElementById('txt').innerHTML=h+":"+m+":"+s
t=setTimeout('startTime()',500)
}

function checkTime(i)
{
if (i<10) 
  {i="0" + i}
  return i
}
