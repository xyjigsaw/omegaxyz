Linux通过Shell进行数学运算

技术域操作系统Linux
在Bash shell环境中，可以利用let、(( ))和[ ]来执行基本的算术操作。而高级的算术操作可以使用expr和bc这两个工具。

let

let操作：let命令可以直接执行基本的算术操作，当使用let时，变量名知其那不需要再添加$.
自加操作：let no1++； let no+=6;
自减操作：let no2--; let no-=4;
自加自减

其他方法

[ ]使用和let类似，[ ]中也可以使用$；也可使用(( ))，但变量名之前要加上$；expr也可用于基本算术操作。
expr 命令读入 Expression 参数，计算它的值，然后将结果写入到标准输出。您必须对 Expression 参数应用以下规则： 用空格隔开每个项。 用 / （反斜杠） 放在 shell 特定的字符前面。 对包含空格和其他特殊字符的字符串要用引号括起来。 整数前面可以放一个一元连字符。在内部，整数被当作 32 位，双互补数。

工具bc

上述方法不可计算浮点数，精密计算工具bc可以执行浮点数运算并应用一些高级函数。bc工具还可以用来设置小数竞速，进制转换和计算平方以及平方跟。
