IntelliJ IDEA创建Java-Web项目

高级语言技术域Web开发JAVAJAVAwebjsp
eclipse和idea都能够创建Java web项目。下面介绍使用idea创建Java web项目的步骤。 需要准备的东西
  • intellij idea（包括jdk）
  • Tomcat7.0+
可选（如果需要搭建servlet服务器或者链接数据库）
  • jdbc扩展包
  • sql server或者mysql

①创建项目

②选择Java->Java EE->web application

③输入项目名称，点击finish

接下来进入到项目界面，左侧是项目的文件目录。src一般是放Java代码的，web是界面（视图）代码

④如果之前没有创建过web项目需要导入Tomcat服务器

选择Tomcat Server->local

⑤选择Tomcat地址导入到Tomcat Home和Tomcat base directory

⑥设置默认打开的浏览器和端口

⑦运行index.jsp

此时将会打开默认浏览器，OK
