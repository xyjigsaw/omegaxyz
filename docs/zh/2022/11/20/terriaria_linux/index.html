泰拉瑞亚Terriaria快速部署Linux服务器

ideas随笔LinuxGameTerraria
最近泰拉瑞亚更新频繁（每次开都要重新部署！！！），多人游戏的Linux服务器也需要同步更新，但是我记不住新版下载地址：(。记录在此。

下载地址

https://terraria.org/api/download/pc-dedicated-server/terraria-server-1447.zip 最后的1447是版本号，如果是1.4.4.8.1版本，则改为14481

部署步骤

unzip terraria-server-1447.zip
cd 1447
cd Linux

chmod +x TerrariaServer
chmod +x TerrariaServer.*
./TerrariaServer -config ./startserver.sh
