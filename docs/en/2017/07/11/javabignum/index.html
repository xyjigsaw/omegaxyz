JAVA operation of very large numbers (BigInteger class)

TechnologyJavaJava
If you want to perform very large number calculations or high-precision floating point number calculations, you can use the BigInteger class in the java.math package. They are all immutable. Note that these are numbers of arbitrary size and precision. Be sure to import java.math.*; at the beginning of the code;
import java.math.*;
public class javaLang {
	public static void main(String[] args){
		BigInteger m=new BigInteger("425287628746279647233986234525425423535345363534532463563463554");
		BigInteger n=new BigInteger("1451454524154154278278278272278272727278272121541213");
		BigInteger x=m.multiply(n);
		System.out.println(x);
		System.out.println("*********************************************");
		System.out.println(factorial(100));
	}
	
	public static BigInteger factorial(long n){
		BigInteger result=BigInteger.ONE;
		for(int i=1;i<=n;i++){
			result=result.multiply(new BigInteger(i+""));
		}
		return result;
	}
}
