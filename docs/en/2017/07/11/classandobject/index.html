Overview of JAVA objects and classes

TechnologyJavaJava
1. A class is a template for an object. It defines the properties of an object and provides constructors for creating objects as well as common methods for manipulating objects. 2. A class is also a data type. You can use it to declare object reference variables. It seems that an object is stored in the object reference variable, but in fact , it contains only a reference to the object. Strictly speaking, object reference variables and objects are different, but in most cases, Their difference is negligible. 3. Objects are instances of classes. You can use the new operator to create an object, and use the dot operator (.) to access the object through its reference variable. members of the object. 4. An instance variable or method belongs to an instance of a class. Its usage is associated with the respective instance. Static variables are used by all members of the same class shared by the instance. Static methods can be called without using an instance. 5. Each instance of a class can access the static variables and static methods of this class. However, for clarity, it's better to use "classname, variable" and. Class name, method. to call static variables and static methods. 6. Visibility modifiers specify how classes, methods, and data are accessed. A public class, method or data can be used by any For client access, private methods or data can only be accessed within this class. 7.
