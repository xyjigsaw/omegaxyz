Creation and calling of JAVA "class" array

TechnologyJavaJava
The creation and calling of JAVA "class" arrays is different compared to C++. Let’s first look at the creation of such an array-like Note: bookFeature is a class Error 1:
class bookList {
	private int bookNum=0;
	final int MAXSIZE=1001;
	public bookList(){
		bookNum=0;
		bookFeature[] List=new bookFeature[MAXSIZE];
		}
	}
Here bookFeature[] List=new bookFeature[MAXSIZE] is defined in the construction method bookList(); This is not possible. JAVA requires that each variable must be initialized. It must beConstructorAdd this line of code:
for(int i=0;i<MAXSIZE;i++){
			List[i]=new bookFeature();
Is this enough? Next look at error 2 Error 2:
class bookList {
	private int bookNum=0;
	final int MAXSIZE=1001;
	bookFeature[] List;
	public bookList(){
		bookNum=0;
		bookFeature[] List=new bookFeature[MAXSIZE];
		for(int i=0;i<MAXSIZE;i++){
			List[i]=new bookFeature();
		}
	}
BookFeature[] List is defined here first; and then in the construction method, this is not possible. The array variable here must be placed in the object. Both errors will report java.lang.NullPointerException Remember arrays are objects in JAVA! ! !The following gives the correct declaration of an array-like
import java.util.*;
class bookList {
	private int bookNum=0;
	final int MAXSIZE=1001;
	bookFeature[] List=new bookFeature[MAXSIZE];
	public bookList(){
		bookNum=0;
		for(int i=0;i<MAXSIZE;i++){
			List[i]=new bookFeature();
		}
	}
