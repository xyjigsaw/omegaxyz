Understand the connection and difference between JAVA and Android in one minute

Programming LanguagesTechnologyAPP开发JavaProgrammingJavaAndroid
Android is an open source operating system based on Linux. JAVA is an object-oriented programming language.Most applications on Android are developed using JAVA, but the Android SDK references most of the Java SDK, and a few are abandoned by the Android SDK. When developing Android, you don't need to know the underlying programming of JAVA, because many commonly used libraries have been written for a long time. Your task is to be familiar with these libraries and call them directly, which will greatly speed up the efficiency of software development. In addition, the UI framework of the Android APP is written using XML (Extensible Markup Language, a subset of the Standard Universal Markup Language, a markup language used to mark electronic documents to make them structural). Can be called via JAVA. In addition, due to mobile phone permission issues, we also need to register for the event.
