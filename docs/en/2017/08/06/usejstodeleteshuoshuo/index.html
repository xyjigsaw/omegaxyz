Use JavaScript to delete QQ Space in batches (only one browser is needed)

Programming LanguagesTechnologyJavaScript&HtmlJavaScript
Everyone has a history, and recording stories is the best way to preserve momentary thoughts. But as time went by, we discovered how funny the flashes of inspiration were in the past, so we started to delete stories, but there were so many stories, so forget it and register a new QQ account. Don’t be like this, here I’ll teach you a way to quickly delete the text without downloading any software (but it is still recommended to use 360, QQ or Google Chrome). You only need to open the QQ space on your computer. Press the F12 key on the keyboard (note that it is a function key!!!), the browser will automatically open an interface and go to the Console tab, paste the code and press the Enter key to delete it. You need to stay on this page and do not switch to other web pages.In this way, the comments on this page will be automatically deleted! ! ! JavaScript deletion code:
var delay = 1000;

function del() {
    document.querySelector('.app_canvas_frame').contentDocument.querySelector('.del_btn').click();
    setTimeout("yes()", delay);
}

function yes() {
    document.querySelector('.qz_dialog_layer_btn').click();
    setTimeout("del()", delay);
}

del();
If you haven’t learned how to do it yet, leave me a message and communicate on QQ! ! !
