Are you still using XX browser~

IdeasDeveloper Life常识Ideas
This is my first time writing a soft article, hahaha. Actually, let me share a few pictures. In my country, most people use IE browser (more commonly used in public institutions), Google Chrome (Chrome), and 360 Security（liu mang）Browser, 360 Speed ​​Browser (actually using the kernel of Google Chrome), Safari, etc., and some people use FireFox.

Internet Explorer

Why is IE browser considered garbage? For web developers, compatibility with IE is like a thorn in the side. A day in the life of a web designer: you spend several hours making a page, and then test it in Google Chrome, Firefox, Safari or even Opera, and the results are perfect, great! Now you open IE and find that your work looks like it has been thrown into a blender. So you curse while trying to adjust the page to make it look less strange in IE6 and IE7, and the time spent will be several times the total time of previous development. Geeks use IE because they are forced to use IE. Firefox "eats" IE. Geeks all over the world are forced to use IE at work. Although there are many better browsers, they must be compatible with IE because of enterprise applications. Websites must run normally under IE, and there is no way to persuade everyone to switch to a better browser. In short, in one sentence, some web page codes must be specifically optimized for the IE browser, and the version number of the IE browser and the SSL certificate must also be considered. More importantly, the IE browser consumes too much memory and greatly reduces the computer speed. Of course, Microsoft has now given up on the IE browser. They pre-installed the latest Microsoft Edge browser on the win10 system, but the icon looks a bit like the IE browser.

Google Chrome browser

The browser with the most active users in the world is none other than Google Chrome (of course, most people first use IE browser to download Google Chrome) Why is Google Chrome so good? Soon, I found that there were 360 ​​Security Guard, 360 Antivirus, 360 Mobile Guard, 360 Safe Navigation, 360... Then I reinstalled the system again. Soon, I found that Kingsoft Guard, Wukong Antivirus, and Kingsoft Mobile Manager were added to the desktop. . . . . Then she reinstalled the system. This time she was explicitly prohibited from installing 360 and Cheetah. Later she installed QQ Browser. I didn’t find anything extra.
