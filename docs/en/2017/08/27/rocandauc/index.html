Introduction to ROC curve/AUC curve and confusion matrix

TechnologyMachine LearningROC

ROC curve

The ROC (Receiver Operating Characteristic) curve is the receiver operating characteristic curve (ROC curve for short), also known as the sensitivity curve (sensitivity curve), which is used to evaluate the quality of a binary classifier.The abscissa of the ROC curve is false positive rate (FPR), and the ordinate is true positive rate (TPR).

confusion matrix

The next picture is the confusion matrix. Understand it first • False positive rate (FPR) The probability of being judged to be a positive example but not a real example • True Positive Rate (TPR) The probability that a positive example is also a true example • False Negative Rate (FNR) The probability that a negative example is not a true negative example • True Negative Rate (TNR) The probability that a negative example is also a true negative example Note that P+N in this figure is the number of all samples. Next, we will examine four points: ①(0,1): That is, FPR=0, TPR=1, which means FN (false negative)=0, and FP (false positive)=0. Wow, it classifies all samples correctly. ②(1,0), that is, FPR=1, TPR=0, the predictions at this point are all wrong. ③(0,0), that is, FPR=TPR=0, that is, FP (false positive)=TP (true positive)=0, this classifier predicts that all samples are negative samples (negative). ④(1,1), the classifier actually predicts that all samples are positive samples. Conclusion: The closer the ROC curve is to the upper left corner, the better the performance of the classifier.

AUC curve

AUC (Area Under Curve) is defined as the area under the ROC curve, and the value of this area will not be greater than 1. And since the ROC curve is generally above the straight line y=x, the value range of AUC is between 0.5 and 1. The AUC value is used as the evaluation criterion because in many cases the ROC curve cannot clearly illustrate which classifier performs better. As a numerical value, the classifier with a larger AUC performs better. Why? Because the closer the ROC curve is to the upper left corner, the larger the AUC area and the better the classifier performance. References: Kong Ming’s blog, Baidu Encyclopedia
