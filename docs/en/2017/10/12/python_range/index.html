Understanding range objects in Python (continuously updated)

高级语言技术域PythonPython
What is the function of python's built-in range() function? It returns a series of consecutively increasing integers and works like sharding to produce a list object. The range function most often appears in for loops and can be used as an index in for loops. In fact, it can also appear in any environment that requires a list of integers. In Python 3.0, the range function is an iterator. Let’s look at a piece of code first:
x = 4  # 1
for j in range(x):  # 2
    for i in range(x):  # 3
        print(i, end=' ')  # 4
        x = 2  # 5
In the second line of the Python loop, the range object has been generated as [0,1,2,3]. Therefore, regardless of whether the value of After outputting 0, the value of x becomes 2. But the range object has already been generated, so the range object in the inner loop will not be regenerated. Wait for the outer loop to end for the first time (that is, 0, 1, 2, 3 is output). The value of x is 2, so the second line generates a range object of [0,1]. Repeat this process 3 more times. The final output is [0,1,2,3,0,1,0,1,0,1].
