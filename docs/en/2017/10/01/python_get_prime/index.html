Python list comprehension to find prime numbers

Programming LanguagesTechnologyPythonPython
A simple application using list comprehensions
from math import sqrt
floor = eval(input("输入下界(>=2)："))
ceil = eval(input("输入上界："))
p = [num for num in range(floor, ceil) if 0 not in [num % d for d in range(2, int(sqrt(num) + 1))]]
print(p)
