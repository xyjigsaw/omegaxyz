Python Machine Learning House Price Prediction (Stanford University Machine Learning Course)

Programming LanguagesTechnologyMachine LearningPythonPython
Questions from the MOOC Stanford Machine Learning Course

question

·The input data has only one dimension: the area of the house ·The target data has only one dimension: the price of the house Machine learning and model prediction based on known house size and price See the data at the end of the article The data needs to be standardized X=(X-aver(sum(Xi)))/std(Xi)

Step ① Data acquisition and processing

# 导入需要用到的库
import numpy as np
import matplotlib.pyplot as plt

# 定义存储输入数据（x）和目标数据（y）的数组
x, y = [], []
# 遍历数据集，变量sample对应的正是一个个样本
for sample in open("C:\\Users\\dell\\Desktop\\house_prices.txt", 'r'):
    _x, _y = sample.split(",")
    # 将字符串数据转化为浮点数
    x.append(float(_x))
    y.append(float(_y))
# 读取完数据后，将他们转化为Numpy数组以方便进一步的处理
x, y = np.array(x), np.array(y)
# 标准化
x = (x - x.mean()) / x.std()
# 将原始数据集以散点的形式画出
plt.figure()
plt.scatter(x, y, c="g", s=6)
plt.show()
The horizontal axis is the area after normalization, and the vertical axis is the price of the house.

Step ②Select and train model

The model loss function is the squared loss function, which is the so-called Euclidean distance. The purpose here is to minimize the loss function. take advantage ofNumpyTraining and optimizationThe mathematical expression of the model is: f(x|p;n)=p0x^n+p1x^(n-1)+...+pn-1x+pn L(p;n)=0.5∑[f(x|p;n)-y]^2 
# (-2,4)这个区间上取100个点作为画图的基础
x0 = np.linspace(-2, 4, 100)
# 利用Numpy的函数定义训练并返回多项式回归模型的次数
# deg参数代表着模型参数中的n，即模型中多项式的次数
# 返回的模型能够根据输入的x（默认是x0）,返回预测的y
def get_model(deg):
    return lambda input_x=x0: np.polyval(np.polyfit(x, y ,deg), input_x)

Step ③Evaluation and display

Polynomial fitting, evaluated using n=1,4,10 No need for cross-validation because there is too little data
# 根据参数n、输入的x,y返回相对应的损失
def get_cost(deg, input_x,input_y):
    return 0.5 * ((get_model(deg)(input_x) - input_y) ** 2).sum()
# 定义测试函数集并根据它进行各种实验
test_set = (1, 4, 10)
for d in test_set:
    # 输出损失
    print(get_cost(d, x, y))
# 画出相应的图像
plt.scatter(x, y, c="g", s=20)
for d in test_set:
    plt.plot(x0, get_model(d)(), label="degree = {}".format(d))
# 将横轴和纵轴的范围分别限制在(-2,4)和(10^5,10^6)
plt.xlim(-2, 4)
plt.ylim(1e5, 1e6)
# 调用legend方法使曲线对应的label正确显示
plt.legend()
plt.show()
Overfitting occurs when n=4 and 10, so n=1 is a better prediction model.
