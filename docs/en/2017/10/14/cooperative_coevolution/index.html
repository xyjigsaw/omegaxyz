Overview of cooperative coevolution algorithm (Cooperative Coevolution)

TechnologyMachine LearningEvolutionary Algorithms多目标优化算法协同进化
Cooperative Coevolution is an effective method for solving large-scale optimization algorithms. Break large-scale problems into sets of smaller sub-problems. The key to cooperative coevolution is the decomposition strategy.   Classification of decomposition strategies: ① Random decomposition: The order of genes is randomly selected, but the user has to decide the number of groups and the size of the groups. ②Perturbation: Use several methods to perturb the decision variables to try to group the variables. ③Model construction: The probability model based on the number of individuals s is updated iteratively during the evolution process.   The following are papers embodying different decomposition strategies of the CC algorithm: Liu, X. Yao, Q. Zhao, and T. Higuchi, “Scaling up fast evolutionary programming with cooperative coevolution,” inProc. IEEE Congr. Evol. Comput., 2001, pp. 1101–1108. (This is the first cooperative co-evolutionary algorithm used to solve the 1000-dimensional benchmark problem) van den Bergh and A. P. Engelbrecht, “A cooperative approach to particle swarm optimization,"IEEE Trans. Evol. Comput., vol. 8, no. 3,225–239, Jun. 2004. (This is the first application of the PSO algorithm to divide an n-dimensional problem into a k-s-dimensional problem. It has not been tested on large-scale problems)   Shi, H. Teng, and Z. Li, “Cooperative co-evolutionary differential evolution for function optimization," inProc. Int. Conf. Natural Comput., 2005, pp. 1080–1088. (When used in the differential evolution algorithm, the decision variable is divided into two sub-components of the same size, which is not suitable for high dimensions) Yang, K. Tang, and X. Yao, “Large scale evolutionary optimization using cooperative coevolution,”Inf. Sci., vol. 178, pp. 2986–2999, Aug. 2008. (Divide the problem into k-s dimensional problems, use random grouping, and achieve good results on 1000 dimensions) N. Omidvar, X. Li, Z. Yang, and X. Yao, “Cooperative coevolution for large scale optimization through more frequent random grouping," inProc. IEEE Congr. Evol. Comput., Jul. 2010,1754–1761. (Modified CPSO random grouping scheme, reaching 2000 dimensions)   N. Omidvar, X. Li, and X. Yao, “Cooperative co-evolution with delta grouping for large scale non-separable function optimization," inProc. IEEE Congr. Evol. Comput., Jul. 2010, pp. 1762–1769. (Incremental grouping strategy, low performance when the objective function has non-molecular components)   Yang, K. Tang, and X. Yao, “Multilevel cooperative coevolution for large scale optimization," inProc. IEEE Congr. Evol. Comput., Jun. 2008, pp. 1663–1670. (MLCC algorithm, a set of different s values (sizes of subcomponents) are provided to the algorithm, which has many shortcomings)In addition, there are many papers that use CC algorithms for large-scale optimization. Therefore, to apply the CC algorithm, the key is to propose a decomposition strategy.
