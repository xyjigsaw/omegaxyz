wxpython displays static text

高级语言技术域PythonPythonwxpythonGUI
As a GUI graphical interface module, wx must display text. Let’s display the static text first.wx.StaticText in wxpython can display static text.  

wx.StaticText constructor parameters

parent: Parent widget. id: identifier. Use -1 to automatically create a unique ID. label: The text you want to display in the static control. pos : A wx.Point or a Python tuple that is the position of the widget. size : A wx.Size or a Python tuple that is the size of the widget. style: style tag. name: The name of the object, used for search purposes. Next we discuss style tags in more detail.

Some styles specific to wx.StaticText.

wx.ALIGN_CENTER: The static text is located in the center of the static text control. wx.ALIGN_LEFT : Text is left aligned in the widget. This is the default style. wx.ALIGN_RIGHT: Text is right-aligned in the widget. wx.ST_NO_AUTORESIZE: If this style is used, the static text control will not resize itself after using SetLabel() to change the text. You should combine this with a centered or right-aligned control to maintain alignment. The wx.StaticText control overrides SetLabel() to adjust itself to the new text unless the wx.ST_NO_AUTORESIZE style is set. When creating a single line of static text that is centered or right-aligned, you should explicitly set the control's dimensions in the constructor. Specify a size to prevent wxPython from automatically resizing the control. The default size of wxPython is a rectangle that just contains the text, so alignment is not necessary. To dynamically change the text in a widget in a program without changing the size of the widget, set the wx.ST_NO_AUTORESIZE style. This prevents the widget from automatically resizing to fit the text after the text is reset. If the static text is in a dynamic layout, changing its size may cause other widgets on the screen to move, which is disruptive to the user. Code:
import wx


class StaticTextFrame(wx.Frame):
    def __init__(self):
        wx.Frame.__init__(self, None, -1, 'Static Text Example',
                          size=(400, 300))
        panel = wx.Panel(self, -1)

        # 这是一个基本的静态文本
        wx.StaticText(panel, -1, "This is an example of static text",
                      (100, 10))

        # 指定了前景色和背景色的静态文本
        rev = wx.StaticText(panel, -1, "Static Text With Reversed Colors",
                            (100, 30))
        rev.SetForegroundColour('white')
        rev.SetBackgroundColour('black')

        # 指定居中对齐的的静态文本
        center = wx.StaticText(panel, -1, "align center", (100, 50),
                               (160, -1), wx.ALIGN_CENTER)
        center.SetForegroundColour('white')
        center.SetBackgroundColour('black')

        # 指定右对齐的静态文本
        right = wx.StaticText(panel, -1, "align right", (100, 70),
                              (160, -1), wx.ALIGN_RIGHT)
        right.SetForegroundColour('white')
        right.SetBackgroundColour('black')

        # 指定新字体的静态文本
        str = "You can also change the font."
        text = wx.StaticText(panel, -1, str, (20, 100))
        font = wx.Font(18, wx.DECORATIVE, wx.ITALIC, wx.NORMAL)
        text.SetFont(font)

        # 显示多行文本
        wx.StaticText(panel, -1, "Your text\ncan be split\n"
                                 "over multiple lines\n\neven blank ones", (20, 150))

        # 显示对齐的多行文本
        wx.StaticText(panel, -1, "Multi-line text\ncan also\n"
                                 "be right aligned\n\neven with a blank", (220, 150),
                      style=wx.ALIGN_RIGHT)


if __name__ == '__main__':
    app = wx.PySimpleApp()
    frame = StaticTextFrame()
    frame.Show()
app.MainLoop()
