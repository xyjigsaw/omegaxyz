Overview of glue language (taking Python as an example)

Programming LanguagesTechnologyPythonPython

What is glue language?

Glue language (a programming language used to connect software components, usually a scripting language.
glue languageGlue language is a programming language (usually a scripting language) used to connect software components.
Example of glue language:
Shell scripts Python Ruby Lua Tcl Perl PHP See also: Category:Scripting languages
Glue language (uses OS commands, interfaces, DLLs, shared libraries, services, objects, etc.)
Glue language: Use input and output commands, interfaces, dynamic link libraries, static link libraries, services, objects, etc.

Among the internal mechanisms of Python are:

1. The standard version of python is compiled in C, also known as Cpython. In addition, there are some other python implementations.
2.java implementation version. This interpreter is written entirely in java and is called Jython. Although there is a difference from the standard version, it is very small and the startup environment is exactly the same. Advantages of Jython: (1) As long as there is a virtual machine, you can run Jython (2) Have the ability to access java packages and class libraries (3) Provides a script engine for the java development environment (4) Encourage the development of Python to Java 3..net implementation. Completed in C# language, called IronPython. Similar to jython.

example:

# file: say.py 
import sys
txt = sys.argv[1]
print "Hello %s!"%(txt)
Then I can type directly into the terminal on Linux or Mac:python say.py WorldThe output will be:Hello World!This ensures intuitive and programmability in terminal calls. In this way, I don't have to call python in other python languages, but can directly connect to other handlers on the shell. thenfile: say2.py

significance:

On the terminal, I can combine it with other programs at will as an intermediate processing module. Moreover, it can be so simple and convenient. You only need to cat some simple scripts to understand what they mean immediately. An expression is one line of code. Code blocks have consistent and standard indentation, standard naming methods, concise language, and support for comments...
  • To access or call other programs, it is very simple. Just input the specified data here with a command, and then splice it how you want;
  • It's okay to process it periodically, output it to a file, and wait for other programs to call it.
On the system script:
  • It has a lot more math and string processing capabilities than SHELL, a lot more useful iteration methods, and a lot more native data structures;
  • It is much more readable than PERL.
