Win10 system disk folder cleaning strategy

TechnologyWindows

Which files can be deleted from C drive in Win10 system?

1. All files in the C:\Windows\Temp\ folder (temporary system files);

2. C:\Windows\ServicePackFiles (the backup file after upgrading sp1 or sp2);

3. C:\Windows\Driver Cache\i386\driver.cab compressed file (driver backup file);

4. All files in the C:\Windows\SoftwareDistribution\download folder;

5. All files in the C:\Windows\Prefetch folder (system prefetch files);

6. Hidden files starting and ending with $ in the C:\Windows folder (temporary files after installing system patches)

7. .log files starting with KB such as KB873339.log, etc. These are log files after installing system patches;

8. All files in the C:\Windows\system32\dllcache folder (dynamic link library file backup);

9. All files in the C:\User\username\Cookies\ folder (keep the index file);

10. All files in the C:\User\username\Local Settings\Temp\ folder (all user temporary files);

11. All files in the C:\User\username\LocalSettings\TemporaryInternet Files\ folder (temporary Internet files);

13. All files in the C:\User\username\Local Settings\History\ folder (history of Internet access);

14. All files in the C:\User\username\Recent\ folder (shortcuts to recently browsed and edited files).

The above is an introduction to some paths about which files can be deleted from the C drive of Win10 system. The User in the path is the same as the Documents and Settings in previous operating systems, and even Documents and Settings actually points to the Users folder.

In addition, you can right-click the file to compress this drive to save disk space.
