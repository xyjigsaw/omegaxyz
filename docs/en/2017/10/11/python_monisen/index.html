Python finds the Nth Meunissen number

Programming LanguagesTechnologyPythonPython
Find the nth Monessen number. The code format is as follows: def prime(num): ... def monisen(no): ... ... return xxx print(monisen(int(input()))) #You don’t need to enter it yourself here, just write such a statement, mainly completing the monisen() function (4 points)
Classic programming problem: find the nth Monessen number. P is a prime number and M is also a prime number, and satisfies the equation M=2**P-1, then M is called the Mounissen number. For example, P=5, M=2**P-1=31, 5 and 31 are both prime numbers, so 31 is the Mounissen number. Input format: follow the prompts and use the input() function to input Output format: int type Input example: 4 Output sample: 127
Time limit: 500msMemory limit: 32000kb
python3 code:
from math import sqrt


def prime(num):
    if 0 not in [num % d for d in range(2, int(sqrt(num))+1)]:
        return 1
    else:
        return 0


def monisen(no):
    flag = 2
    x = 1
    a = []
    a.append(0)
    while x - 1 != no:
        if prime(flag) == 1:
            temp = 2**flag-1
            if prime(temp) == 1:
                a.append(temp)
                x += 1
        flag += 1
    return a[no]

print(monisen(int(input())))
