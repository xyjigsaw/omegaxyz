3. Traditional C bit operations:

1.Basic operations:Use an unsigned int variable as a bit container.2. Operator:| Bitwise OR operator: result=exp1|exp2; when at least one of the corresponding bits in exp1 and exp2 is 1, the corresponding bit in result is 1, otherwise it is 0. & Bitwise AND operator:: result=exp1&exp2; When the corresponding bits in exp1 and exp2 are all 1, the corresponding bit in result is 1, otherwise it is 0. ^ Bitwise XOR operator: result=exp1^exp2; when the corresponding bits in exp1 and exp2 are different, the corresponding bit in result is 1, otherwise it is 0. ~ Inversion operator: Inverts all bits in the bit container, changing 1 to 0 and 0 to 1. << Bitwise left shift operator: exp<<n, shifts all the bits in the container to the left by n bits, and fills the vacated bits with 0. >> Bitwise right shift operator: exp>>n, shifts all the bits in the container to the right by n bits, and fills the vacated bits with 0. |=,&=,^= correspond to the compound operators of the three operators |&^ respectively.3. Common operationsHere we assume that there is an unsigned int variable of result used to store the scores of 32 students (0 and 1 are used for pass and fail respectively), so that the result has 33 bits (the result counts from right to left, starting from 0, and in this example 0 bits are wasted). (a) Set bit 27 to pass (set to 1) and leave the other bits unchanged: result|=(1<<27) //Any bitwise OR operation with 1 will result in a value of 1, and a bitwise AND operation with 0 will result in the same value. (b) Set bit 27 to fail (set to 0). result&=~(1<<27) //If any bit value is ANDed with 0, its value will be 0, and if it is ANDed with 1, its value will remain unchanged. (c) Invert the value of bit 27. result^=(1<<27) //If any bit value is XORed with 1, its value will be 1, and if it is XORed with 0, its value will remain unchanged.
