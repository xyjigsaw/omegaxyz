Overview of a faster and more accurate differential grouping large-scale black-box optimization algorithm

TechnologyMachine LearningEvolutionary Algorithms多目标优化算法
DG2 has two main parts ① Generate the original interaction relationship structure matrix Contains the threshold value |Δ(1) − Δ(2)| for each pair of variables ② Find a suitable threshold ǫ to transform the original interaction structure matrix Λ into the processed matrix Θ Θij takes 1 if Λij > ǫ It should be pointed out that unlike DG, the threshold of DG2 obtains information based on a large number of function values and the original interaction matrix. This is generated by the ISM function. ③Divide variables into irreducible sub-components which is performed by identifying the connected components of the graph with the node adjacency matrix of Θ The author pays attention to two major issues ①Find an efficient ISM function generation matrix to minimize function evaluation ②Find an effective threshold search method to improve the accuracy of decomposition   A Improve the efficiency of DG algorithm Reduce function duplication calculations For an n-dimensional function, the total interaction relationship is n(n-1)/2. Theorem 1 states that each comparison needs to be calculated 4 times. Therefore, the total number of evaluations is 2n(n-1) times. Assume that this function has 3 decision variables f(x1, x2, x3) and the total number of evaluations is 12 times x1↔x2:∆(1)=f(a′, b, c)−f(a, b, c),∆(2)=f(a′, b′, c)−f(a, b′, c) x1↔x3:∆(1)=f(a′, b, c)−f(a, b, c),∆(2)=f(a′, b, c′)−f(a, b, c′) x2↔x3:∆(1)=f(a, b′, c)−f(a, b, c),∆(2)=f(a, b′, c′)−f(a, b, c′), It can be seen from the figure that only 7 points are needed to satisfy In order to calculate Δ(1) and Δ(2) four different points are needed to form a rectangle. To calculate Δ(1), a base point (a, b, c), and (a’,b,c), in order to find the interactive relationship between x1 and x2, so the difference function Δ(1) is also used to evaluate x2, so (a’,b’,c’) and (a,b’,c) are found. The point (a, b, c) is used three times. Points that are only different in one dimension (a’, b, c), (a, b’, c), (a, b, c’) It was repeated twice, and two different points appeared once. Therefore, in order to calculate the interaction relationship between the i dimension and the j dimension, the author will arrive at n(n+1)/2+1 times. See the ISM equation for details. BImprove grouping accuracy. The grouping accuracy depends on ǫ. Theoretically, this value can be set to 0. If the difference between Δ(1) and Δ(2) is positive, it indicates that there is an interactive relationship between the two variables. However, the grouping accuracy will decrease due to rounding bias.
