What properties do variables and labels have? How to get attribute value? Write the instructions. Answer: Variables are operands provided for instructions, and labels provide identifiers for instructions, both for reference in instructions. Their main attributes are: offset attributes, segment attributes, and type attributes. For example: MOV BX, OFFSET VAL; get the offset attribute MOV BX, SEG VAL; get segment attributes MOV BX, TYPE VAL; get the type attribute  6.3 What is the difference between instructions and pseudo-instructions? Can directives appear in code sections? Can instructions be in the data segment? Answer: Instructions can only appear in the code segment. Pseudo-instructions that define data are usually in the data segment. Pseudo-instructions can also be at both ends of the code segment, but not between instructions.  6.4 Can the following program output characters 0 to 9? If not, how should it be modified?
