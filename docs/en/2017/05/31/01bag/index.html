01Dynamic programming of knapsack problem

Programming LanguagesC & C++TechnologyC++C动态规划
First of all, it must be clear that this table is generated from bottom to top and from left to right. For the convenience of description, cell e2 is used to represent the cell in row 2 and column e. The meaning of this cell is to indicate that when there is only item e, there is a backpack with a load-bearing capacity of 2. Then the maximum value of this backpack is 0, because the weight of item e is 4 and the backpack cannot carry it. For cell d2, it means that when there are only items e and d, the maximum value that can be loaded into a backpack with a load-bearing capacity of 2 is still 0, because neither items e nor d can be loaded in this backpack. In the same way, c2=0, b2=3, a2=6. For a backpack with a load capacity of 8, a8=15, how do you get it? According to the state transition equation of the 01 backpack, two values need to be examined, One is f[i-1,j], which for this example is the value of b8 9, and the other is f[i-1,j-Wi]+Pi; here, f[i-1,j] means that I have a backpack with a load capacity of 8. When there are only four items b, c, d, and e available, the maximum value that can be loaded into this backpack f[i-1,j-Wi] means that I have a backpack with a load capacity of 6 (equal to the current backpack load minus the weight of item a). When there are only four items b, c, d, and e available, the maximum value that can be loaded into this backpack f[i-1,j-Wi] refers to cell b6, the value is 9, Pi refers to the value of item a, which is 6 Since f[i-1,j-Wi]+Pi = 9 + 6 = 15 is greater than f[i-1,j] = 9, item a should be placed in a backpack with a load capacity of 8 Code:
#include<stdio.h>
#include<conio.h>
#include<string.h>
int f[1010],w[1010],v[1010];//f记录不同承重量背包的总价值，w记录不同物品的重量，v记录不同物品的价值
int max(int x,int y){//返回x,y的最大值
    if(x>y) return x;
    return y;
}

int main()
{
    int t,m,i,j;
    memset(f,0,sizeof(f));  //总价值初始化为0
    printf("输入背包承重量t、物品的数目m\n");
    scanf("%d %d",&t,&m);  //输入背包承重量t、物品的数目m
    for(i=1;i<=m;i++)
    {
        printf("%d:",i);
        scanf("%d %d",&w[i],&v[i]);  //输入m组物品的重量w[i]和价值v[i]
    }
    for(i=1;i<=m;i++)//尝试放置每一个物品
    {  
        for(j=t;j>=w[i];j--)
        {
            f[j]=max(f[j-w[i]]+v[i],f[j]);
            
            //在放入第i个物品前后，检验不同j承重量背包的总价值，
            //如果放入第i个物品后比放入前的价值提高了，则修改j承重量背包的价值，否则不变
        }
    }
    printf("%d",f[t]);  //输出承重量为t的背包的总价值
    printf("\n");
    getch();
    return 0;
}
