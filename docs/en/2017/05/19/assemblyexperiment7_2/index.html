Assembly experiment 7-2 (input 7-digit binary number to display characters)

汇编语言Technology汇编语言
Experimental requirements: Enter a 7-digit binary number from the keyboard, display the corresponding character, and press Enter to exit the loop and terminate the program.
code segment
assume cs:code
start:
mov bl,0
mov cx,7;循环为7次

L1:
mov ah,1
int 21h;接受数字
cmp al,0dh;回车键退出
jz exit
cmp al,30h
jl exit
cmp al,31h
jg exit;无效输入

shl bl,1;算数左移，每输入一个字符，执行L1一次，BL向左移动一次
sub al,30h
and al,01h
add bl,al;将判断之后的AL的值给BL,防止程序执行过程中改变部分值
loop L1

output:
mov dl,13
mov ah,2
int 21h;输出换行符

mov dl,10
mov ah,2
int 21h
mov dl,bl
mov ah,2
int 21h
exit:
mov ah,4ch
int 21h
code ends
end start
Enter 0110011, that is, 33H is the code for 3. Note that you only need to enter 7 digits, and the first 0 is omitted.
