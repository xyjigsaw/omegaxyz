Assembly experiment 4-2 (32-bit addition, subtraction, multiplication and division problems)

汇编语言Technology汇编语言
Use 16-bit instructions to compile programs to handle the four 32-bit arithmetic operations of addition, subtraction, multiplication and division. This article calculates the value of (3*X+Y-Z)/5 The values are:
x dw 7;大家自己定义一下数据，最好不要雷同哦！！！
y dw -3
z dw 1
After dividing by 5, the quotient is 3, leaving a remainder of 2
data segment
x dw 7;自己定义数据，最好不要雷同！！！
y dw -3
z dw 1
v dd ?
data ends;定义数据段，其中X,Y,Z为一个字节，V为32位结果双字
code segment
 assume cs:code,ds:data
start:
mov ax,data
mov ds,ax
mov ax,3
imul x;IMUL结果保存在AX和DX中
mov cx,ax
mov bx,dx;转移到CX和DX中
mov ax,y
cwd;将Y扩展为32位，以下若干行相同
add cx,ax
adc bx,dx
mov ax,z
cwd
sub cx,ax
sbb bx,dx
mov ax,cx
mov dx,bx
mov cx,5
idiv cx
lea bx,v;取V的地址
mov [bx],dx
mov [bx+2],ax;将结果保存在[bx]到[bx+4]的内存位置32位
mov ah,4ch
int 21h
code ends
end start
Register status for the first four steps ZF=0 SF=0 CF=0 OF=0final stepsAL=03, DL=02 consistent
