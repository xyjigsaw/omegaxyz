Algorithm Basics-Longest Increasing Subsequence

Programming LanguagesC & C++TechnologyC++C动态规划
As a novice, this article serves as a query template [Problem 1] Longest increasing subsequence problem [Problem Description] Suppose L=<a1,a2,…,an> is a sequence of n different real numbers. The increasing subsequence of L is such a subsequence Lin=<ak1,ak2,…,akm>, where k1<k2<…<km and aK1<ak2<…<akm. Find the maximum value of m. Use an array temp[] to save the length of the longest increasing subsequence ending with the current element, and finally find the global optimal solution. The conditions for updating the longest increasing subsequence: a[i]>a[j] (i>j) and the length of the previous increasing sequence is greater than or equal to the length of the current increasing sequence Dynamic programming:   Code (non-repeating subsequence)
#include<iostream>
#define MAX 100
using namespace std;
int main()
{
    int num[MAX],temp[MAX];
    int i,j,n,maxl,m;
    while(cin>>n)
    {
        temp[0]=1;
        maxl=0;
        for(i=0;i<n;i++)
            cin>>num[i];
        for(i=1;i<n;i++)
        {
            m=0;
            for(j=0;j<i;j++)
            {
                if(num[i]>num[j]&&temp[j]>m)
                    m=temp[j];
            }
            temp[i]=m+1;
            if(temp[i]>maxl)
                maxl=temp[i];
        }
        cout<<maxl<<endl;
    }
}
