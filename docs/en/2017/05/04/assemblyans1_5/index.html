What is a compilation language? Brief description of the language of the compilation. The language of the machine is the assembly of the machine commands with binary codes and the rules for the use of a set of machine commands. The compilation language is an abbreviation description of the operating code in the machine command in English, and a description of the operating number by the label, variable, constant. Characteristics of the compilation language: (1) Machine-related: Poor portability but direct control of hardware. (2) Process is efficient. (3) Limitations: subject to instructions, such as considering storage units. (4) Debugging difficulties. 1.2 What is the difference between the compilation procedure and the compilation source procedure? Response: The computer consists of the central processor CPU, the repository, the input system, the output system, which is connected to the system bus. CPU includes an operator and controller, which executes instructions and the controller is responsible for computer control. Storage is a memory component of the computer, which stores the program and data in binary form. The input-output system includes large-volume storage, such as hard drives, and other external settings, such as mouse, keyboard, monitor, etc. 2.2 A brief description of the main functions of the 16-bit aircraft ' s various repositories. Answer: (1) Data repository: Ax, Bx, Cx, Dx; AX: As a loader, is the main repository for arithmetical operations. Stores the operation number in the multiplier command, which transmits information with external equipment in the I/O instruction. BX: The base address repository is used when the universal repository is used to calculate the memory address. CX: When a universal repository is used, it is often used to save a number and when a counter is used. DX: When a universal repository is used, the DX and AX are usually used in a two-word long operation, for some I/O operations, and DX is used to store the port address of I/O. (2) Address repository: Sp, Bp, Si, Di (3) A sector repository: Cs, Ds, Es, Ss; the role of a segment repository is to be used exclusively to locate the repository and to store a section address directly or indirectly. (4) Specialized senders: Ip, Flags; Ip repositorys are dedicated to the address of the next instruction, the Flags logo repository, or program status register. Answer: When the processor operates on two operands, it obtains the result as an unsigned number and sets the carry flag CF accordingly; at the same time, it sets the overflow flag OF according to whether it exceeds the range of signed numbers. It is up to the programmer to decide which flag should be used. In other words, if the operand participating in the operation is considered to be an unsigned number, you should care about the carry; if you think it is a signed number, you should pay attention to whether it overflows. 5.2 Comparison of signed numbers.
