How to install assembly environment under win10 64-bit (including edit commands that can be used on 64-bit machines)

汇编语言Technology汇编语言
Online tutorials can only use the debug command in win10 and cannot use the edit command. After searching for a long time, I finally found the edit source file that can be used on a 64-bit machine and inserted it into the MASM package downloaded from the Internet. Assembly file https://pan.baidu.com/s/1w27JQtrklbaM4ZpzGVZr7A First, download the file and install DOSBox0.74. Be careful not to install it on the C drive. Copy the downloaded MASM file (as shown in the picture) to a folder in the root directory (D or E drive) (create it yourself, for example, I created the xuyi folder in the E drive) Contents of new folderNow open DOSbox Two programs will appear. Use the first program, but do not close the latter one.At this time, the drive letter is Z (for a virtual disk) Next use the mount command to change the mapping point of DOSBOX (note that xuyi is the name of the folder, you enter the name of the folder you created) Input: mount c e:\xuyi Then press the Enter key, When Drive C is mounted as local directory e:\xuyi\ appears, it indicates that the mapping is successful. Use the dos command "c:" to change the drive letter below.Enter virtual disk C: This virtual disk C is the folder xuyi just created Next, you can happily design assembly programs.Use edit command The cursor may disappear after entering. At this time, call the task manager alt+ctrl+del to release the cursor.         Use the mount command every time you open it. If you find it troublesome Turn off dosbox and configure dosbox.conf 1. Open DOSBox 0.74 Options 2. Notepad search [autoexec] 3.[autoexec] MOUNT C E:\xuyi C: xuyi # Lines in this section will be run at startup. # You can put your MOUNT lines here. Remember--4.
