A must-have website for programmers

技术域其它web
1.CSDN http://www.csdn.netFounded in 1999, CSDN (Chinese Software Developer Network) is China's largest IT community and service platform. It provides Chinese software developers and IT practitioners with full life cycle services such as knowledge dissemination, career development, and software development to meet their rigid needs such as learning and sharing knowledge and information during career development, establishing a social circle for career development, and realizing technology commercialization through software development. It has more than 30 million registered members (including 8 million active members) and 500,000 registered companies and partners. 2.Githubhttp://github.comGitHub is a hosting platform for open source and private software projects. Because it only supports Git as the only version library format for hosting, it is named GitHub. GitHub was officially launched on April 10, 2008. In addition to Git code warehouse hosting and basic web management interface, it also provides subscription, discussion groups, text rendering, online file editor, collaboration map (report), code snippet sharing (Gist) and other functions. Currently, it has more than 3.5 million registered users and a large number of hosted versions, including well-known open source projects Ruby on Rails, jQuery, python, etc. 3.Oschina Open Source Chinahttp://www.oschina.netCode cloud: http://git.oschina.net Code Cloud is a fast, free and stable online code hosting platform based on Git launched by the open source Chinese community team. It does not limit the number of private and public libraries. 4. Blog Parkhttp://www.cnblogs.com/Founded in 2004, it is an online learning community for programmers. "Live to grow old and learn to grow old" is the truest portrayal of programming life. Learning not only requires one person's hard work, but also requires a community with a strong learning atmosphere, willingness to share, and mutual help. 5. Zhihuhttp://www.zhihu.com/The largest social Q&A community in China, with a lot of programmer skills, workplace discussion topics and experts sharing content. 6.UiSourcehttp://uisource.com/A comprehensive front-end and interface resource website provides a platform for IT developers and designers to discover, obtain, and share results. What programmers hate most is reinventing the wheel, and this is a good place to avoid reinventing the wheel. 7.Stackoverflowhttp://stackoverflow.com/Programmer Q&A website. If you have any questions, you can ask them or search for answers. You must have a basic knowledge of English.
