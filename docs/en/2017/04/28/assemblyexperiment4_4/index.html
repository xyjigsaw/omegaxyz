Assembly experiment 4-4 (AL byte content reverse sorting)

汇编语言Technology汇编语言
Please comment if there are any errors or issues, thank you! ! Note that it is binary, mainly examining the assembly shift RCL and RCR instructions. SHL(Shift Left): Logical shift left SHR(Shift Right): Logical shift right SAL(Shift Arithmetic Left): Arithmetic shift left SAR (Shift Arithmetic Right): arithmetic right shift The SHL and SAL are the same, but the SHR and SAR are different. SHL, SAL: Shift each bit left, fill the low bit with 0, carry the high bit to CF SHR : Shift each bit to the right, carry the low bit to CF, and fill the high bit with 0 SAR : Shift each bit to the right, carry the low bit to CF, keep the high bit unchanged Their results affect OF, SF, ZF, PF, CF Their command formats are the same: SHL r/m, i8 SHL r/m, CL   The value of AL used in this article is: 10110001
code segment
assume cx:code
start:
mov al,10110001b
mov cx,8
xuyi:
  rcl al,1
  rcr ah,1
loop xuyi
mov al,ah
mov ah,4ch
int 21h
code ends
end start
Note that the initial value of AL is 10110001, which is B1At the end, the value of AL is reversed to 10001101, which is 8D
