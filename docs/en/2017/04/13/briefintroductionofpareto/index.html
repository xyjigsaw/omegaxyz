Overview of multi-objective optimization problems

TechnologyMachine LearningEvolutionary Algorithms多目标优化算法
Definition: Searching for the best possible solution (non-inferior solution) within a given area under the constraints of several conflicting or interacting conditions. Keywords: conditional constraints, compromise optimal solution (the fact that the solution is not unique is the essential difference from single-objective optimization problems) Text description: D decision variable parameters; N objective functions; m+n constraints. Mathematical description:X (lowercase) is the D-dimensional decision vector; y is the target vector; N is the total number of optimization goals; gi(x)<=0 and hj(x) are conditional constraints (the feasible region is known and determined); Note:The figure below is the definition of optimal solution or non-inferior optimal solution in multi-objective optimization problemSome literature explanations about Pareto optimal solutions: If x*∈C, and there is no solution x superior to x* in C, then x* is said to be the Pareto optimal solution of the multi-objective optimization model, also known as the effective solution. Generally speaking, there is no optimal solution to a multi-objective optimization problem. All possible solutions are called non-inferior solutions, also known as Pareto solutions. Traditional optimization techniques can generally obtain one Pareto solution set at a time, while using genetic algorithms to solve can obtain more Pareto solutions, or even the entire solution becomes a Pareto solution. These solutions form an optimal solution set, called the Pareto optimal solution. It is a set of solutions in which the improvement of any objective function value must be at the expense of other objective function values. It is called the Pareto optimal domain, or Pareto set for short. One of the basic concepts of multi-objective programming. For multi-objective decision-making problems (see "Multi-objective decision-making problems") that include quantitative and qualitative attributes, the non-inferior solution means that in the given set of alternative solutions, no solution can be found that can improve each indicator. In multi-objective planning, it refers to effective solutions (see "Effective Solutions") and more optimal solutions (see "More Optimal Solutions"). The so-called non-inferior solution refers to a set of solutions (also called a non-inferior solution set): compared with any solution outside the set, at least one of their objective functions is better than the solution outside the set and the other objective functions are no worse than the solution outside the set. Application of different algorithms in multi-objective optimization There is no unique global optimal solution to multi-objective optimization problems. But still need to find a final solution. There are three categories of methods: 1. Generative method: Find a large number of non-inferior solutions and find the optimal solution. 2. Interaction method: analyze the target to find the optimal solution. 3.
