C++STL generic programming container library

Programming LanguagesC & C++TechnologyC++
STL = Standard Template Library, the collective name for a series of software developed by HP Labs. It was developed by Alexander Stepanov, Meng Lee, and David R Musser while working at HP Labs. Fundamentally speaking, STL is a collection of "containers" such as list, vector, set, map, etc. STL is also a collection of algorithms and other components. The collection of "containers" and algorithms here refers to the masterpieces of many smart people in the world over many years. The purpose of STL is to standardize components so that they do not need to be redeveloped and ready-made components can be used. STL is now part of C++, so there is nothing extra to install. sequence container Vector (vector) Continuously stored elements <vector> List (list) A doubly linked list composed of nodes, each node contains an element <list> Double-ended queue (deque) An array of consecutively stored pointers to different elements <deque> adapter container stack (stack) Last-in-first-out arrangement of values <stack> Queue (queue) First-in first-out arrangement of values <queue> Priority queue (priority_queue) A queue in which the order of elements is determined by some predicate acting on the stored value pairs <queue> associative container A set is a red-black tree composed of nodes. Each node contains an element. The nodes are arranged according to some predicate that acts on pairs of elements. No two different elements can have the same order <set> Multiset allows the existence of two sets of elements with equal order <set> Map (map) A collection of {key, value} pairs, arranged with some predicate acting on the key pairs <map> Multimap: A mapping that allows key pairs to be in equal order <map> 1. In addition, we can define the priority ourselves according to the needs and overload the '<' symbol so that the elements in the queue are arranged in a certain order. Suppose we define a job structure in the operating system. There are two elements in it, one is the job name char name[5]; the other is the arrival time; put the job object into the priority queue. It is required that the priority queue is arranged in ascending order of job arrival time, that is, the job with the smallest arrival time is at the head of the queue.
struct Job //定义作业结构体  
	{  
	    char name[5];//作业名  
	    int arriveTime;//到达时间  
	    bool operator < (const Job another)const{//如果当前作业到达时间大于另一个，则当前作业优先级小  
	        if(arriveTime > another.arriveTime)  
            return true;  
	        return false;  
	    }  
	}job[3];  
	  
	int main()  
	{  
	    strcpy(job[0].name, "no1");  
	    strcpy(job[1].name, "no2");  
	    strcpy(job[2].name, "no3");  
     job[0].arriveTime = 2;  
	    job[1].arriveTime = 1;  
	    job[2].arriveTime = 3;  
	  
	    priority_queue<Job> que;  
	    que.push(job[0]);  
	    que.push(job[1]);  
     que.push(job[2]);  
	    for(int i = 0; i < 3; ++ i)  
	    {  
	        Job tempJob = que.top();  
	        que.pop();  
	        cout<< tempJob.name <<"  "<<tempJob.arriveTime<<endl;  
	}  
	return 0;  
	}
The output is: no2 1 no1 2 no3 3 Implemented the ordering of jobs in the priority queue according to arrival time 4.Vector

6.Map Map is a mapping relationship, one-to-one, the first is the keyword (first), the second is the key value (second), the keyword is unique, and the elements in the map are ordered by the keyword. In practical applications, the meanings of keywords and key values must be considered and used flexibly.

7.Header file #include<algorithm> Arrays can be sorted easily using sort, which can pass two or three parameters. The first parameter is the first address of the interval to be sorted, and the second parameter is the next address of the end address of the interval, that is, the sorted interval is [a, b). For example, there is an array int a[5], so that a[0] to a[4] are ordered from small to large. Just write sort(a, a + 5). The general sort(a, a+ n); // n is the number of elements. Sort uses quick sort internally, which is generally very efficient.
const int n = 3;  
	int arr[n] = {1, 3, 2};  
	sort(arr,arr+n);  
	for(int i = 0; i < n; ++ i)  
	cout<< arr[i] <<endl;
In addition, we can also sort the elements according to our own needs. The elements can be structures. The third parameter, the comparison function, is used here to tell the computer in what order to sort. For example: sort from largest to smallest.
bool cmp(int a, int b)//定义比较函数，从大到小
