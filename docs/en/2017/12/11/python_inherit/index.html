Python class inheritance (calling parent class members and methods)

Programming LanguagesTechnologyPythonPython
The initialization method of a class in python is __init__(), so the initialization method of the parent class and subclasses is this. If the child class does not implement this function, the parent class's initialization function is called during initialization. If the child class implements this function, it will overwrite the parent class's function. Since it inherits the parent class, it is necessary to explicitly call the parent class's __init__() in this function. This is different from C++ and jAVA, which automatically call the parent class's initialization function. The following methods are available for calling parent class functions:
class Animal():
    def __init__(self, name):
        self.name = name

    def saySomething(self):
        print("I am " + self.name)

class Dog(Animal):
    def __init__(self, name):
        super().__init__(name)

    def saySomething(self):
        print ("I am "+ self.name + ", and I can bark")
1. Directly write the class name to call: parent_class.parent_attribute(self) 2. Use the super(type, obj).method(arg) method to call: super(child_class, child_object).parent_attribute(arg) [No need to write self] 3. Call the parent class method of this class in the class definition, you can directly super().parent_method(arg) [I personally recommend this writing method] An example is as follows:
class Animal():
    def __init__(self, name):
        self.name = name

    def saySomething(self):
        print("I am " + self.name)

class Dog(Animal):
    def __init__(self, name):
        super().__init__(name)

    def saySomething(self):
        print ("I am "+ self.name + ", and I can bark")

    def animal_say_1(self):
        # 子类调用父类的方法
        #  方式1
        super(Dog, self).saySomething()

    def animal_say_2(self):
        #  方式2 [推荐]
        super().saySomething()
    
    def animal_say_3(self):
        # 方式3
        Animal.saySomething(self)


if __name__ == "__main__":
    dog = Dog("Blake")
    dog.saySomething()
    dog.animal_say_1()
    dog.animal_say_2()
    dog.animal_say_3()
    # 子类对象调用被覆盖的父类方法
    super(Dog, dog).saySomething()

'''
输出结果如下：
python python_class_inheritance_example.py
I am Blake, and I can bark
I am Blake
I am Blake
I am Blake
I am Blake
'''
If you want to call members of the parent class, remember that they cannot be private variables in the parent class, and self.__NAME cannot be called in the subclass!
