Python recognizes natural language (words) in strings

Programming LanguagesTechnologyPythonPython
Generate a random string (without spaces) and identify whether there are words in natural language in this string. For example, "ervmothersdclovecsasd", the two words "mother" and "love" exist in this string. So I thought of dividing the input words. At this time, the external dictionary file will be used. I imported the commonly used words into the txt file (I am not sure whether NTLK has a similar function. I will study it during the winter vacation. Here I will implement what I want to do first) as shown in the figure. In this way, each continuous string can be divided into natural languages. The forward maximum matching algorithm is used here. Dictionary file (preferably an Oxford dictionary):If you need a dictionary file download link, please leave me a message.
note1 = input("Enter your input: ")
wordlist = []
get_list = []
f = open('C:\\Users\\dell\\desktop\\big.txt', 'r')

for line in f:
    nlen = len(line)-1
    wordlist.append(line[:nlen])
f.close()


def divide_str(note, wordlist):
    i = 100
    head = 0
    flag = 0
    while head <= len(note) - 1:
        if head >= (len(note)-i):
            i = len(note)-head
        for p in range(i):
            rear = head + i - p
            flag = 0
            for each in wordlist:
                if note[head:rear] == each:
                    get_list.append(each)
                    # print(each)
                    head = head + len(each)
                    flag = 1
                    break
            if flag == 1:
                break
        if flag == 0:
            head = head + 1

divide_str(note1, wordlist)
print(get_list)
