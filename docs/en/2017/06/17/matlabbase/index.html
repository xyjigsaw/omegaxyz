Matlab basic syntax and instructions

TechnologyMATLABMATLAB
1.clear: Clear memory variables and functions, that is, clear the WorkSpace variables; 2.clc: Clear the contents of the current Matlab command window; it is equivalent to clearing the screen and starting again. Be concise. Note that clc does not know the variables. Generally when writing an m file, if it is not a function, clear;clc; is usually added in front. The purpose is to clear the memory and screen when the m file is running, so as to avoid the influence of variables with the same name and the screen being easy to observe; 3.help: This is also the most important function. When you want to know the relevant instructions of a certain function, you can use the help function name to obtain it. Of course, you can also open the help document. However, this method is faster. 4.zeros: Create a matrix with all 0s; ones: Create a matrix with all 1s. Parameters can be multi-dimensional, such as zeros(2,3)... 5.size: You can calculate the size of the matrix. Also related is length. Size can calculate the size in different dimensions, such as size(A,1).. 6.rand: Random number generator, which can be used directly to generate a matrix of any latitude, such as rand(2,3), which is also a commonly used thing; 6.plot: The operation of drawing images. Please refer to the help for the specific format. Operations should be very frequent. When doing scientific research, you often need to look at trends. Drawing is a very common thing in MATLAB. . . There is also a subplot function that divides a figure into multiple blocks for operation. Thinking about using mschart to draw in C#, I am also drunk. . .
