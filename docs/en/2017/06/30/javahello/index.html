The first line of JAVA (getting started with JAVA)

Programming LanguagesJavaJava
If a worker wants to do his job well, he must first sharpen his tools. First, you must download JAVA IDE (Integrated Development Environment (IDE, Integrated Development Environment)) Here are some JAVA IDEs:

1. JAVA IDE

1.NetBeans

NetBeans is an open source IDE written in Java. NetBeans IDE supports standard out-of-the-box development for all Java application types (Java SE, JavaFX, Java ME, web, EJB and mobile apps). The modular design of NetBeans means that it can be extended by third parties to create plug-ins that improve functionality (for example, the PDF plug-in for NetBeans is a good example). NetBeans IDE can be used for Java development, but also supports other languages, especially PHP, C/C++, and HTML5. Official website:https://netbeans.org/

2.Eclipse

Eclipse is a free java IDE written primarily in Java. Eclipse allows you to create a variety of cross-platform Java applications for mobile, web, desktop and enterprise domains. Its main functions include Windows generator, integrated Maven, Mylyn, XML editor, Git client, CVS client, PyDev, and Eclipse also has a basic workspace with an extensible plug-in system to meet your custom IDE needs. Through plug-ins, you can also develop applications in other programming languages, including C, C++, JavaScript, Perl, PHP, Prolog, Python, R, Ruby (including the Ruby on Rails framework), and more. Official website:https://eclipse.org/

3.IntelliJ IDEA

IntelliJ IDEA is a free Java IDE (Integrated Development Environment) mainly used for Android application development, Scala, Groovy, Java SE and Java programming. It is designed to be lightweight and provides useful features like JUnit testing, TestNG, debugging, code inspection, code completion, support for multiple refactorings, Maven build tool, ant, visual GUI builder and XML and Java code editor. Official website:https://www.jetbrains.com/idea/features/I am using IntelliJ IDEA
