Curriculum Design TV Contest Voting System

Programming LanguagesC & C++Data StructuresTechnologyC++Data StructuresC
Design and implementation of audience voting and ranking system for TV competitions (8 hours)Problem description:In many TV competitions, usually after the contestants' performances, the audience votes for the contestants through the buttons in their hands, and then counts the number of votes the contestants received and sorts them in descending order from high to low, thus automatically producing the champion, runner-up and third runner-up. It is required to write algorithms to simulate the functions of the above system.1.Basic requirements(1) First enter the number of contestants (range is 1-9), and then use the malloc function to allocate a sequence table for storing contestant information based on the number of contestants; (2) Store the player's number and name in the sequence table accordingly; (3) The audience votes by pressing the buttons. Press '1' to vote for contestant No. 1, press '2' to vote for contestant No. 2, and so on, and press '0' as the end of voting; (4) After the voting ends, sorting is performed. Various sorting algorithms are used here, and then the ranking is calculated for each contestant. Those with the same votes also have the same ranking;2. Key points and difficultiesKey points: (1) Design of contestant data structure; (2) Design of sorting algorithm; Difficulty: Performance analysis and comparison of various sorting algorithms3. Homework and extracurricular study requirements:Design independently according to the requirements of the question. After the design is completed, a course design report must be written as required. The teaching and learning of this knowledge point can support the evaluation of the achievement of the indicators in the "Graduation Requirement 4 Research" of "Indicator Point 4.1 Able to use scientific research methods to conduct demand analysis and research on complex software engineering problems; Indicator Point 4.2 Familiar with the development and application environment of complex software systems, research and formulate reasonable software design and development plans. Indicator Point 4.3 Able to conduct research and analysis of prototype verification methods, and reasonably verify software systems." In the process of solving specific problems, students can flexibly and skillfully choose appropriate data structures and design effective algorithms, thereby deepening their understanding of commonly used data structures, strengthening students' logical thinking and hands-on abilities, consolidating good programming habits, mastering the basic methods of engineering software design, and laying a solid foundation for subsequent courses. class player
{
public:
    int num;
    char name[30];
    int score;
    int ranking;
};

class ope
{
public:
    void init(player *a,int number);
    void output_info(player *a,int number);
    void outcome(player *a,int number);
    void voting(player *a,int number);
    void shell_insert(player *a,int n,int dk);
    void shell_sort(player *a,int n,int t);
    void sort_outcome(player *a,int number);
};

void ope::shell_insert(player *a,int n,int dk)
{
    for(int i=dk+1;i<=n;i++)
    {
        int j;
        if(a[i].score>a[i-dk].score)
        {
            strcpy(a[0].name,a[i].name);
            a[0].score=a[i].score;
            a[0].num=a[i].num;
            for(j=i-dk;j&&a[j].score<a[0].score;j-=dk)
            {
                strcpy(a[j+dk].name,a[j].name);
                a[j+dk].score=a[j].score;
                a[j+dk].num=a[j].num;
            }
            strcpy(a[j+dk].name,a[0].name);
            a[j+dk].score=a[0].score;
            a[j+dk].num=a[0].num;
        }
    }
}
void ope::shell_sort(player *a,int n,int t)
{
    int i;
    for(;t>=1;t/=2)
    {
        ope::shell_insert(a,n,t);
        for(i=1;i<=n;i++)
            a[i].ranking=i;
        for(i=1;i<n;i++)
            if(a[i+1].score==a[i].score)
                a[i+1].ranking=a[i].ranking;
    }
}

void ope::voting(player *a,int number)
{
    if(flag==0)
        cout<<"\n输入选手编号进行投票,按0结束。\n";
    else
        cout<<"\n补投开始：输入选手编号,按0结束。\n";
    flag=1;
    int vote;
    while(scanf("%d",&vote)&&vote)
    {
        if(vote>0&&vote<=number)
        {
            a[vote].score++;
            n++;
            cur_sys();
            cout<<"请继续，按0结束"<<endl;
        }
        else if(vote==0)break;
        else
        {
            cur_sys();
            cout<<"请继续，按0结束"<<endl;
            continue;
        }
    }
    cout<<"投票成功!"<<endl;
}
