How to Make a Website (How do I build a website?)

TechnologyWebWebwordpress
First of all, making a website requires:①The domain name is example.com ②Virtual space or cloud server ③Website code production tools or website backend management tools (you can use Baidu to create templates) ④ Certain programming skills and code understanding abilities (excellent website makers need to know PHP, JavaScript, CSS, HTML, SQL, Markdown, etc.)

Part One: Domain Name

A domain name is the entrance and identification of a website. The domain name needs to be purchased. Generally purchased from Alibaba Cloud (Wanwang), Tencent Cloud, JD Cloud, Godaddy and other websites. You need to think of a good name for the domain name. Once selected, it cannot be changed. It is recommended that you purchase domain names with top-level domain names ending in .com and .cn. These domain names are common internationally and will help your website stand on an international level. Of course, you can also purchase other top-level domain names in a unique way. For example, Xiaomi spent 20 million yuan to purchase the domain name mi.com. The domain name elem.me is quite unique to see if it is very powerful. Of course, domain names need to be registered in China. Domain names purchased from Alibaba Cloud and Tencent Cloud must be registered with the Ministry of Industry and Information Technology. The filing process is as follows:Filing is usually done at the place where you purchased the domain name, and Alibaba Cloud will automatically submit it to the Ministry of Industry and Information Technology for you. Generally speaking, the registration requirements are very strict. On average, it takes two to be approved by the Ministry of Industry and Information Technology, which takes about two to three weeks. During the registration process, you will be asked to print a "Personal Website Commitment Letter" and take a photo to upload. In order to verify your identity, you will also need to record a personal video of blinking, smiling, etc. Of course, your personal information is accurate to your address. (It is estimated that it will be convenient to arrest people, control public opinion, and suppress thoughts.) If you don’t want to register, it is recommended not to buy the domain name from a large domestic company, but from outside the mainland (Hong Kong and Taiwan are also acceptable) Of course, if you don't register, if the Internet police find out, your website will be blocked or can only be accessed using a VPN. (It’s better to register!!!) After the registration with the Ministry of Industry and Information Technology is completed, if you are a large enterprise or company or need a formal website, it is recommended that you also go to the Public Security Bureau to register, that is, go to the Public Security Bureau where the address where your website information is filled in is located. (Not required for personal website)

Part 2 Virtual Space or Cloud Server

1.Virtual space Virtual space is a small portion of space set aside by the cloud server to store your website data. After purchasing the domain name, they may provide you with a virtual space for free, or you can purchase it, or use a third-party virtual space. Taking the domain name omegaxyz.com as an example, resolve the domain names www.omegaxyz.com and omegaxyz.com to the virtual host (assuming that the IP address of the virtual host is 21X.5.7X.1XX). After the resolution is completed, you can use www.omegaxyz.com and omegaxyz.com to access the website. Of course (after the resolution is completed, it must be bound on the virtual host for it to take effect. The virtual host of this site is fully capable of autonomous domain name binding). ①First, please enter domain name management. (Log in as a member -> English domain name management -> Find omegaxyz.com in the list and click "Domain name resolution"). ②First, parse www.omegaxyz.com to IP: 21X.5.7X.1XX. Click "Add host record", and then enter "www" in the pop-up box. Note that "www.omegaxyz.com" cannot be entered. ③Press "OK" and enter the IP address: 21X.5.7X.1XX. ④ Press "OK" again, the analysis is completed, and the analysis takes effect in real time, which greatly facilitates customer management. ⑤ Generally speaking, in addition to using the URL www.omegaxyz.com with www to access the website, users also like to use omegaxyz.com without www. Let's resolve omegaxyz.com to IP: 21X.5.7X.1XX, or click "Add Host Record", as shown in the figure below, there is no need to enter any characters at this time, just click "OK". (Do not enter characters to represent the domain name omegaxyz.com itself) ⑥After clicking OK, the IP address input box appears again, enter IP: 21X.5.7X.1XX, and then click OK. ⑦After pressing OK, the analysis is completed. As shown in the picture:The third-party virtual space recommends Qiniu Cloud. Please study this part by yourself.Note: DNS resolution is similar to domain name resolution. There will be a prompt when you go to the console interface, so I won’t go into details here. 2.Cloud server Abbreviated as ECS (elastic cloud servicer) The industry name for cloud servers is actually called computing units. The so-called computing unit means that this server can only be regarded as a person's brain, which is equivalent to the CPU of an ordinary computer. The resources inside are limited. If you want to get better performance, the first solution is to upgrade the cloud server, and the second is to deploy other software that consumes computing unit resources on the corresponding cloud service. For example, there are dedicated cloud database services for databases, and dedicated file storage services for static web pages and images. Of course, using a cloud server to store web pages is a bit overkill, but it can make the website perform better and load more subdomains such as jiaotong.omegaxyz.com. The most important thing is that if you want to make an APP, cloud server is the best choice. The only disadvantage is that it is expensive (look for discount policies). When purchasing a cloud server, you must pay attention to the system and environment of the cloud server (Alibaba Cloud is a Centos system based on Linux, Java and PHP environments, etc.), which will be discussed in detail in the third part.
