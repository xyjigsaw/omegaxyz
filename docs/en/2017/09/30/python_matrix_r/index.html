Python list comprehension matrix transpose

Programming LanguagesTechnologyPythonPython
Assume A is an m×n order matrix (i.e. m rows and n columns), and the elements in the i-th row and j column are a(i,j), that is: A=a(i,j)
Define the transpose of A as an n×m order matrix B that satisfies B=a(j,i), that is, b (i,j)=a (j,i) (the i-th row and j-th column element of B is the j-th row and i-th column element of A), note A'=B
Here is the list comprehension code which is very simple:
matrix = eval(input('输入矩阵（格式为 [[a11,a12],[a21,a22]] 以此类推）: \n'))
t_2 = [[row[i] for row in matrix] for i in range(len(matrix[0]))]
print(t_2)
