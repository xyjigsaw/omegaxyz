Overview of basic methods for feature selection

TechnologyMachine LearningEvolutionary Algorithms特征选择
There are three main methods of feature selection: 1. Filter method The main idea is to "score" the features of each dimension, that is, to assign weights to the features of each dimension. Such weights represent the importance of the features of that dimension, and then they are sorted according to the weight. The main methods are:Chi-squared test Information gain (information gain), see "Simple and easy-to-learn machine learning algorithm - ID3 algorithm of decision tree" for details correlation coefficient scores 2. Wrapper method The main idea is to regard the selection of subsets as a search optimization problem, generate different combinations, evaluate the combinations, and then compare them with other combinations. In this way, the selection of subsets is regarded as an optimization problem, which can be solved by many optimization algorithms, especially some heuristic optimization algorithms, such as GA, PSO, DE, ABC, etc. For details, see "Optimization Algorithm - Artificial Bee Colony Algorithm (ABC)", "Optimization Algorithm - Particle Swarm Optimization (PSO)".The main methods are: recursive feature elimination algorithm (recursive feature elimination algorithm) 3. Embedded method The main idea is to learn the attributes that best improve the accuracy of the model under a given model. This sentence is not easy to understand. It actually talks about selecting those attributes that are important for model training in the process of determining the model. Main method: regularization, see "Simple and Easy-to-Learn Machine Learning Algorithm—Ridge Regression". Ridge Regression is a regularization term added to the basic linear regression process.
