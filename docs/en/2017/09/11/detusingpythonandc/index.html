n-order determinant calculation implemented in Python and C language

Programming LanguagesC & C++TechnologyPythonC++CPython
In mathematics, the determinant is a function whose domain is the matrix A of det, and whose value is a scalar, written as det(A) or | A |. Whether in linear algebra, polynomial theory, or calculus (such as the substitution method of integration), determinants, as basic mathematical tools, have important applications. The determinant can be seen as the generalization of the concept of directed area or volume in general Euclidean space. In other words, in n-dimensional Euclidean space, the determinant describes the impact of a linear transformation on the "volume".
Here we introduce a simple method for computer calculation of determinants. It is only used for our general calculation of determinants and is not suitable for scientific research and calculation of big data. Here we use the diagonal method and the recursive price reduction algebraic cofactor summation method. Python recursive determinant code:
def det(m):
    if len(m) <= 0:
        return None
    elif len(m) == 1:
        return m[0][0]
    else:
        s = 0
        for i in range(len(m)):
            n = [[row[a] for a in range(len(m)) if a != i] for row in m[1:]]  # 这里生成余子式
            s += m[0][i] * det(n) * (-1) ** (i % 2)
        return s
print('答案为: ', det(eval(input('输入行列式（格式为 [[a11,a12],[a21,a22]] 以此类推）: \n'))))
