Python uses os.walk to traverse folders

Programming LanguagesTechnologyPythonPython
I have written before about traversing folders using ordinary methods.https://omegaxyz.com/2017/09/19/pythongetfiles/method, last week the python class teacher directly used the os.walk() method to traverse, which saves a lot of trouble. This block of os.walk is a commonly used module for traversing a directory. It returns a tuple containing 3 elements: dirpath, dirnames, and filenames. dirpath returns all absolute paths in the directory in the form of a string; dirnames returns the folder name under each absolute path in the form of a list; filesnames returns all the file names under the path in the form of a list.
import os 

def visitDir2(path):
    if not os.path.isdir(path):
        print('Error:"',path,'" is not a directory or does not exist.')
        return
    list_dirs = os.walk(path)                #os.walk返回一个元组，包括3个元素：
                                                           #所有路径名、所有目录列表与文件列表
    for root, dirs, files in list_dirs:       #遍历该元组的目录和文件信息
        for d in dirs: 
            print(os.path.join(root, d))     #获取完整路径
        for f in files: 
            print(os.path.join(root, f))     #获取文件绝对路径

visitDir2('h:\\music')
