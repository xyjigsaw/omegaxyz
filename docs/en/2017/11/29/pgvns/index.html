High-dimensional feature selection algorithm based on VNS and Markov blanket grouping

TechnologyMachine LearningEvolutionary Algorithms特征选择
Keywords: high-dimensional feature selection, Markov blanket, variable neighborhood search algorithm, feature grouping Journal: Information Science (Region 1)

①Markov blanket:

Divide a complete set U of random variables into three mutually exclusive parts, variable In formula (2-16), the set MB is the set A I am talking about, {U-MB-{X}} is the set B I am talking about, the symbol "⊥" means "independent", and the symbol "|" means under a given xx condition, so it can be read as "when the set MB is given, the variable X is independent of {U-MB-{X}}". For example, U is the entire society, X is you personally, and MB is the people in your life circle. According to philosophy, everything is connected; however, you do not have any relationship with everyone in the society, but have an indirect relationship with them through your life circle. That is, given your life circle, you have no relationship with the rest of the society (independent).

②Symmetrical Uncertainty

For each feature fi, calculate its symmetric uncertainty, where IG(c l fi) represents the information gain between category c and feature Ji, H(c) and H(fi) represent the information entropy (entropy) of category c and feature Fi respectively. Given a threshold value γ, such as SU(fi,c) ≥γ, then fi is a strongly relevant feature for category c and should be retained, otherwise fi should be removed.

③Approximate Markov Blankets

④Predominant group generation algorithm——GPGG

STEP1：Redundancy analysis. For each feature Xi, a threshold α is given according to its symmetric uncertainty, such as SU(Xi,y) > α, then and It is a strongly relevant feature for category y and is put into the set L.STEP2：Sort in descending order according to the value of SU.STEP3：Use the approximate Markov blanket method (to determine whether a Markov blanket exists for each element) to find the main elements from L and put them into the set g.

⑤Variable Neighborhood Search (VNS)

The main idea of ​​the variable neighborhood search algorithm is:Systematic search using multiple different neighborhoods. First use the smallest neighborhood search, and when the solution cannot be improved, switch to a slightly larger neighborhood. If the solution can continue to be improved, return to the smallest neighborhood, otherwise continue to switch to a larger neighborhood.

⑥PGVNS algorithm (PredominantGroupbasedVariableNeighborhoodSearch)

In order to apply the VNS algorithm to high-dimensional data sets, the author proposed a feature selection method based on the VNS principal element group. First, the GPGG algorithm is used to generate the principal element group as the initial solution, and then the shaking method is used to select a new solution set. Through this iteration, the best solution set is finally obtained.original:García-Torres M, Gómez-Vela F, Melián-Batista B, et al. High-dimensional feature selection via feature grouping: A Variable Neighborhood Search approach[J]. Information Sciences, 2016, 326(C):102-118.
