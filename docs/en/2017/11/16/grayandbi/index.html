Conversion between Gray code and binary code

Programming LanguagesC & C++TechnologyC++C数字逻辑
Gray code is also called cyclic binary code or reflected binary code. Gray code is a coding method that we often encounter in engineering. Its basic feature is that any two adjacent codes have only one binary number that is different. This will be explained in detail below. The basic characteristic of Gray code is that any two adjacent codes only differ by one binary number, which is very important. The conversion relationship between commonly used binary numbers and Gray codes is as follows:Convert binary code to binary Gray code The rule for converting a binary code into a binary Gray code is to retain the highest bit of the binary code as the highest bit of the Gray code, and the second-highest bit of the Gray code is the XOR of the high-order bit of the binary code and the second-highest bit, and the remaining bits of the Gray code are similar to the second-highest bit.C language code:
#include<iostream>
#include <bitset>
using namespace std;

int D2G(int x)
{
    return x^(x>>1);
}

int main()
{
    int x;
    cin>>x;
    cout<<D2G(x);
    //注意输入与输出均为十进制
}
Convert Gray code to binary code The rule of converting a binary Gray code into a binary code is to retain the highest bit of the Gray code as the highest bit of the natural binary code, and the second-highest natural binary code is the high-bit natural binary code and the second-highest Gray code, and the remaining bits of the natural binary code are similar to the second-highest natural binary code.
