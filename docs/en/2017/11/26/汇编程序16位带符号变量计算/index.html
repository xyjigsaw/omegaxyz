Assembler 16-bit signed variable calculation

汇编语言Technology汇编语言
Write a complete program using 16-bit instructions and test it on the computer to calculate W=(x+y)*z, where all variables are 16-bit signed variables. Pay attention to the use of directives.
data segment
x db 7
y db -3
z db 2
w db ?
data ends;定义数据段，其中X,Y,Z为一个字节，V为16位结果字节

code segment
	assume cs:code,ds:data
	
start:
mov ax,data
mov ds,ax
mov dx,0
mov al,x
adc al,y
mov bl,z
imul bl
lea bx,w
mov [bx],ax

add al,30h

mov dl,al
mov ah,02h
int 21h
	 
mov ah,4ch
int 21h
code ends
end start
Use 16-bit instructions to compile programs to handle the four 32-bit arithmetic operations of addition, subtraction, multiplication and division. Please see this article to calculate the value of (3*X+Y-Z)/5https://omegaxyz.com/2017/05/05/assemblyexperiment4_2/
