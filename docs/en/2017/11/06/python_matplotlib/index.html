Python matplotlib module draws simple images

高级语言技术域Python
Matplotlib is aPythonA 2D plotting library that produces publication-quality graphics in a variety of hardcopy formats and in a cross-platform interactive environment[1]  。
With Matplotlib, developers can generate plots, histograms, power spectra, bar charts, error plots, scatter plots, etc. with just a few lines of code.
Matplotlib basics
1. Elements included in basic charts in Matplotlib
x-axis and y-axis
horizontal and vertical axes
x-axis and y-axis scale
The scale marks the separation of the coordinate axes, including the minimum scale and the maximum scale.
x-axis and y-axis tick labels
Represents the value of a specific axis
drawing area
actual drawing area
2.hold attribute
The hold attribute defaults to True, allowing multiple curves to be drawn in one figure; modify the hold attribute to False, and each plot will overwrite the previous plot.
However, it is currently not recommended to use the hold attribute (there will be a warning). So just use the default settings.
3. Grid lines
grid method
Use the grid method to add grid lines to the diagram
Set the grid parameters (the parameters are the same as the plot function)
.lw represents linewidth, the thickness of the line
.alpha represents the lightness and darkness of the line
4.axis method
If the axis method does not have any parameters, it returns the upper and lower limits of the current coordinate axis.
5.xlim method and ylim method
In addition to the plt.axis method, you can also set the coordinate axis range through the xlim and ylim methods.
6.legend method
Two methods of passing parameters:
[Recommended] Add label parameter to plot function
Pass in a list of strings in the legend method
Configure matplotlib parameters
Permanent configuration
matplotlib configuration information is read from the configuration file. Permanently valid default values ​​can be specified in the configuration file for almost all properties of matplotlib.
Per installation configuration file
Python's site-packages directory (site-packages/matplotlib/mpl-data/matplotlibrc)
System-level configuration, the configuration file will be overwritten every time matplotlib is reinstalled
If you want to maintain a persistent and effective configuration, it is best to set it in a user-level configuration file
The best way to use this configuration file is as a default configuration template
User-level .matplotlib/matplotlibrc files (Per user .matplotlib/matplotlibrc)
User's Documents and Settings directory
You can use the matplotlib.get_configdir() command to find the current user's configuration file directory.
current working directory
The directory where the code is run
In the current directory, matplotlib configuration items can be customized for the current project code contained in the directory. The file name of the configuration file is matplotlibrc
In Windows systems, there is no global configuration file, and the location of the user configuration file is C:\Documents and Settings\yourname\.matplotlib.
In Linux systems, the global configuration file is located in /etc/matplotlibrc, and the user configuration file is located in $HOME/.matplotlib/matplotlibrc.
Dynamic configuration
Configuration code in the program
To finetune settings only for that execution; this overwrites every configuration file.
The priority of configuration methods is:
Matplotlib functions in Python code
matplotlibrc file in the current directory
User matplotlibrc file
Global matplotlibrc file
rcParams method
Access and modify all loaded configuration items through the rcParams dictionary
A simple example:
import matplotlib.pyplot as plt
import numpy as np

# **coding=utf-8

x = np.array([-0.5, -1.5, -2, -3.5,-5,-6,-6.2,-6.25,-6.3,-6.35,   0.2,0.4,0.6,0.7,0.72,0.74,0.76,0.78,0.8,0.82])
y = np.array([-0.1,-0.5,-0.6,-1,-1.5,-1.75,-2,-6,-46.5,-74.5,    0,0.45,0.5,1,1.6,3.25,5.5,8,15.4,35.5])

z1 = np.polyfit(x, y,1)  # 用3次多项式拟合

p1 = np.poly1d(z1)

print(p1)  # 在屏幕上打印拟合多项式

yvals = p1(x)  # 也可以使用yvals=np.polyval(z1,x)

plot1 = plt.plot(x, y, '*', label='original')
plot2 = plt.plot(x, yvals, 'r', label='ployfit')

plt.xlabel(u'X')
plt.ylabel(u'Y')
plt.legend(loc=4)  # 指定legend的位置,读者可以自己help它的用法
plt.title('UnLiner R Curve ')
plt.grid()
plt.show()
