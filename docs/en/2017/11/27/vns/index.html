Variable Neighborhood Search (VNS)

TechnologyMachine LearningEvolutionary Algorithms
Let’s talk about local search first: Local search is a heuristic algorithm for solving optimization problems. For some optimization problems that are very complex to calculate, such as various NP-complete problems, the time required to find the optimal solution increases exponentially with the size of the problem. Therefore, various heuristic algorithms were born to find sub-optimal solutions. This is an approximate algorithm (Approximate algorithms) that trades time for accuracy. Local search is one of these methods.The main idea of ​​the variable neighborhood search algorithm is:Systematic search using multiple different neighborhoods. First use the smallest neighborhood search, and when the solution cannot be improved, switch to a slightly larger neighborhood. If the solution can continue to be improved, return to the smallest neighborhood, otherwise continue to switch to a larger neighborhood. The characteristic of variable neighborhood search is to use neighborhood structures composed of different actions to conduct alternate searches, achieving a good balance between centralization and dispersion.
