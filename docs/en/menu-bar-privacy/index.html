Menu Bar Privacy Policy

Menu Bar Privacy Policy

Last update: 18 May 2026

The use of Menu Bar (hereinafter referred to as “this application”) is welcome.

The application attaches great importance to user privacy and data security. This privacy policy is intended to explain how this application collects, uses, stores and protects your information.

Use this application to indicate your agreement with what is in this privacy policy.

Information-gathering instructions

Menu Bar designed to be a locally operated lightweight system tool.

This application default does not proactively collect, upload or share your personal identity information.

II. Information on visits to Clipboard

Based on user-enabled functionality, this application may read the system's clipboard contents for:

  • Show Recent Copy History
  • Provides quick copying
  • View and manage clipboard records locally

Key note:

  • Clipboard data only processed locally for user devices
  • No upload to developer server
  • Not for advertising, analysis or user image
  • User can shut down the clipboard listening at any time

System information access

This application may read information on the operational status of some of the equipment, including but not limited to:

  • CPU Usage
  • Memory occupation
  • Web upload/download speed
  • Battery status
  • System Time

The above information will be used only for:

  • Show in real time in PiP (painted) or suspended components
  • Provide equipment status monitoring to users

This information will not be associated with the user identity and will not be uploaded to external servers.

IV. Backstage run and painting (PiP)

This application may use iOS / iPados to provide:

  • Backstage audio capability
  • Picture in Picture

is used to perform suspended display functions.

The function is used only for the interface display and does not record audio, video or screen content.

V. Local data storage

To enhance user experience, this application may be stored locally:

  • User Settings
  • PiP Layout Configuration
  • Custom Template
  • Clipboard History
  • Interface preferences

The data are only stored in the user device by default.

Users can remove relevant data through unmounting applications.

Data sharing and third-party services

This application default does not sell, share or transmit user personal data to a third party.

This privacy policy will be updated simultaneously if third-party services are to be accessed in the future (e.g. crash analysis, statistical analysis, etc.).

Children's privacy

The application does not proactively collect personal information on children.

VIII. Update on privacy policy

We may make changes to this privacy policy based on functional updates or legislative requirements.

An updated version will be published in the application or on the relevant page.

Contact us.

If you have any questions, suggestions or comments on this privacy policy, you can contact the developers by:

Mailbox: @gmail.com
