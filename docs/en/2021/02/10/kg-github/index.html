Knowledge graph application Github repository (long-term updates)

TechnologyMachine LearningKnowledge GraphsNatural Language ProcessingNatural Language ProcessingKnowledge Graphs

Knowledge graph visualization demo

Github: https://github.com/xyjigsaw/Knowledge-Graph-And-Visualization-DemoThis project supports 2D and 3D knowledge graph visualization and query, mainly JavaScript code.Detailed description：https://omegaxyz.com/2020/03/03/knowledge-graph-demo/

Aminer academic social network data knowledge graph construction

Github: https://github.com/xyjigsaw/Aminer2KGThis project extracted the Aminer scholar social network data set, generated entity relationship csv files, and used Pytorch-BigGraph to represent the knowledge graph.Detailed description：https://omegaxyz.com/2020/07/12/aminer-academic-social-network/

Demo of question and answer system based on knowledge graph

Github: https://github.com/xyjigsaw/COVID19-KBQA-DEMOA question answering system based on knowledge graph, namely KBQA. The implementation method is to convert the natural language questions input by the user into relational queries in the graph database, and finally present the entities and relationships in the database to the user.Detailed description：https://omegaxyz.com/2020/04/28/kbqa-demo/

Academic interviews (very large projects, private messages if you have questions)

Github: https://github.com/xyjigsaw/AcaFinderAcademic social network data query and visualization system based on knowledge graph.

COVID-19 patient trajectory map

Github: https://github.com/xyjigsaw/COVID-19-traces-Knowledge-GraphThe data comes from Tencent News’ Covid-19 patient tracking data. This project is based on Scrapy, PhantomJS and HanLP. HanLP is used to identify entities (location, time, action, etc.) and extract events.

PDF paper to knowledge graph (under development)

Github: https://github.com/xyjigsaw/paper2kgThis project extracts basic information and relationship relationships in PDF papers to generate paper knowledge graphs.

Linux knowledge graph

Github: https://github.com/xyjigsaw/Linux-Knowledge-GraphThis project extracts the basic primitives of Linux and generates a Linux relationship diagram.Detailed description:https://omegaxyz.com/2020/08/04/linux-knowledge-graph/

Subject area ontology relationship data and visualization

Github: https://github.com/xyjigsaw/academiaOrganize and classify the first-level subjects and second-level subjects of human society, and generate a visual network of subjects.Detailed description:https://omegaxyz.com/2020/07/13/domain_data_visualization/
