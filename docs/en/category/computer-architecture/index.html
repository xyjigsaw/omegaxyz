Hamming code assembly language implementation

Author: Xu Yi, Qiu Kang Introduction to Hamming code The performance of Hamming code is to detect one bit error. It is developed based on the idea of ​​parity check code. It adds r check bits to K bits of data to form a new codeword of K+r bits, and distributes each binary bit of the data to several different parity check groups. In this way, when one of the bits changes, it will cause parity errors in several related groups.
