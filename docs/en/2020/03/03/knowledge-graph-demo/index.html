Knowledge graph visualization demo

Programming LanguagesTechnologyWebJavaScript&HtmlMachine LearningPythonDatabasesKnowledge GraphsWebJavaScriptPythonDatabasesGithub社交网络Knowledge GraphsNeo4jDjango

Introduction

This project supports 2D and 3D knowledge graph query and visualization. Knowledge graph data set Import2Neo4j folder.For complete code and configuration instructions, please visit Github:https://github.com/xyjigsaw/Knowledge-Graph-And-Visualization-Demo

Neo4j

  • Database name: COVID-19
  • Username: neo4j (default username)
  • Password: 123456

Import2Neo4j

1. Place the CSV file into the Import folder of the corresponding Neo4j database 2. CYPHER4csvImport: Enter 8 statements in the input file in the Neo4j console. After importing you can see:
  • Node Label: (EVENT, LOCATION, PATIENT, TOPIC)
  • Relationship Types: (EVENT_LOCATION, EVENT_TOPIC, PATIENT_EVENT, PATIENT_LOCATION)

Neo4j-3D

(Pure js, run directly after Neo4j configuration)at index.htmlConfigure the server address (no port number required), Neo4j username and password in the first box The second box is the CYPHER statement, which limits the number of returns to 20,000. A search box can be added later.

KG-Search

Required Python packages
  • django
  • py2neo
  • neo4j
  • cpca (Chinese place name package)
  • jieba
Support multi-keyword query Run:
python manage.py runserver
Address: local port 8000 Preview image at top.For complete code and configuration instructions, please visit Github:https://github.com/xyjigsaw/Knowledge-Graph-And-Visualization-Demo
