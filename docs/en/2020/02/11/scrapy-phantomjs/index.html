scrapy selenium phantomJS dynamic web crawler

Programming LanguagesTechnologyPythonPython
At present, phantomJS has stopped updating. The latest selenium package also issues a warning when calling phantomJS, saying that phantomJS is no longer supported. You can use the headless parameter to call other browsers to replace phantomJS. But this hidden browser is really easy to use, and it also supports Windows, Linux, and macOS. The following is a general middleware that enables scrapy to call phantomJS to implement dynamic web crawlers.

middlewares.py

where js is the operation to be performed in the browser.
from scrapy import signals
from selenium import webdriver
from scrapy.http import HtmlResponse
import time

class JSMiddleware(object):
    def process_request(self, request, spider):
        driver = webdriver.PhantomJS("/Users/reacubeth/Downloads/phantomjs-2.1.1-macosx/bin/phantomjs")  # 指定使用的浏览器
        driver.get(request.url)
        time.sleep(1)
        js = "var q=document.documentElement.scrollTop=10000"
        driver.execute_script(js)  # 可执行js，模仿用户操作。此处为将页面拉至最底端。
        body = driver.page_source
        print("访问" + request.url)
        return HtmlResponse(driver.current_url, body=body, encoding='utf-8', request=request)
js = "var q=document.documentElement.scrollTop=10000"This purpose is to slide the page to the bottom of the browser If the page is loaded dynamically, that is, the page continues to grow longer and has no bottom (such as Zhihu), you can use the following code (250 is a preset value):
js = "window.scrollTo(0,document.body.scrollHeight)"
for i in range(250):
    driver.execute_script(js)
    time.sleep(random.randint(1, 3))
