Introduction to visualization graph layout algorithm

TechnologyMachine LearningKnowledge Graphs社交网络

Fruchterman Reingold (FR)

The FR algorithm treats all nodes as electrons, and each node receives the action of two forces: 1. Coulomb force (repulsion) of other nodes \(f_{a}(d)=\frac{d^{2}}{k}\) 2. Edge-to-point Hooker’s force (gravity). \(f_{r}(d)=\frac{-k^{2}}{d}\)The algorithm follows two simple principles:Nodes connected by edges should be close to each other; nodes should not be too close to each other. The FR algorithm is based on particle physics theory, simulating the nodes in the graph as atoms, and calculating the positional relationship between nodes by simulating the force field between atoms. The algorithm calculates the speed and acceleration of nodes by considering the interaction of attraction and repulsion between atoms. According to the movement laws similar to atoms or planets, the system eventually enters a state of dynamic equilibrium.

OpenOrd

OpenOrd is an open source algorithm for large-scale graph layout, from the paper OpenOrd: An Open-Source Toolbox for Large Graph Layout. This is a force-directed layout algorithm designed to handle very large graph structures.The left (a) shows the Swiss roll data set, which is composed of 20,000 points randomly selected from the Swiss roll copy manifest19,30. Middle (b) shows a force-directed layout obtained from Swiss roll using the VxOrd 20 Nearest Neighbors algorithm. Only nodes are plotted, and the graph is colored to illustrate the global structure of entanglement tangling. The correctly drawn graph structure is shown on the right (c), generated using the multilevel version of OpenOrd.For details, please refer tohttps://cloud.tencent.com/developer/article/1483027
