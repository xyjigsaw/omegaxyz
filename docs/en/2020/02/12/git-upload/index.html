git uploads local files to the warehouse (can be appended)

技术域转载Github
1、(Enter the folder where the file (folder) you need to upload is located)Use the command git init to turn this directory into a warehouse that git can manage.
git init
2. Add the file to the repository and use the command git add. Add it to the temporary storage area. Don’t forget the decimal point “.” at the end, which means to add all the files in the folder.
git add .
3. Use the command git commit to tell Git to submit the file to the warehouse. Submission instructions are enclosed in quotation marks
git commit -m '提交文件'
4. Connect to the remote repository (URL of your remote repository)
git remote add origin https://github.com/xxx/xxx
5. Get the remote library and local synchronization merge (This step must be done if the remote library is not empty, otherwise subsequent submissions will fail.） 
git pull --rebase origin master
6. Push the contents of the local library to the remote using the git push command, which actually pushes the current branch master to the remote. After executing this command, you will be asked to enter your username and password. Once the verification is passed, the upload will begin.
git push -u origin master
7. Status query command
git status
