Linux non-root users deploy jdk and other commands

高级语言技术域JAVA操作系统JAVALinux

step

1. Download the jdk Linux installation package from the Oracle official website to the user folder. You can use commands to download or ftp transfer. 2. Extract the jdk compressed package to the specified folder, and remember the path of the folder.
tar -zxvf jdk-linux-x64.tar.gz
3. Open the .bash_profile or .profile file
vi .bash_profile
4. Add environment variables at the end of the file
JAVA_HOME="${HOME}/jdk1.8.0"
CLASSPATH=$JAVA_HOME/lib:$JAVA_HOME/jre/lib 
PATH=$PATH:$JAVA_HOME/bin:$JAVA_HOME/jre/bin 
export PATH CLASSPATH JAVA_HOME
Among them, ${HOME} is your user folder, and / is followed by the relative path of the folder after jdk is decompressed. (Only the first line needs to be modified) 5. The source command re-executes the newly modified initialization file to make it effective immediately.
source ~/.bash_profile
6. Restart the terminal
Similar operations are also used by other commands, such as npm's node.js After downloading, add environment variables
export PATH="${PATH}:${HOME}/nodejs/bin"
That’s it.
