Fast KNN code based on leave-one-out method

Programming LanguagesTechnologyMATLABMachine LearningMATLAB特征选择

Leave-one-out cross-validation (LOOCV)

Leave-One-Out Cross Validation. This method is relatively simple and easy to understand. It is to divide a large data set into k small data sets, of which k-1 are used as training sets, and the remaining one is used as a test set. Then the next one is selected as a test set, and the remaining k-1 are used as a training set, and so on. Its main purpose is to prevent overfitting and evaluate the generalization ability of the model. The calculation time is longer.Applicable scenarios:The data set is small. If you divide the training set and the verification set for training as normal, then the data that can be used for training is originally small, and a part of it is divided, so that there is even less data that can be used for training. loocv can make full use of data.

Fast leave-one-out KNN

Because LOOCV needs to be divided N times to generate N batches of data, N models must be trained in one round of training, which greatly increases the training time. In order to solve this problem, according to the characteristics of the leave-one-out method, we can calculate the distance (or the intermediate value of the distance) between different samples in advance and store it. When using LOOCV, just take it out directly from the index. The following code uses feature selection as a demo to verify the fast KNN leave-one-out method. Among them, FSKNN1 is ordinary KNN and FSKNN2 is fast KNN.Main function main.m

Simple KNN FSKNN1.m 
function [error,fs] = FSKNN1(x,train_F,train_L)
global choice
inmodel = x>choice;%%%%%设定恰当的阈值选择特征


k=1;
train_f=train_F(:,inmodel);
train_length = size(train_F,1);
flag = logical(ones(train_length,1));


error=0;
for j=1:train_length
    flag(j) = 0;
    CtrainF = train_f(flag,:);
    CtrainL = train_L(flag);
    CtestF = train_f(~flag,:);
    CtestL = train_L(~flag);
    classifyresult= KNN1(CtestF,CtrainF,CtrainL,k);
    if (CtestL~=classifyresult)
        error=error+1;
    end

    flag(j) = 1;
end
  error=error/train_length;
  fs = sum(inmodel);

end
KNN1.m 
function relustLabel = KNN1(inx,data,labels,k)
%% 
%   inx 为 输入测试数据，data为样本数据，labels为样本标签 k值自定1~3
%%

[datarow , datacol] = size(data);
diffMat = repmat(inx,[datarow,1]) - data ;
distanceMat = sqrt(sum(diffMat.^2,2));
[B , IX] = sort(distanceMat,'ascend');
len = min(k,length(B));
relustLabel = mode(labels(IX(1:len)));

end
Fast KNN preKNN.m 
function [knnIndex] = preKNN(x,train_F)
    inmodel = x > 0;
    train_f=train_F(:,inmodel);
    train_length = size(train_F,1);
    flag = logical(ones(train_length,1));
    knnIndex = cell(train_length,1);
    for j=1:train_length
        flag(j) = 0;
        CtrainF = train_f(flag,:);
        CtestF = train_f(~flag,:);
        [datarow , ~] = size(CtrainF);
        diffMat = repmat(CtestF,[datarow,1]) - CtrainF ;
        diffMat = diffMat.^2;
        knnIndex{j,1} = diffMat;
        flag(j) = 1;
    end
end

  error=error/train_length;
