Personal home page information extractor

Programming LanguagesTechnologyMachine LearningPythonDeep LearningNatural Language ProcessingPython神经网络Natural Language Processing

Introduction

Extracts information from a researcher's homepage (HTML) and automatically categorizes the information into three categories (you can add more categories). Supports Chinese and English pages. Categories that can be divided into:
  • publication
  • education
  • honor

detailed

The process of extracting and classifying academic behaviors from Internet text data is shown in the figure below. The entire process is linear. Before formally extracting academic behaviors, a small amount of academic behaviors on the Internet are first manually annotated. After generating a training set, fastText is used to train the generated model and save it.Next, the HTML metadata is obtained through a Python crawler, and the HTML data is passed into the web page text extraction algorithm WNBTE to obtain the text. The text extraction algorithm determines the location of the text by counting the ratio of the number of text words in different HTML tags, and can effectively remove redundant and irrelevant HTML tags. Then the text paragraphs are segmented at the phrase level, and then passed into the fastText neural network after word segmentation. The trained data model is used to classify the phrases and label them.

Project structure

  • ff_classifier: Use fasttext to automatically train and predict academic behavior (training time is less than 1s)
  • text_toolkit.py: Extract strings with a certain pattern, such as email, mobile phone number, time, etc.
  • profile_class.py: researcher class
  • html_extract.py: Extract HTML elements
  • demo.py: demo
