User identity linking method - DeepLink

TechnologyMachine Learning社交网络
Paper: DeepLink: A Deep Learning Approach for User Identity Linkage UIL (User Identity Linkage): User identity link, which uses methods to discover the same user or entity across social platforms. That is, same object recognition across social platforms, UIL plays a great role in user behavior prediction, identity recognition and privacy protection.

Summary

Currently, in order to handle user identity links, a learning framework based on user introduction (portrait), user content generation and user behavior is adopted that relies on the extraction of relevant features from multiple social platforms. The problem is that these features are based on prior knowledge and vary greatly between different platforms and applications. Therefore, the author in the paper uses a deep neural network to automatically extract and represent UIL features, which is DeepLink. DeepLink represents network nodes as vectors by sampling the network in order to better obtain the local and global structure of the network. These structures, in turn, enable alignment of anchors through deep neural networks (note that fully aligned networks almost do not exist in the real world, so we utilize partially aligned anchor nodes (labeled data) to map the user latent space of two SNGs.). Utilizing the binary learning paradigm, gradient learning is used for knowledge transfer and association updating.keywords：user identity linkage, social networks, deep learning, reinforcement learning

What does DeepLink need to gain?

DeepLink needs to obtain this function φ_u. However, since each social network graph (SNG) is unknown to other SNGs, this mapping function is difficult to obtain. Therefore, Graph Mapping Function is used to approximate φ_u in most UIL implementations.

UIL problem classification

Feature-based methods:Utilize domain knowledge and user activities to deeply understand artificial feature engineering.Web-based approach:Using network structure to describe user associations across social platforms has recently attracted widespread attention and research.

Limitations of current methods

No comprehensive framework is provided to address the heterogeneity of users and different social network platforms (which are independent). User representation: It is difficult to capture the underlying semantic relationships between users based on network structure, and many existing works focus on cross-platform behavior prediction and association based on transfer learning. Lack of labeled data: Obtaining a set of users with the same identity across platforms is not easy.

Algorithm overview

① Use random walk strategy to generate node sequence samples to construct a corpus. ② In the initialization step, each node is embedded and represented in a low-dimensional latent space, and two mapping functions are pre-trained from G→G′ using anchor points. ③ Use policy gradient-based methods to learn cross-network node alignment based on aligned anchor node pairs in a supervised manner (using known alignments).

DeepLink’s contribution

DeepLink utilizes deep neural networks to learn the latent semantics of user activities and network structures in an end-to-end manner, thereby avoiding labor-intensive feature engineering and being easily generalized to various OSNs. DeepLink utilizes semi-supervised graph regularization to predict the context (neighborhood structure) of nodes in the network. Encode the structural information of anchor nodes to achieve network alignment. DeepLink uses a dual learning process to improve identity linking performance and improves the supervised training algorithm. This not only exploits unlabeled anchor nodes but also improves the mapping between multiple networks through a reinforcement learning process. Additionally, the number of required alignment (marker) anchor nodes can be significantly reduced. To demonstrate the effectiveness of DeepLink, the authors conducted experiments on real and synthetic datasets. The results show that DeepLink can significantly improve the accuracy of identity recognition compared with state-of-the-art methods. For example, in terms of link accuracy, top-1 can reach 30% and top-5 can reach 40%.

Algorithm details

DeepLink composition

  • Network Sampling: Form "corpus" through network sampling to maintain the network structure to the greatest extent: for efficiency, a simple random walk strategy is used here to detect the network structure and generate social sequences.
  • Users Latent Space Embedding: Use network embedding (Embedding) to represent each node in the network as a vector in a low-dimensional space: use the Skip-gram model to update the social representation (maximizing log-likelihood estimation).
  • Unsupervised Automapping: Input anchor point vectors into a deep neural network to train a nonlinear transformation of cross-network user alignment: After obtaining the potential embedding space of each social network, DeepLink uses two multi-layer perceptrons (MLP) to learn the mapping function in any two anchor-based social networks.
  • Supervised Linkage Dual Learning: Use the dual learning process to improve UIL performance and improve supervised training algorithms.
