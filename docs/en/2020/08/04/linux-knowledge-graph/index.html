Linux knowledge graph

TechnologyMachine Learning操作系统Knowledge GraphsLinuxKnowledge Graphs
The Linux knowledge graph based on Triples and Neo4j currently includes 401 entities and 377 relationships, and more data will be added in the future.Project address：https://github.com/xyjigsaw/Linux-Knowledge-Graph

Folder 📂Linux KG

These folders contain entities and relationships for Linux stored in csv. The raw data is located in LinuxKG.xlsx.
file name describe quantity
e_LinuxDir.csv Linux directory 34
e_system.csv System commands 54
e_device.csv device command 14
e_dir.csv Directory commands 13
e_file.csv file command 40
e_network.csv network commands 38
e_software.csv software commands 4
e_tool.csv Tool commands 4
r_linux.csv relation 182

Ops KG

This folder contains entities and relationships for Linux operations and maintenance stored in csv. The raw data is in opKG.xlsx.
file name describe quantity
e_ops.csv Operation and maintenance entity 200
r_ops.csv relation 195

Import into Neo4j

Move all .csv files to the Neo4j import directory. Then execute the CYPHER code in the Neo4j console.
