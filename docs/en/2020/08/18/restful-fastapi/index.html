RESTful-based FastAPI service template

Programming LanguagesTechnologyWebPythonWebPython
FastAPI is a high-performance web framework and a Python package used to build APIs. It is suitable for building server backends with very little code and achieving front-end and back-end separation.
RESTful API is a REST style API. Nowadays, there are various terminal platforms. After mobile, tablet, PC and many other media send requests to the server, if the RESTful API is not applicable, the corresponding return format needs to be defined for the data request of each platform to adapt to the front-end display. However, RESTful API requires the front-end to send requests in a predefined syntax format, so the server only needs to define a unified response interface and does not have to parse various requests as before.
Common API methods
Python packages required: fastapi and uvicorn.
Code: apiCore.py 
import uvicorn
from fastapi import FastAPI, Query, Form, APIRouter, File, UploadFile
from fastapi.middleware.cors import CORSMiddleware
import time


app = FastAPI(
    title="demo",
    docs_url='/api/v1/docs',
    redoc_url='/api/v1/redoc',
    openapi_url='/api/v1/openapi.json'
)

router = APIRouter()


@router.get('/paper')
async def fetch_paper(
    num: int = Query(..., description='returned paper num', example='10')
):
    start = time.time()
    print(num)
    return {'time': time.time() - start, 'data': num}


@router.post('/add_paper')
async def add_paper(
        name: str = Form(..., description='paper name', example='Attention is all you need'),
        info: str = Form(..., description='paper info', example='NIPS 2017')
):
    start = time.time()
    print(name, info)
    return {'time': time.time() - start}


@router.put('/update_paper')
async def update_paper(
        p_id: str = Form(..., description='paper id', example='1234'),
):
    start = time.time()
    print(p_id)
    return {'time': time.time() - start}


@router.delete('/delete_paper')
async def delete_paper(
        p_id: str = Query(..., description='paper is', example='1234')
):
    start = time.time()
    print(p_id)
    return {'time': time.time() - start}



app.include_router(router)

app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)

if __name__ == '__main__':
    uvicorn.run(app=app, host="127.0.0.1", port=8000, workers=1)
Console startup (under LAN): 
uvicorn apiCore:app --reload --port 9918 --host 0.0.0.0
