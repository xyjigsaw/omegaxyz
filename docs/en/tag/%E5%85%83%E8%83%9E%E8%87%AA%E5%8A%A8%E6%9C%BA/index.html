What is a cellular automaton? Cellular automata (CA) is a grid dynamics model in which time, space, and state are discrete, and spatial interaction and temporal causality are local. It has the ability to simulate the spatiotemporal evolution process of complex systems. It can build systems where state transitions occur over time, where cells exist in a one- or multi-dimensional grid, each cell has one or more states, and each cell has neighbors (i.e. neighboring cells).
