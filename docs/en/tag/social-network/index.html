FR algorithm (Fruchterman-Reingold) Python implementation

Introduction The FR algorithm treats all nodes as electrons, and each node receives the action of two forces: 1. Coulomb force (repulsion) from other nodes 2. Edge-to-point Hooker force (gravity). The algorithm follows two simple principles: nodes connected by edges should be close to each other; nodes should not be too close to each other. The FR algorithm is based on particle physics theory, simulating the nodes in the graph as atoms, and calculating the positional relationship between nodes by simulating the force field between atoms.
