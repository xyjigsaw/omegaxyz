Application of binary search algorithm——LeetCode1011

Binary search, also known as half search, is a more efficient search method. Principle: Divide the array into three parts, in order: before the median value (the so-called median value is the value in the middle of the array), the median value, and after the median value; compare the value to be found with the median value of the array. If it is less than the median value, it will be searched in front of the median value. If it is greater than the median value, it will be searched after the median value.
