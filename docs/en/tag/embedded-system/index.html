C language determines processor big endian/little endian mode

Write a C program to determine the endianness mode of a processor. If the processor is in big-endian mode, output on the console: The processor is in big-endian mode; otherwise, output, the processor is in little-endian mode. What is big endian/little endian mode? Big endian mode. The so-called big endian mode means that the high byte of data is stored in the low address of the memory, and the low byte of the data is stored in the high address of the memory.
