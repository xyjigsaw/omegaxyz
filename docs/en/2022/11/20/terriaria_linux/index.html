Terraria Terriaria quickly deploys Linux servers

Ideas随笔LinuxGamesTerraria
Terraria has been updated frequently recently (it has to be redeployed every time it is launched!!!), and the Linux server of the multiplayer game also needs to be updated simultaneously, but I can’t remember the download address of the new version: (. Record it here.

Download address

https://terraria.org/api/download/pc-dedicated-server/terraria-server-1447.zip The last 1447 is the version number. If it is version 1.4.4.8.1, change it to 14481

Deployment steps

unzip terraria-server-1447.zip
cd 1447
cd Linux

chmod +x TerrariaServer
chmod +x TerrariaServer.*
./TerrariaServer -config ./startserver.sh
