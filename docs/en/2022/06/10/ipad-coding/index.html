iPad Productivity Guide – Programming

Ideas随笔appiOS
Productivity before buying, iQiyi after buying? No! It's you who are missing this guide. This article introduces how to use iPad as a programming tool.
equipment:iPad Pro 2021 M1, 11 inchesUsage scenarios:CS programming and paper writing  

Textastic

Textastic is an advanced code editor for iPad. it supports
  • syntax highlighting
  • SSH terminal to connect to the server
  • SFTP file transfer
When using Textastic, I often get so immersed in the code that I forget that the app is running on an iPad rather than a full desktop computer interface. I'm sure coders will appreciate this amazing mobile text editor.
 

Working Copy

Git management tool: Working Copy Working Copy is a powerful iOS Git client that can perform operations such as cloning, editing, submitting, and pushing. Those with educational email addresses can use the premium version for free.Specific features:

Secure ShellFish

SFTP server management tool: Secure ShellFish This software can map remote server folders to the system's own file App, and download and modify files without opening the software. At the same time, you can use other software to modify files on the server from the File App and automatically upload them, such as Textastic.

iSH

Linux client: iSH iSH is a Linux virtual machine used to simulate the x86 architecture on ARM-based iOS devices, allowing the iOS devices to run the Linux Shell environment locally [1]. The Linux system image used by iSH is Alpine Linux, and the file system can be viewed in the File App that comes with iOS. After testing, most packages can be used, and you can even download videos from bilibili or YouTube. Python can be installed, but there are currently some bugs in numpy.iSH-File App

Code

Desktop-level programming software: Code An iPad programming software similar to VSCode, it supports git and can run Node.js, Python, PHP, C++ and other languages offline.

TablePlus

Database management software: TablePlus Many database management software require payment. If you just want to view remote data, it is recommended to use TablePlus, which is also very good-looking.  

CodeSandbox

JS editor: CodeSandbox (formerly known as play.js) Supports most front-end frameworks such as nodejs and vue

CodeServer in Safari

Set up Code Server on the remote server and bind the domain name to use it in Safari. The premise is that you need a remote server with a network connection. If it is Alibaba Cloud or Tencent Cloud, you can bind it to your own domain name. If you are using a server on the laboratory intranet, you need to use the intranet to penetrate your own public network server and then bind the domain name. You can refer to the minority configuration plan [2]. Of course, if you find Safari inconvenient, you can use a dedicated CodeServer to connect to the App—Servediter.

Servediter

AnyDesk

Remote Desktop: AnyDesk Teamviewer is nice, but it can only support one machine for personal use. AnyDesk can be used as an alternative.

Overleaf in Safari

Paper writing: Overleaf in Safari Stable and shared. You can use Tex Pro or texpad locally. The former is free and the latter is 168¥. If you give up, it is better to use Overleaf.

Paper reading

Cloud drive (iCloud or OneDrive both support Windows and Apple devices) + Zotero + DocumentZotero is used to manage documents and store PDF folders in iCloud, so that they can be opened with the built-in file app on the iPad or opened with Document to read notes. If you need to retrieve or quickly locate the PDF location, just use spotlight. Currently Zotero is also available on iPad.

All developer apps

Usage experience: With a keyboard with a touchpad, basic programming operations and workflow can be completed smoothly. However, debugging and advanced code editing functions still have to rely on desktop computers. Therefore it is only lightly productive.
