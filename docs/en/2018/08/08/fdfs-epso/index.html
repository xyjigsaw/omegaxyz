Detailed explanation of feature discretization and selection EPSO algorithm

TechnologyMachine LearningEvolutionary AlgorithmsPSO算法特征选择

Introduction to EPSO

The main idea of ​​EPSO is to use BBPSO to directly evolve a tangent point that can be any value within the corresponding eigenvalue range [MinF···MaxF]. The position of each particle represents a candidate solution, which is an n-dimensional real vector corresponding to the dimension of the problem. Figure 2 gives an example of a particle position and its corresponding candidate solution. In this example, the first dimension of the particle, representing the cut point of the first characteristic (F1), needs to have a value in the range [8.5, 25.7]. If the update point of a feature F is outside this range, it will be set to the nearest boundary.

detail

The particles (individuals) in the population are the cut-pointsNext, select a particle, which is a set of tangent points, and perform the following operations:If all the values ​​of a feature are set to 0 or 1, then it can be considered an irrelevant feature because it cannot distinguish the categories in the instance. Feature selection will remove these features. (The 2nd and 3rd dimensions of the above two examples are removed)

step

(1) Particle initialization: Due to the huge search space of multi-variable discretization on high-dimensional data. This means that for those features that were not selected among the initial candidates, their cut points will be set to the maximum value of the corresponding feature. For other selected features, their cut points are initialized using the best entropy-based cut point that satisfies the MDLP. In principle, they can be initialized based on any value within the range of the corresponding characteristic. However, a completely random initial cut point may result in slower convergence. Furthermore, the information gain of a feature’s optimal cut point is an indicator of its relevance. Therefore, features with larger information gain have a greater probability of being selected during the initialization process.
(2) Particle evaluation: Based on the cut points generated by the particles, the training data is converted into a new training set of discrete values ​​and a smaller number of features, thanks to the elimination of features whose cut points are equal to the minimum or maximum values. For example, in Figure 2, the cut point of F3 is equal to its maximum value and the cut point of F5 is equal to its minimum value, both features will be discarded.
The discretization and FS solution of each particle are then evaluated based on the classification accuracy of the transformed training set. By evaluating the entire discrete data, the proposed method can evaluate the segmentation points of all selected features while taking feature interactions into account.
