The difference between HTML5 and HTML4

Programming LanguagesTechnologyWebJavaScript&HtmlWebHTML

What is html5?

HTML5 was first named a hypertext markup language by WHATWG (Web Hypertext Application Technology Working Group), and then combined with W3C's xhtml2.0 (standard) to produce the latest generation of hypertext markup language. It can be simply understood as: HTML 5 ≈ HTML+CSS 3+JS+API.

The difference between hmtl5 and html

The static web pages we currently develop on the web front-end are generally HTML4.0. At the same time, it complies with the W3C's xhtml1.0 specification. So what are the substantial differences between them?

1. On the document type declaration

html:<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd"> <html xmlns="http://www.w3.org/1999/xhtml"> html5:<!doctype html> From the comparison between the two, it can be seen that in the document declaration, html has a long code, and it is difficult to remember this code. Presumably many people use tools to generate it directly, right? But html5 is different, there are only simple declarations, which is also convenient for people to remember.

2. In terms of structural semantics

html: There is no tag that reflects the structural semantics. We usually name it like this <div id="header"></div>. This represents the head of the website. html5: It has great advantages in semantics. Some new tags are provided, such as: <header><article><footer>

3. Label

  • <canvas> tag replaces Flash
  • Added <header> and <footer> tags, semantic tags, for search engines.
  • Adding new <section> and <article> tags will help clarify the structure of the web page and be more conducive to SEO.
  • Added <menu> and <figure> tags
  • Added <audio> and <video> tags, which are used to play audio and video.
  • Brand new form
  • Remove <b> and <font> tags
  • Remove <frame>, <center>, <big> tags
