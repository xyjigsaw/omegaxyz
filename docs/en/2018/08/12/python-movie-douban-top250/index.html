Python crawls Douban movie rankings

Programming LanguagesTechnologyPythonPython
Last year, there was an article crawling recently released movies from Douban.com, the URL: https://omegaxyz.com/2017/11/09/python_douban/?hilite=%27%E7%94%B5%E5%BD%B1%27 Nowadays, I am mainly studying some algorithms of machine learning. The Python code has basically been discontinued. In order not to be unfamiliar with Python, I looked for some simple Python codes from CSDN to practice. The following is the code for Python to crawl the top 250 Douban movie rankings, and save it in Excel as a csv file. The modules used are requests and bs4.Code: 
import requests
import re
import os
from bs4 import BeautifulSoup


def download(url, page):
    html = requests.get(url).text   # 这里不加text返回<Response [200]>
    soup = BeautifulSoup(html, 'lxml')
    lis = soup.select("ol li")
    for li in lis:
        index = li.select_one("em").text
        title = li.select_one(".hd .title").text
        rating = li.select_one(".bd .star .rating_num").text
        strInfo = re.search("(?<=<br/>).*?(?=<)", str(li.select_one(".bd p")), re.S | re.M).group().strip()
        infos = strInfo.split('/')
        year = infos[0].strip()
        area = infos[1].strip()
        type = infos[2].strip()
        write_fo_file(index, title, rating, year, area, type)
    page += 25
    if page < 250:
        download("https://movie.douban.com/top250?start=%d&filter=" % page, page)


def write_fo_file(index, title, rating, year, area, type):
    f = open('douban_top250.csv', 'a')
    f.write("%s,%s,%s,%s,%s,%s\n" % (index, title, rating, year, area, type))
    f.close()


def main():
    if os.path.exists('douban_top250.csv'):
        os.remove('douban_top250.csv')

    url = 'https://movie.douban.com/top250'
    download(url, 0)
    print("爬取完毕。")


if __name__ == '__main__':
    main()
