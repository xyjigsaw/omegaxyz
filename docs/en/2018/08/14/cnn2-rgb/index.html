Convolution of color images

The convolutions described above are all for grayscale images. If you want to perform convolution on an RGB image, the size of the filter is no longer 3*3 but 3*3*3. The last 3 corresponds to the number of channels (channels). Each pixel value in the convolution generated image is 3*3*3. The corresponding position of the filter and the corresponding position of the image are multiplied and accumulated. The filter slides on the RGB image in turn, and the final generated image size is 4*4.Another problem is that if we not only detect one type of feature in the image, but also detect vertical edges, horizontal edges, 45-degree edges, etc., it is a problem of multiple filters. If there are two filters, the final generated image is a 4*4*2 cube. The 2 here comes from the fact that we use two filters. If there are 10 filters then the output image will be a 4*4*10 cube.

Single layer convolutional network

Through the description in the previous section, the image obtained two 4*4 matrices through two filters, and biases were added to the two matrices.b1b1andb2b2, and then perform a nonlinear Relu transformation on the matrix with the bias added to obtain a new 4*4 matrix. This is the complete calculation process of the single-layer convolutional network. Expressed by the formula:
The input image isa[0]a[0], for filterw[1]w[1]Representation, linearly changing the image and adding deviation to obtain the matrixz[1]z[1]，a[1]a[1]It is the result of applying Relu activation.
  • If there are 10 filter parameters, how many are there?Each filter has 3*3*3+1=28 parameters, 3*3*3 is the filter size, 1 is the deviation coefficient, and the number of 10 filter parameters is 28*10=280. The number of parameters will not change regardless of the size of the input image.

Pooling layer

Max pooling The idea of max pooling is very simple. Take the following figure as an example. Divide the 4*4 image into 4 different areas, and then output the maximum value of each area. This is what max pooling does. In fact, here we chose a 2*2 filter with a step size of 2. Extracting the maximum value in a real image may mean extracting some specific features, such as vertical edges, an eye, etc.The following is a pooling process with a filter size of 3*3 and a step size of 1. The specific calculation is the same as above. The calculation method of the size of the output image in the maximum pooling is the same as that in the convolutional network. If there are multiple channels that need to be pooled, then the pooling operation is calculated by channel.The only difference between average pooling and max pooling is that it calculates the average value within the region while max pooling calculates the maximum value. The most commonly used method in daily applications is max pooling.Hyperparameters of pooling: step size, filter size, pooling type max pooling or average pooling
