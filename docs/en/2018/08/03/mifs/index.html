Feature selection algorithm based on mutual information MATLAB implementation

Programming LanguagesTechnologyMATLABMachine LearningMATLAB特征选择信息论
In probability theory and information theory, the relationship between two random variablesMutual Information (MI)Or transfer information (transinformation) is a measure of interdependence between variables. Unlike the correlation coefficient, mutual information is not limited to real-valued random variables, it is more general and determines how similar the joint distribution p(X,Y) and the product of decomposed marginal distributions p(X)p(Y) are. Mutual information (Mutual Information) measures the correlation (mutual dependence) between two event sets. The most commonly used unit of mutual information is bit.

Definition of mutual information

Formally, the mutual information of two discrete random variables X and Y can be defined as:where p(x,y) is the joint probability distribution function of X and Y, and p(x) and p(y) are the marginal probability distribution functions of X and Y respectively.In the case of continuous random variables, the summation is replaced by a double definite integral:where p(x,y) is currently the joint probability density function of X and Y, and p(x) and p(y) are the marginal probability density functions of X and Y respectively. The statistical average value of mutual information I(xi;yj) in the joint probability space P(XY). The average mutual information I(X;Y) overcomes the randomness of the mutual information quantity I(xi;yj) and becomes a certain quantity. If the logarithm is based on 2, the unit of mutual information is bit. Intuitively, mutual information measures the information shared by X and Y: it measures how much knowing one of the two variables reduces uncertainty about the other. For example, if X and Y are independent of each other, then knowing that X provides no information about Y and vice versa, their mutual information is zero. At the other extreme, if X is a deterministic function of Y, and Y is also a deterministic function of Therefore, the mutual information in this case is the same as the uncertainty contained in Y (or X) alone, called the entropy of Y (or X). Moreover, this mutual information is the same as the entropy of X and the entropy of Y. (A very special case of this is when X and Y are the same random variables.) Mutual information is the intrinsic dependence between the joint distribution of X and Y relative to the joint distribution if X and Y are assumed to be independent. Mutual information then measures dependence in the following way: I(X; Y) = 0 if and only if X and Y are independent random variables. Steps of mutual information feature selection algorithm

  • ①Divide the data set
  • ②Use mutual information to sort features
  • ③Select the first n features for training using SVM
  • ④Evaluate the feature subset and calculate the error rate on the test set

shortcoming

This feature selection method is to maximize the correlation between features and categorical variables, that is, to select the top k variables with the highest correlation with categorical variables. However, in feature selection, the combination of a single good feature cannot increase the performance of the classifier, because the features may be highly correlated, which leads to redundancy of feature variables.

code

Note that the data set used is dlbcl, which has about five thousand dimensions and can be downloaded from UCI. The top 100 features are finally selected for training.Main function code:
clear all
close all
clc;
[X_train,Y_train,X_test,Y_test] = divide_dlbcl();
Y_train(Y_train==0)=-1;
Y_test(Y_test==0)=-1;
% number of features
numF = size(X_train,2);



[ ranking , w] = mutInfFS( X_train, Y_train, numF );
k = 100; % select the Top 2 features
svmStruct = svmtrain(X_train(:,ranking(1:k)),Y_train,'showplot',true);
C = svmclassify(svmStruct,X_test(:,ranking(1:k)),'showplot',true);
err_rate = sum(Y_test~= C)/size(X_test,1); % mis-classification rate
conMat = confusionmat(Y_test,C); % the confusion matrix
fprintf('\nAccuracy: %.2f%%, Error-Rate: %.2f \n',100*(1-err_rate),err_rate);
mutInfFS.m 
function [ rank , w] = mutInfFS( X,Y,numF )
rank = [];
for i = 1:size(X,2)
    rank = [rank; -muteinf(X(:,i),Y) i];
end;
rank = sortrows(rank,1);	
w = rank(1:numF, 1);
rank = rank(1:numF, 2);

end
muteinf.m 
function info = muteinf(A, Y)
n = size(A,1);%实例数量
Z = [A Y];%所有实例的维度值及标签
if(n/10 > 20)
    nbins = 20;
else
    nbins = max(floor(n/10),10);%设置区间的个数
end;
pA = hist(A, nbins);%min(A)到max(A)划分出nbins个区间出来，求每个区间的概率
pA = pA ./ n;%除以实例数量

i = find(pA == 0);
pA(i) = 0.00001;%不能使某一区间的概率为0

od = size(Y,2);%一个维度
cl = od;
%下面是求实例不同标签的的概率值，也就是频率
if(od == 1)
    pY = [length(find(Y==+1)) length(find(Y==-1))] / n;
    cl = 2;
else
    pY = zeros(1,od);
    for i=1:od
        pY(i) = length(find(Y==+1));
    end;
    pY = pY / n;
end;
p = zeros(cl,nbins);
rx = abs(max(A) - min(A)) / nbins;%每个区间长度
for i = 1:cl
    xl = min(A);%变量的下界
    for j = 1:nbins
        if(i == 2) && (od == 1)
            interval = (xl <= Z(:,1)) & (Z(:,2) == -1);
        else
            interval = (xl <= Z(:,1)) & (Z(:,i+1) == +1);
        end;
        if(j < nbins)
            interval = interval & (Z(:,1) < xl + rx);
        end;
        %find(interval)
        p(i,j) = length(find(interval));
        
        if p(i,j) == 0 % hack!
            p(i,j) = 0.00001;
        end
        
        xl = xl + rx;
    end;
end;
HA = -sum(pA .* log(pA));%计算当前维度的信息熵
HY = -sum(pY .* log(pY));%计算标签的信息熵
pA = repmat(pA,cl,1);
pY = repmat(pY',1,nbins);
p = p ./ n;
info = sum(sum(p .* log(p ./ (pA .* pY))));
info = 2 * info ./ (HA + HY);%计算互信息
