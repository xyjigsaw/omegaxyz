Basic concepts of machine learning data sets

TechnologyMachine Learning特征选择
A data set, also known as a data set, data set or data set, is a collection composed of data. A data set (or dataset) is a collection of data, usually in tabular form. Each column represents a specific variable. Each row corresponds to a question for a certain member of the data set. It lists the values ​​for each variable, such as the height and weight of an object or the value of a random number. Each numerical value is called a data point. The data for this dataset may include one or more members corresponding to the number of rows.——Baidu EncyclopediaThe following is my personal understanding
Dataset name GLIOMA GIOMA contains two matrices, one is the instance matrix (ins) and the other is the label matrix (lab)The Ins matrix size is 50*4434, which means that the GLIOMA data set has 50 instances (samples) and 4434 features. Each of these 50 instances (samples) has a corresponding label lab, and the label is the category. Open the Ins matrix. There are 50 rows indicating that there are 50 instances (samples), and 4434 columns indicating that there are 4434 features (too many to display). Any value (scalar) here is called a feature value, any column is a feature vector (column vector), and any row is an instance vector (row vector).Open lab matrixThere are 50 labels, and the labels are categories (for example, 1 represents young children, 2 represents youth, and so on). You can see that this is a data set with 4 categories.
In addition, do not confuse instances with individuals. Instances only refer to the data set (original space), and the number of instances is generally unchanged. Individuals are regarded as populations in evolutionary calculation algorithms, and the number of individuals can be set according to your own preferences. For example, individuals in the PSO algorithm are particles. The one similarity between instances and individuals is that they have the same number of features (dimensions).
Download of the dataset (downloaded from UCI):http://archive.ics.uci.edu/ml/index.phpOf course, the downloaded data set may have labels and features put together and can be separated by yourself.
