Use the C++ library sstream to interchange string and int types

Programming LanguagesC & C++TechnologyC++
When programming in C++, we often want to convert the string type into the int type for addition and subtraction operations, or we want to convert the int type into the string type for large number operations. It will be troublesome to directly use ASCII code during the operation. Of course, this operation would be very simple in Python, just one line of cast, but not in C++. The following introduces sstream to perform type exchange: Code:
#include<iostream>
#include<string>
#include<sstream>
using namespace std;

string convert2Str(double x)
{
    ostringstream o;
    if(o << x)
        return o.str();
    return "Error";
}

double convertFromStr(const string &s)
{
    istringstream i(s);
    double x;
    if(i >> x)
        return x;
    return 0.0;
}

int main()
{
    string cc=convert2Str(2022);
    cout<<cc<<endl;
    string dd="2018";
    int p =convertFromStr(dd)+2;
    cout<<p<<endl;
}
