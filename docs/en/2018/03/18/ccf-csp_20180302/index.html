CCF CSP certified collision ball

Programming LanguagesC & C++TechnologyC++
Problem description
There is a line segment with length L (L is an even number) on the number axis. The left endpoint is at the origin and the right endpoint is at coordinate L. There are n small balls regardless of volume on the line segment. At the beginning, all the small balls are on even coordinates, the speed direction is to the right, and the speed is 1 unit length per second. When the ball reaches the endpoint of the line segment (left endpoint or right endpoint), it will immediately move in the opposite direction, and the speed will still be the original size. When two small balls collide with each other, the two small balls will continue to move in the opposite direction to their original movement and at the original speed. Now, tell you the length L of the line segment, the number of balls n, and the initial positions of the n balls. Please calculate the position of each ball after t seconds.
hint
Because the initial positions of all the balls are even numbers, and the lengths of the line segments are even numbers, it can be proved that no three balls will collide at the same time. The times when the balls reach the endpoints of the line segments and when the balls collide between them are all integers. At the same time, it can also be proved that the position where the two balls collide must be an integer (but not necessarily an even number).
Input format
The first line of input contains three integers n, L, and t, separated by spaces, which respectively represent the number of balls, the length of the line segment, and the position of the ball that you need to calculate after t seconds. The second line contains n integers a1, a2, ..., an, separated by spaces, indicating the positions of n balls at the initial moment.
Output format
The output line contains n integers, separated by spaces. The i-th integer represents the position of the ball located at ai at the initial moment after t seconds.
Sample input
3 10 5 4 6 8
Sample output
7 9 9
Sample description
　　
Sample input
10 22 30 14 12 16 6 10 2 8 20 18 4
Sample output
6 6 8 2 4 0 4 12 10 2
Data size and conventions
For all evaluation cases, 1 ≤ n ≤ 100, 1 ≤ t ≤ 100, 2 ≤ L ≤ 1000, and 0 < ai < L. L is an even number. Ensure that the initial positions of all balls are different from each other and are even numbers.
Code:
#include<iostream>
#include<vector>
using namespace std;
class Ball{
public:
    int position;
    bool lorr = true;
};

int main()
{
    int L;
    int n;
    int time;
    cin>>n>>L>>time;
    vector<Ball> vec;
    for(int i=0;i<n;i++){
        Ball temp;
        int temp_p;
        cin>>temp_p;
        temp.position=temp_p;
        vec.push_back(temp);
    }
    for(int i=0;i<time;i++){
        for(int j =0; j<vec.size();j++){
            if(vec[j].position == L){
                vec[j].lorr = false;
            }else if(vec[j].position == 0){
                vec[j].lorr = true;
            }
            for(int k=0;k<vec.size();k++){
                if(vec[j].position == vec[k].position){
                    if(vec[j].lorr==false){
                        vec[j].lorr = true;
                        vec[k].lorr = false;
                    } else{
                        vec[j].lorr = false;
                        vec[k].lorr = true;
                    }
                    break;
                }
            }
        }
        for(int j=0;j<vec.size();j++){
            if(vec[j].lorr==true){
                vec[j].position++;
            }
            else{
                vec[j].position--;
            }
        }
    }
    for(int i =0;i<vec.size();i++){
        cout<<vec[i].position<<" ";
    }
    return 0;
}
