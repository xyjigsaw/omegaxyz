Gaussian mutation

TechnologyMachine LearningEvolutionary Algorithms
Gaussian mutation is another mutation operation method that improves the local search performance of the genetic algorithm for key search areas. During mutation, the original gene value is replaced with a random number from a normal distribution with mean μ and variance σ. The operation process is similar to uniform mutation. Gaussian mutation operatorThe so-called Gaussian mutation operation refers to replacing the original gene value with a random number that conforms to the normal distribution with mean and variance when performing mutation operation. It can be seen from the characteristics of normal distribution that Gaussian mutation also focuses on searching a local area near the original individual. Gaussian mutation consists of adding a random value from a Gaussian distribution to each element of an individual's vector to create a new offspring. Gaussian mutation has good local search ability, but its ability to guide individuals to jump out of local optimal solutions is weak, which is not conducive to global convergence.
