Use GithubPage to build a static blog

技术域Web开发webHTMLGithub
gitHub is a hosting platform for open source and private software projects. Because it only supports git as the only version library format for hosting, it is named gitHub. gitHub was officially launched on April 10, 2008. In addition to git code warehouse hosting and basic Web management interface, it also provides subscription, discussion groups, text rendering, online file editor, collaboration map (report), code snippet sharing (Gist) and other functions. Currently, it has more than 3.5 million registered users and a large number of hosted versions, including well-known open source projects Ruby on Rails, jQuery, python, etc. githubpage is one of its functions ①Apply for a github accounthttps://github.com/②First, we need to create a new Repositories. Repositories are equivalent to a library to store our project files.③Enter the settings page, find the Github page, select master branch, set the domain name to your own resolved domain name such as en.omegaxyz.com, and the record value is your github username.github.io Finally, upload your own HTML files and dependencies to this library. Note that there must be an index.html file. Please see my two github pages: en.omegaxyz.com and movie.omegaxyz.com
