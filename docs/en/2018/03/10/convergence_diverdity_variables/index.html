The difference between convergence variables and diversity variables

TechnologyMachine LearningEvolutionary Algorithms特征选择
Let’s first introduce the Pareto solution
Pareto optimal solution: If x*∈C*, and there is no solution x that is superior to x in C, then x* is said to be the Pareto optimal solution of the multi-objective optimization model, also known as the effective solution. Generally speaking, there is no optimal solution to a multi-objective optimization problem. All possible solutions are called non-inferior solutions, also called Pareto solutions. Traditional optimization technology can generally obtain one Pareto solution set at a time, but using intelligent algorithms to solve the problem can obtain more Pareto solutions. These solutions form an optimal solution set, which is called the Pareto optimal solution. It is a set of solutions in which the improvement of any objective function value must be at the expense of other objective function values. It is called the Pareto optimal domain, or Pareto set for short. The non-inferior solution set of Pareto efficient (optimal) solutions refers to a set of solutions that have at least one objective function that is better than any solution outside the set.
Convergence variables: The solution set values ​​generated by the perturbed decision variables are mutually dominated.
Diversity variables: The solution generated by the perturbed decision variables is non-dominated
Optimizing the convergence variable can move the Pareto forward.
Optimizing diversity variables can make Pareto spread to both sides.
