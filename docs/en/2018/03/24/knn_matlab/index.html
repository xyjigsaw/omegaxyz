KNN algorithm MATLAB implementation

Programming LanguagesTechnologyMATLABMachine LearningMATLAB

1. Concept description of kNN

Please refer to:https://omegaxyz.com/2018/01/08/knn/The kNN algorithm is also called the k-nearest neighbor classification algorithm. The so-called k nearest neighbor refers to the closest k neighbors (data), that is, each sample can be expressed by its K neighbors. The core idea of ​​the kNN algorithm is that in a space containing unknown samples, the data type of the sample can be determined based on the data types of the k samples closest to the sample. This algorithm involves 3 main factors: training set, distance and similarity measurement, and the size of k; Main considerations: distance and similarity;

2. Examples

In the picture on the right, which class should the green circle be assigned to? Is it a red triangle or a blue square? If K=3, since the proportion of the red triangle is 2/3, the green circle will be assigned to the red triangle class. If K=5, since the proportion of the blue square is 3/5, the green circle will be assigned to the blue square class.3. MATLAB implementation KNN.m: KNN label prediction, input test data, sample data, sample label, K value Output data: predicted label value for a single sample
function relustLabel = KNN(inx,data,labels,k)
%% 
%   inx 为 输入测试数据，data为样本数据，labels为样本标签
%%

[datarow , datacol] = size(data);
diffMat = repmat(inx,[datarow,1]) - data ;
distanceMat = sqrt(sum(diffMat.^2,2));
[B , IX] = sort(distanceMat,'ascend');
len = min(k,length(B));
relustLabel = mode(labels(IX(1:len)));

end
