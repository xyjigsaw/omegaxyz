Self-Evaluation Algorithm (SEE) Framework

技术域机器学习进化算法协同进化
Disadvantages of Cooperative Coevolution:CC faces the difficulty of correctly evaluating and ranking partial solutions when dealing with subproblems with interactive dependencies. The possible reason for this problem is that the variables with mutual dependence are divided into different groups, which provides wrong guidance for genetic variation.Solution:① Use an algorithm that is insensitive to the interactive dependence of variables during iterative optimization. ② Use the grouping strategy to group variables with a certain common characteristic (interaction) into one group. The idea of ​​the SEE algorithm is to group them like the CC algorithm, and then establish a predictive meta-model (meta-model) for individuals. The amount of calculation can be reduced a lot during iterative optimization.SEE algorithm framework:
