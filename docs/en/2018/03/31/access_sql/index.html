Use Microsoft Access to write SQL statements

TechnologyDatabasesDatabasesSQL
Microsoft Office Access is a relational database management system released by Microsoft. It combines the two features of MicrosoftJet Database Engine and graphical user interface, and is one of the system programs of Microsoft Office. The following describes how to use Access to write SQL statements. 1. First open the Access software and create a new blank database. (Of course, you can also open a previously saved database). 2. The software will open a blank form named [Table 1] by default. 3. Change the name of the blank form to [my], add fields and a few lines of test data.4. The default software toolbar is the [Start] toolbar. We click [Create] to enter the creation toolbar, and click [Query Design] in the toolbar.5. The [Display Table] window pops up, click [Close] to close the window. 6. At this time, the software will enter the [Design] toolbar. We click [SQL View] on the left side of the toolbar.7. [SQL View] selects [Design View] by default. We select [SQL View] in the drop-down menu. 8. A query window will automatically open below the toolbar, where you can enter the SQL statement for query.9. We enter a line of standard SQL query statement.10. After the SQL statement is executed, the query window will automatically convert to table view to display the queried data in a table.
