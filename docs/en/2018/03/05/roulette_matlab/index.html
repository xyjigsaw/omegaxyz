Roulette algorithm MATLAB implementation

Programming LanguagesTechnologyMATLABMATLAB
The roulette algorithm is a selection algorithm based on probability. This algorithm can be imagined as throwing dice into a sectored roulette wheel, and generating a set of random numbers that satisfy a uniform distribution in advance, representing n throws of dice or n dice thrown together. The roulette wheel does not move, and the area where the dice is located is the selection result. Suppose P(i), where i=1..n, is the probability of n individuals being selected, expressed as the area percentage of the occupied sector on the roulette wheel. Obviously sum(P)=1 here. select is used to save the results of n selections. MATLAB code:
D = 5;
P = [1, 1, 1, 1, 1]./D; %占比概率
m = length(P);
num = 10000;
Select = zeros(1, num);
r = rand(1, num);
for i=1:num
    sumP = 0;
    j = ceil(m*rand); %产生1~m之间的随机整数
    while sumP < r(i)
        sumP = sumP + P(mod(j-1, m)+1);
        j = j+1;
    end
    Select(i) = mod(j-2, m)+1;
end
average_S = mean(Select)
Here we use the equal-probability P matrix to select five numbers 1, 2, 3, 4, and 5. After running it ten thousand times, the average is about 3.00, which is the same as the average of this set of numbers.
