Writing and compiling C language under Linux (Ubuntu)

高级语言C&C++技术域C语言Linux
Everyone knows that running C language in Windows is very simple. Just open an IDE (VS or codeblocks) to write code and click a button to run it. In Linux, people are not very accustomed to using IDEs, and more developers choose to manually edit, compile, connect, and run. The following describes how to run C language under Linux (Ubuntu).①Open Ubuntu, create a new empty file in an empty folder, and rename it to .c fileAs shown in Figure test.c ②Open the file, edit the C language code and save it ③Return to the current folder and open the terminal ④ Next, you need to compile and connect in the terminalCompiled code:gcc test.c -o testFormat:gcc [C file name] -o [C file name without suffix] A compiled file will appear in the current folder. ⑤Run the test file in the current folder (no suffix)Run the code:./testFormat:./name So we can see the output.
