Linux math operations via shell

Technology操作系统Linux
existBash shellenvironment, you can use let, (( )) and [ ] to perform basic arithmetic operations. For advanced arithmetic operations, you can use the two tools expr and bc.

let

let operation: The let command can directly perform basic arithmetic operations. When using let, there is no need to add $.
Self-add operation: let no1++; let no+=6;
Decrement operation: let no2--; let no-=4;
self-increasing, self-decreasing

Other methods

[ ] is used similarly to let, and $; (( )) can also be used, but $ must be added before the variable name. expr can also be used in [ ]; expr can also be used for basic arithmetic operations.
The expr command reads the Expression parameter, evaluates its value, and writes the result to standard output. You must apply the following rules to the Expression parameter: Separate each item with a space. Use / (backslash) in front of shell-specific characters. Enclose strings containing spaces and other special characters in quotation marks. An integer can be preceded by a unary hyphen. Internally, integers are treated as 32-bit, double's complement numbers.

toolsbc

The above method cannot calculate floating point numbers. The precision calculation tool bc can perform floating point number operations and apply some advanced functions. The bc tool can also be used to set up decimal racing, base conversion and calculation of squares and sums of squares.
