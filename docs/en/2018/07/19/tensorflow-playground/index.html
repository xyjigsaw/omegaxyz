Neural Network (TensorFlow) Playground

TechnologyMachine LearningDeep LearningTensorFlow-Pytorch神经网络
TensorFlow Playground is a tool that can train simple neural networks through a web browser and visualize the training process. You can simply simulate the process of deep learning. Thicker lines indicate greater weight. URL:http://playground.tensorflow.org/

Parameter introduction

Epoch：Number of iterationsLearning rate：Learning rate, in order to make the gradient descent method have better performance, we need to set the value of the learning rate within an appropriate range. A learning rate that is too large will lead to unstable learning, and a value that is too small will lead to extremely long training time. The adaptive learning rate achieves a reasonably high rate while ensuring stable training, which can reduce training time. According to experience, we generally set the learning rate between 0.01 and 0.8Activation：Activation function. Common activation functions include Sigmoid, ReLu, Linear, and Tanh. The so-called activation function (Activation Function) is a function that runs on the neurons of an artificial neural network and is responsible for mapping the input of the neuron to the output.Regularization：Regularization, one goal of regularization is to reduce the degree of overfittingRegularization Rate：regularization rateProblem Type：There are classification and regression

data area

Select the data you want to classify (regress) Next adjust the ratio of test set to training set (default is enough)Noise：Noise and unusual data will increase the difficulty of classification or regressionBatch size：Batch size, Batch_size should not be selected too small. If it is too small, it will easily fail to converge, or it will require a large epoch to converge. There is no need to select it too large. If it is too large, the video memory will not be able to bear it. Secondly, the parameter correction may become slow due to the reduction in the number of iterations.

Main interface

The leftmost side is the input attributes, including X, Y, X^2, Y^2, sin(X), sin(Y), etc. In the middle is the visualization of the neural network. Each column is a hidden layer. At this time, there are two hidden layers. Click + to increase the number of hidden layers or neurons. The far right is the output, which is the loss (the smaller the better). Generally speaking, the loss of the training set is smaller than the loss of the test set.
