Feature engineering - feature discretization and selection

TechnologyMachine LearningEvolutionary Algorithms特征选择

summary

Since many feature selection problems deal with discrete data, a common approach is to apply discretization before feature selection. In order to improve efficiency, features are usually discretized univariately. This scheme is based on the assumption that each feature affects the task independently, which may not be true in the presence of feature interactions. Therefore, univariate discretization reduces the performance of the feature selection stage after information is displayed. Feature interactions may be lost during the discretization process. In addition to feature selection, feature discretization is also a crucial method for processing high-dimensional data. Discretization of features converts continuous values ​​into discrete values. Through discretization, small fluctuations or possible noise can be discovered.Therefore, feature selection + feature discretization can effectively improve classifier performance and reduce computing time and storage space.In this paper, the author proposes an algorithm called PPSO to reduce the search space. Of course, the fitness function here has also been adjusted accordingly to improve search performance. The result is a compression rate of up to 95%.Keywords: PSO, feature selection, discretization, classification, high dimensionIn PSO feature selection, PSO represents an N-dimensional variable. Each value in real encoding is between 0 and 1. One possible result of this is that two different vectors select the same feature subset. In discretization, a slightly different evolutionary cut-point may lead to different discrete characteristics.The entropy base cut point can reduce the search space.

Target

Propose a new way to combine discretization and feature selection into the same stage (BBPSO)

The research questions include

① How to perform multi-variable discretization and selection to improve the discriminative power of feature sets in a single stage. ② Can PPSO achieve better classification accuracy by discretizing and selecting features than using the full feature set? ③Whether the performance of PPSO is better than the corresponding one. Accuracy of two-stage methods on feature subsets. size and computation time. ④Whether PPSO is better than EPSO and achieves better classification performance without FS bias. ⑤Whether the results of PPSO can be well generalized to algorithms other than packaging methods in learning.

Feature Discretization (quantification and discretization of features)

Research on feature discretization has a long history. There are many discretization methods, but these methods are all based on the same goal: to determine cut-points to divide features into discrete values. Cut-points, or split-points, are real values ​​within a feature range that divide the feature range into multiple intervals. Existing discretization methods fall into the following categories: ①Direct method, the interval is generated by predefined parameters ② Incremental method, recursive division or merging interval guidance to meet the requirements These are known as top-down or bottom-up approaches Of course there are supervised and unsupervised approaches. The effect of the unsupervised method is not as good as the supervised method. The following mainly explains the supervised method. Minimizing the discretization length (MDL) is currently a popular method. It is a discretization method based on entropy-based incremental partitioning. Information gain is used to evaluate cut-points. Through recursion, MDL is able to select the best cut-point to turn one interval into two intervals until the minimum discretization length is reached. In this paper, the author proposes a cut-point method that uses the entropy-based cut point as an initial candidate for BBPSO.

Feature selection using discretization

dual stage

There are not many studies that combine feature selection and discretization. First, the features are ranked through the discretization process, and then the TOP-RANKED features will be selected. This method is divided into two steps, the first step is discretization and the second step is feature selection.

Single stage (feature selection while discretizing)

In order to achieve discretization, the eigenvalue features below the cut-point are set to 0, and vice versa. If all the values ​​​​of a feature are set to 0 or 1, it can be considered an irrelevant feature because it cannot distinguish the categories in the instance. Feature selection will remove these features.
