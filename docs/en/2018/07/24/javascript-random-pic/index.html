JavaScript implements random pictures

Programming LanguagesTechnologyWebJavaScript&HtmlWebJavaScriptHTML
In many websites, there are many top featured images, and the images are displayed differently every time you enter, that is, the images are displayed randomly. A careful analysis shows that the principle of changing the special effects should be to embed JavaScript code in HTML, store the image address in an array, then generate a random index through a random number, and then call the write function, so that the image can be randomized.
This article mainly introduces the JS implementation of the effect of randomly displaying images when the page is loaded, and involves the operation techniques related to the dynamic modification of page elements based on javascript based on random numbers and arrays. Friends in need can refer to it.
Code:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>
<script type="text/javascript">
    //<!CDATA[
    var pic = [];
    pic[0] = "https://cdn.omegaxyz.com/2018/06/new.jpg";
    pic[1] = "https://cdn.omegaxyz.com/2018/01/tensorflow.jpg";
    pic[2] = "https://cdn.omegaxyz.com/2017/06/htmlandcss.jpg";
    pic[3] = "https://cdn.omegaxyz.com/2018/07/tbp.jpg";

    var randomBgIndex = Math.round( Math.random() * 3 );

    //输出随机的背景图
    //document.write('<style>body{background:url(' + bodyBgs[randomBgIndex] + ') no-repeat 50% 0}</style>');
    document.write("<img src=" + pic[randomBgIndex] + " width='400'>")
    //]]>
</script>
</body>
</html>
Note that the document.write() function is used to write HTML statements here. Copy the above code and click the link below to test it: https://omegaxyz.com/onlinetools/web-running/
