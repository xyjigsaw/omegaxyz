Baidu library downloader Python implementation

Programming LanguagesTechnologyAPP开发PythonPythonwxpythonGUI

Introduction

In the past, you could use your school's corporate account to download documents. Later, Baidu Library was revised and you needed to buy a download coupon or membership. Therefore, a Baidu Library downloader is needed. There was a Freezing Library downloader before, but it didn’t work anymore.The following is a Baidu library downloader written in Python.This downloader can download doc type, txt type text documents and PPT, but the PPT will eventually be saved in the form of pictures. Due to limitations of Baidu Library, it is temporarily impossible to break through the documents for trial reading. PDF documents can be downloaded at will.

Implement ideas and steps

①Document link webpage GBK transcoding ②Get the document type ③ Get the document ID. Each document has an ID. Get the ID value and get the image resources in the following format (taking PPT as an example). https://wenku.baidu.com/browse/getbcsurl?doc_id=" +doc_id + "&pn=1&rn=99999&type=ppt The principle is that this in the preview page is not a PPT but a PPT image. We need to understand where this PPT was obtained. Once we find the address, we can download these images.
