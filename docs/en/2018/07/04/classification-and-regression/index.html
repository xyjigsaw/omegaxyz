Machine learning classification and regression

TechnologyMachine Learning
Classification, that is, classification: a process in which you put something into the group or class it belongs to, this refers to the process of putting something into the group or class to which it belongs.return, that is, regression: the relation between selected values ​​of x and observed values ​​of y (from which the most probable value of y can be predicted for any value of x) For example, if the father is very short, his son will generally not be shorter than the father, but will be closer to the average value of the average person. At that time, the British geneticist called this phenomenonreturn. In computer science, other answers have said it well on how to distinguish between the two. How to distinguish classification from regression depends not on the input, but on the continuity of the output (a criterion), or in other words,The fundamental difference between regression and classification is whether the output space is a metric space。 Here are a few examples (from Zhihu): 1. Logistic Regression and Linear Regression:
  • Linear Regression: Outputs a scalar wx+b. This value is a continuous value, so it can be used to handle regression problems.
  • Logistic Regression: Map the above wx+b to (0,1) through the sigmoid function, and divide a threshold. Those greater than the threshold are divided into one category, and those less than or equal to the threshold are divided into another category. It can be used to deal with two-classification problems.
  • Going further: For N classification problems, N groups of wx+b with different w values ​​are first obtained, and then normalized, such as using the softmax function, and finally become probabilities on N classes, which can handle multi-classification problems.
2. Support Vector Regression and Support Vector Machine:
  • SVR: The output wx+b, which is the distance from a certain sample point to the classification surface, is a continuous value, so it is a regression model.
  • SVM: Use this distance with the sign(·) function. The sample points with a positive distance (on one side of the hyperplane) are of one type, and the negative ones are of another type, so it is a classification model.
3. Neural Networks for Classification and Regression:
  • Used for regression: The last layer has m neurons, and each neuron outputs a scalar. The output of the m neurons can be regarded as a vector v. Now all are connected to one neuron, then the output of this neuron is wv+b, which is a continuous value and can handle regression problems, the same as the idea of ​​Linear Regression above.
  • Used for N classification: Now these m neurons are finally connected to N neurons, and there are N groups of wv+b with different w values. In the same way, it can be normalized (for example, using softmax) to become Probability on N classes.
Extension: The above examples are actually from the perspective of prediction. From the perspective of training, the objective functions of classification models and regression models are different.
