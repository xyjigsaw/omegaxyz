Overview of feature discretization

TechnologyMachine Learning特征选择

definition

Continuous feature discretization uses various methods to divide continuous intervals into small intervals, and associates these continuous small intervals with discrete values. The essence of continuous feature discretization is to decide how many segmentation points to select and determine the location of the segmentation points.

Background

Continuous data are often discretized before being put into the model. Discretization can be understood as the process of extracting features. For example, in the LR model, since the expression ability of the generalized linear model is limited, the nonlinear learning ability is improved through feature discretization.

step

  • 1. Sort continuous feature values ​​according to a specified rule
  • 2. Preliminarily determine the breakpoints for dividing continuous attributes
  • 3. Continue to divide breakpoints or merge breakpoints according to a given judgment standard
  • 4. If the termination condition of the judgment standard is obtained in the third step, terminate the entire continuous feature discretization process, otherwise continue to the third step.

main method

equidistant

The value range is evenly divided into n equal parts, with equal spacing between each part.

Equal frequency discrete

Evenly divided into n equal parts, each part contains the same number of observation points

Optimize discrete

There are roughly two types of methods: 1. Chi-square test method: (The degree of deviation between the actual observed value of the statistical sample and the theoretically inferred value. The larger the chi-square value, the more inconsistent it is; the smaller the chi-square value, the smaller the deviation, the more consistent it is.) - Splitting method: Find a split point and see if there is a significant difference in the target value distribution between the left and right intervals. If there is a significant difference, split it, otherwise ignore it. This point can be used to find the point with the largest difference every time. - Merging method: first divide the unit interval if it is very small, and merge adjacent intervals with insignificant distribution on the target value in order until convergence 2. Information gain method: - Split method: Find a split point and look at the two intervals on the left and right to see the information gain change threshold before and after splitting. If the difference exceeds the threshold (positive value, information entropy before splitting - after splitting), split. Find the point with the largest difference each time as the split point until convergence - Merging method: first divide if the unit interval is very small, merge adjacent intervals whose information gain is less than the threshold in order until convergence

  • 2. Sparse vector inner product multiplication is fast, and the calculation results are easy to store and easy to expand.
