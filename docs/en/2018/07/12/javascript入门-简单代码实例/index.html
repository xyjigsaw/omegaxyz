Getting Started with JavaScript - Simple Code Example

Programming LanguagesTechnologyWebJavaScript&HtmlWebJavaScript
JavaScript, also known as ECMAScript, is a scripting language that is object- and event-driven, relatively safe, and widely used in client-side web development. It is also a scripting language that is widely used in client-side web development. JavaScript is the most popular scripting language in the world, because all web pages you browse on computers, mobile phones (Android, iOS), tablets, and countless mobile apps based on HTML5 have interactive logic driven by JavaScript. Simply put, JavaScript is an interpreted programming language that runs in the browser. ① Generate simple text and HTML tags and loops

Take the 99 multiplication table as an example
document.write("<table border='1' bordercolor= 'red'>");
for(var i = 1; i <= 9; i++){
  document.write('<tr>');
  for(var j = 1; j <= i; j++){
    document.write('<td>');
    document.write(j+'*'+i+'='+i*j+'</td>');
  }
  document.write('</tr>');
}
document.write('</table>');
