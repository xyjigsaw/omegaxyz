Introduction

Text markup language, namely HTML (Hypertext Markup Language), is a markup language used to describe web page documents. HTML is called Hypertext Markup Language because the text contains so-called "hyperlink" points. The so-called hyperlink is a URL pointer. By activating (clicking) it, the browser can easily obtain a new web page. This is also one of the most important reasons why HTML is widely used. HTML is a specification, a standard that uses markup symbols to mark various parts of a web page to be displayed. The web page file itself is a text file. By adding tags to the text file, you can tell the browser how to display the content (such as how to process text, how to arrange pictures, how to display pictures, etc.). The browser reads the web page files in order, and then interprets and displays the marked content according to the tags. The error will not be pointed out for wrongly written tags, and the interpretation and execution process will not be stopped. The compiler can only analyze the cause and location of the error through the display effect. However, it should be noted that different browsers may not have exactly the same interpretation of the same tag, and thus may have different display effects.Note: HTML tags are not case-sensitive, but lowercase is recommended!
