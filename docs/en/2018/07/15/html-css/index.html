HTML and CSS to organize static web pages

Programming LanguagesTechnologyWebJavaScript&HtmlWebHTMLCSS
If HTML is the raw material for building a house, such as steel, cement, and wood, then CSS is the design drawing of the house, that is, the style of the house, whether it is a classical garden style or a Gothic architecture. Therefore, HTML and CSS complement each other. If there is only HTML without CSS, the resulting web page will have poor readability, poor interactivity, and lack of beauty. If there is only CSS but no HTML, then the web page is simply empty talk. Next, use HTML and CSS to build a static web page.Note that the following elements can be understood as labels.

Legal HTML5

  • ①Must start with <!doctype html> unless you are writing HTML4 and XHTML.
  • ② The <html> element cannot exist without it. This is the top-level element or root element of the Web page, and all other elements are nested in it.
  • ③ Please indicate the encoding method when using head and bodyhead.
  • ④There must be a title in the head.

CSS embedded in HTML

There are two ways to embed CSS into HTML. One is to use style tags in the head.
<head>
    <style type="text/css">
      ......
    </style>
</head>
Another way is to use the link tag. The rel attribute in CSS specifies the relationship between the HTML file and the linked file, that is, we want to create a style sheet.
<link type="text/css" rel="stylesheet" href="newindex.css">
Note: The order of style sheets is important, the ones below will override the ones above.

ID of the element

You can add an ID attribute to any element, so that you can use CSS to create a different design for each element. For example, I specify an attribute for the div<div id="main"></div>

element's class

You can specify a class for different elements<span class="slogan"></span>The span class here is slogan

Element CSS

There are several ways to use CSS to create different designs for each element (the most commonly used ones are given) ① Design the specified element (without using ID)
body{
    background-color: antiquewhite;
    font-family: Georgia, "Times New Roman", Times,Serif;
    font-size: small;
    margin: 0px;
}
②Design based on ID Suppose the ID of an element is main For example:<div id="main"></div>Then the CSS is:
#main{
    background: azure top left;
    font-size: 105%;
    padding: 20px;
    margin: 0px 330px 10px 10px;
}
③Design according to categories Suppose the class of an element is slogan For example:<span class="slogan"></span>Then the CSS is:
.slogan{
    color: indigo;
}
④ You can use commas to specify the same CSS for different elements.

CSS

body{
    background-color: antiquewhite;
    font-family: Georgia, "Times New Roman", Times,Serif;

    margin: 0px;
}

    background-color: cornsilk;

nav{

    top: 0;
}

#main{

    font-size: 105%;
    padding: 20px;
    margin: 0px 330px 10px 10px;
