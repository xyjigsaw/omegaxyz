Introduction to MATLAB Genetic Algorithm Toolbox

Programming LanguagesTechnologyMATLABMachine LearningEvolutionary AlgorithmsMATLAB
MATLAB is a commercial mathematics software produced by the American MathWorks Company. It is an advanced technical computing language and interactive environment used for algorithm development, data visualization, data analysis and numerical calculations. It mainly includes MATLAB and Simulink. MATLAB software is a very powerful tool, in which the GA toolbox can be used to perform data optimization of genetic algorithms. A specific example of simple operation is given below.

①Use the GUI to open and use the MATLAB genetic algorithm toolbox.

Open MATLAB, select the application, and click Optimization.

Display optimized interface after openingThe first solver chooses ga

The MATLAB code of the function (note that a single target can only have one function output value) is filled in the Fitness Function, and the input variable value is 2. Other setting options can be customized or default. Here, my population is set to 200. After setting up, click start My optimized function code: 
function f=GA_demo(x)
f1=4*x(1).^3+4*x(1)*x(2)+2*x(2).^2-42*x(1)-14;
f2=4*x(2).^3+4*x(1)*x(2)+2*x(1).^2-26*x(1)-22;
f=f1.^2+f2.^2;

Commonly used options

Options Function value
CrossoverFraction probability of crossover Decimals from 0 to 1
EliteCount for meritocratic principles, The number of individuals that will definitely survive each inheritance positive integer
FitnessLimit fitness range Scalar/{-Inf}
Generations Number of iterations of inheritance positive integer
InitialPopulation initial population The population generated last genetically can be used As the initial population for the next GA
Of course, MATLAB officials say that this optimization toolbox will be removed in future versions, so please pay attention.

②Use the command to run the GA toolbox

Population size 200 The number of elite solutions is 20 Crossover rate 0.75 Number of iterations 1000 Stall generations and fitness function value deviation (function tolerance). If within the number of generations set by Stallgenerations, the weighted average change value of the fitness function value is less than the function tolerance, the algorithm stopsThe optimized function is the function given in the GUI above 
clear
clc
fitnessfcn = @GA_demo;
% 适应度函数句柄
nvars = 2;% 个体的变量数目
options = gaoptimset('PopulationSize',200,'EliteCount',20,'CrossoverFraction',0.75,'Generations',1000,'StallGenLimit',500,'TolFun',1e-100,'PlotFcns',{@gaplotbestf,@gaplotbestindiv}); %参数设置
[x_best,fval] =ga(fitnessfcn,nvars,[],[],[],[],[],[],[],options);   % 调用ga函数
