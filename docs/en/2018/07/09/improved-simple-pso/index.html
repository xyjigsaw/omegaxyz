Improvement strategies of PSO algorithm

TechnologyMachine LearningEvolutionary AlgorithmsPSO算法
PSO (PSO——Particle Swarm Optimization) (population-based stochastic optimization technology algorithm) The particle swarm algorithm imitates the swarming behavior of insects, herds, birds, fish, etc. These groups search for food in a cooperative manner, and each member of the group continuously changes its search pattern by learning from its own experience and the experience of other members.Introduction: https://omegaxyz.com/2017/05/04/introductionofpso/ For more PSO related articles and codes, please visit: https://omegaxyz.com/allarticles/machine-learning-navigator/#%e7%b2%92%e5%ad%90%e7%be%a4%e4%bc%98%e5%8c%96%e7%ae%97%e6%b3%95%ef%bc%88pso%ef%bc%89

Improved PSO algorithm

① gbest is the key to the PSO algorithm. After multiple iterations, the reason why gbest no longer improves is likely to be that it falls into a local optimum. In order to prevent it from permanently converging, we need to reset some genes of gbest, that is, some genes randomly mutate and then evaluate whether they are improved. If they are improved, replace them, and if not, roll back. ②Pbest’s local search strategy. Similarly for pbest, we need to perform a local search to speed up the convergence of the population. In binary-coded PSO, we can improve it through the flip strategy of pbest partial genes.Schematic diagram: shortcoming:Although the above two points can improve algorithm performance, they increase the number of evaluations and increase time consumption, so they need to be improved in large-scale problems.References: Tran B, Xue B, Zhang M. Improved PSO for Feature Selection on High-Dimensional Datasets[C]// Asia-Pacific Conference on Simulated Evolution and Learning. Springer International Publishing, 2014:503-515.
