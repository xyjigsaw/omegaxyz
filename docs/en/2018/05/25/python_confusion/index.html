Introduction to confusion matrix and Python implementation

Programming LanguagesTechnologyMachine LearningPythonPython

What is confusion matrix

The confusion matrix is ​​a situation analysis table that summarizes the prediction results of the classification model in machine learning. It summarizes the records in the data set in matrix form according to two criteria: the true category and the classification judgment made by the classification model. The name comes from the fact that it can be very easily indicatedIs there confusion among multiple categories?(that is, aclass is predicted to be another class） As shown below: The green part is the correct prediction, and the red part is the incorrect prediction. For a two-category (true or false) problem: refer to:https://omegaxyz.com/2017/08/27/rocandauc/

Use of Python confusion matrix

Use of confusion_matrix functionThe usage given in the official documentation is sklearn.metrics.confusion_matrix(y_true, y_pred, labels=None, sample_weight=None) y_true: is the true classification result of the sample, y_pred: is the predicted classification result of the sample labels: is the category given, through which the category can be selected sample_weight: sample weightImplementation code: 
from sklearn.metrics import confusion_matrix

y_true = [2, 1, 0, 1, 2, 0]
y_pred = [2, 0, 0, 1, 2, 1]

C=confusion_matrix(y_true, y_pred)
print(C, end='\n\n')


y_true = ["cat", "ant", "cat", "cat", "ant", "bird"]
y_pred = ["ant", "ant", "cat", "cat", "ant", "cat"]
C2 = confusion_matrix(y_true, y_pred, labels=["ant", "bird", "cat"])
print(C2)
