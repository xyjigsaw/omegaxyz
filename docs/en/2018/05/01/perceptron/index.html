Overview of Neural Network Perceptron

TechnologyMachine LearningDeep Learning神经网络

definition

Assume that the input space (feature vector) is X⊆Rn and the output space is Y={-1, +1}. The input x∈X represents the feature vector of the instance, corresponding to the point in the input space; the output y∈Y represents the category of the example. The function from input space to output space is

f(x)=sign(w⋅x+b)

called a perceptron. Among them, the parameter w is called the weight vector weight, and b is called the bias. w⋅x represents the dot product of w and x

∑i=wixi=w1x1+w2x2+...+wnxn

sign is a symbolic function,In a binary classification problem, the value of f(x) (+1 or -1) is used to classify xx as a positive sample (+1) or a negative sample (-1). The perceptron is a linear classification model and is a discriminative model. What we need to do is to find an optimal w and b value that satisfies w⋅x+b=0, that is, the separating hyperplane. As shown below, a linearly separable perceptron model.The straight line in the middle is the straight line w⋅x+b=0. The geometric representations of linear classifiers include: straight line, plane, and hyperplane.

learning strategies

Core: Minimize the loss function.If the training set is separable, the learning purpose of the perceptron is to find a separation hyperplane that can completely separate the positive instance points and negative instance points of the training set. In order to find such a plane (or hyperplane), that is, to determine the perceptron model parameters w and b, we use the loss function and minimize the loss function. For the selection of the loss function, we use the distance from the misclassified point to the hyperplane (you can calculate it yourself, the geometric distance is used here, which is the distance from the point to the straight line): 1∥w∥|w∗x0+b| If there are no misclassified points, the value of the loss function is 0, and the fewer misclassified points, the closer the misclassified points are to the hyperplane, and the smaller the value of the loss function.. At the same time, the loss functionL(w,b)L(w,b)is a continuously differentiable function.

learning algorithm

Perceptron learning is transformed into solving the loss functionL(w,b)L(w,b)optimization problem. The optimal method is stochastic gradient descent (stochastic gradient descent), this is the method used here. For more details on gradient descent, refer towikipedia Gradient descent. To explain, gradient descent is actually a local optimum. The perceptron learning algorithm itself is misclassification driven, so we use stochastic gradient descent. First, choose a hyperplanew0w0andb0b0, and then use the gradient descent method to continuouslyMinimize objective function Then, randomly select a misclassified point and calculate a neww,bw,b, and then update:
w:=w+ηyixi
b:=b+ηyi
inηis the step size, greater than 0 and less than 1, it is called the learning rate in statistical learning (learning rate). In this way, by iterating we can expect the loss functionL(w,b)Keep decreasing until it is 0.
