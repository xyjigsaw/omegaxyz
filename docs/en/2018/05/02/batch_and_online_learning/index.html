Overview of batch learning and online learning

TechnologyMachine Learning在线学习

1. Batch learning

In batch methods of supervised learning, the adjustment of the salient weights of the multi-layer perceptron is performed after all N examples of the training sample set have appeared, which constitutes one round of training. In other words, the cost function of batch learning is defined by the average error energy. The adjustment of synaptic weights of multilayer perceptrons is on a turn-by-turn basis. Accordingly, one implementation of the learning curve is obtained by plotting the average error energy versus the number of epochs. For each epoch of training, examples from the training sample set are randomly selected. The learning curve is calculated by averaging over a sufficiently large number of such implementations, where each implementation is performed under randomly chosen different initial conditions. This feature is in line with the rules of cross-validation. The experimental set, verification set, and test set in the experiment are generally typical examples of batch processing.advantage:
  • (1) Eliminate the influence of sample order
  • (2) Accurate estimation of the gradient vector, therefore, under simple conditions, ensuring the fastest descent of this method to the local minimum point.
  • (3) Parallelism of learning.
shortcoming:
  • (1) There are storage requirements, and all training sets need to be loaded into memory at one time, which is obviously not suitable for real-world problems.
  • (2) Not suitable for processing high-dimensional data

2. Online learning

Under the online method of supervised learning, the adjustment of the synaptic weights of the multi-layer perceptron is based on example-example, and the cost function used to minimize is the overall instantaneous error energy. Like batch learning, the learning curve of online learning is calculated by taking the ensemble average over a sufficiently large number of randomly selected initial conditions. For a given network structure, the learning curve obtained under online learning is very different from that obtained under batch learning. Given that training samples are presented to the network in a random manner, the use of online learning makes the search in the multidimensional weight space virtually random; it is for this reason that online learning methods are sometimes called stochastic methods.advantage:
  • (1) Easy to execute
  • (2) It provides effective solutions to large-scale and difficult pattern classification problems.
  • (3) Randomness makes it difficult to fall into local extreme points
  • (4) Much less storage
