Is Github code communism?

TechnologyIdeas其它Developer LifeGithub

What is GitHub

gitHub is a hosting platform for open source and private software projects. Because it only supports git as the only version library format for hosting, it is named gitHub. gitHub was officially launched on April 10, 2008. In addition to git code warehouse hosting and basic Web management interface, it also provides subscription, discussion groups, text rendering, online file editor, collaboration map (report), code snippet sharing (Gist) and other functions. Currently, it has more than 3.5 million registered users and a large number of hosted versions, including well-known open source projects Ruby on Rails, jQuery, python, etc. ——Baidu Encyclopedia Although it's nothingMy github homepage:https://github.com/xyjigsaw

What does Zhihu say?

Github is one of the concrete implementation examples of communism in the field of software development.That is, it is based on the social public ownership of the means of production (code is open source), takes social cooperation (discrete individuals, organizations and other social particles) as the main production method, and mainly adopts the technical support platform of cooperative organizational form (project team). This description is broad enough. The vast majority of individuals and teams participating in the open source software community do not receive labor wages or remuneration. These individuals or teams may be affiliated with a commercial organization, but simply participating in open source project development itself does not receive labor remuneration. Instead, the results are distributed through the sharing of cooperative output. With the same distribution method, why the tragedy of the People's Commune, which resulted in a large number of people starving to death due to eating from a big pot, has not been repeated (that is, the reduction in rent value inevitably caused by public ownership) is because the output of the software cooperative is code, which supports unlimited supply and zero cost of copying. Therefore, it satisfies a premise of the communist production model, that is, unlimited supply of production materials, complete surplus of production results, and everyone can get what they need. Because of the complete surplus of labor results (unlimited copies, inexhaustible), the production results and production materials of open source software cooperatives are not only distributed in unlimited quantities to workers participating in labor, but also provided to other members of society who do not participate in labor in unlimited amounts for free (the code is open source, free to download, copy, and use at will). As I explained in another answer, if the means of production have the characteristics and conditions of unlimited supply or zero-cost reproduction, it can eliminate the problem of rent dissipation that public ownership has been unable to overcome. If this prerequisite is not met, it can still be explained by rent dissipation. The advancement of this production model is reflected in the following two aspects:
  • First, because there is no exclusivity in the fruits of labor and no specific labor remuneration, the main purpose of participating in open source community labor is labor itself. This is in line with what Marx said, labor should not be a means, but an end. This purer driving force comes from human nature itself, that is, the need for labor (to put it bluntly, finding something to do is our human need), so it can inspire more creative passion and selfless work and labor. There are currently hundreds of thousands of great and highly productive open source software products on github.
  • Second, give full play to the optimal allocation and utilization efficiency of social resources. In the traditional capitalist organized production model, many resources are limited within the organization. No company is willing to pay high salaries to hire coders to work for free. This results in a certain waste of resources, and agglomerated allocation makes it easy for advantageous resources to be highly concentrated. The result is a monopoly-shaped organization, which inhibits innovation vitality and further reduces resource utilization efficiency. In the software open source cooperative, people make the best use of it, and the code makes the best use of it. There are no barriers to obtaining production results, and there is no monopoly behavior in decentralization. It breaks organizational barriers and greatly optimizes the allocation and utilization efficiency of social resources.
This production model can also effectively hedge the failure costs of social experiments in the field of innovation. Github responds to mass innovation and entrepreneurship, and there are millions of crappy projects on it. However, these failure costs have also been borne by socialization (no wages are paid anyway), disappearing into the invisible (code farmers for the purpose of labor, relatively do not care much about the success or failure of the project itself), and not a grain of rice of the country's subsidy funds is wasted. From this perspective, large-scale standardized industrial production is not suitable for the public ownership model (rent value dissipation problem). In the field of software innovation, due to the unrestricted supply of production materials and the need for a large amount of experimental practice and exploration, the communist production model can exert great socialized cost hedging advantages. At present, the main productivity of github comes from the capitalist society with highly developed productivity. For our country, because it is in the primary stage of socialism with not yet highly developed productivity (it still needs to improve the productivity of the current stage through a work model that uses labor as a means and with the main purpose of paying off mortgage loans. Please search for "Industrial Party" on Zhihu here) , in order to prevent our country's code farmers from being transformed into work for the purpose of labor itself, resulting in the loss of productivity, contributing to the realization of Western communism, and possibly delaying the historical process of socialism in our country. Therefore, based on this understanding, our country currently adopts effective firewall blocking measures for platforms such as github, so that downloads and uploads are often inexplicably disconnected.
