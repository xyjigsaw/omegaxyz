online learning

Under the online method of supervised learning, the adjustment of the synaptic weights of the multi-layer perceptron is based on example-example, and the cost function used to minimize is the overall instantaneous error energy. Like batch learning, the learning curve of online learning is calculated by taking the ensemble average over a sufficiently large number of randomly selected initial conditions. For a given network structure, the learning curve obtained under online learning is very different from that obtained under batch learning. Given that training samples are presented to the network in a random manner, the use of online learning makes the search in the multidimensional weight space virtually random; it is for this reason that online learning methods are sometimes called stochastic methods.advantage:
  • (1) Easy to execute
  • (2) It provides effective solutions to large-scale and difficult pattern classification problems.
  • (3) Randomness makes it difficult to fall into local extreme points
  • (4) Much less storage

Online feature selection (OFS)

Most research on online learning requires training instances using all features. This traditional method is not suitable for real-world applications. This can be computationally expensive when the data instance is high dimensional or requires access to obtain complete features. To solve this problem, Online Feature Selection (OFS) for online learners only allows to retain a small classifier with a fixed number of features. OFSKey points and challenges:Choose how to make accurate predictions using a small, fixed number of activated features. This is different from classic online learning that uses all the features (all the features are active) to make predictions. Feature selection usually assumes that the feature selection task is performed in an offline learning manner with all features. This may not always apply to some practical problems. Different from existing research on feature selection, we study the problem of online feature selection. (OFS) aims to solve the feature selection problem through a method.Goals of online feature selection:Construct online classifiers containing only a small and fixed number of features. The goal of online feature selection is to develop online classifiers that involve only a small and fixed number of features. In most traditional online feature selection, features are considered to arrive one at a time when all training instances are before the learning process starts. Their purpose is to select a subset of features and return a suitable model each time based on the features given so far.
