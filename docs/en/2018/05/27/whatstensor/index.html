Introduction and application of Tensor

Programming LanguagesTechnologyMachine LearningPythonDeep LearningPythonTensorFlow-Pytorch神经网络
Tensor(Tensor) is a very important concept. existTensorFlowAll data in are represented by tensors. Of course, tensors can be simply understood as multi-dimensional arrays. Among them, a zero-order tensor represents a scalar, a first-order tensor represents a vector, and an n-order tensor is an n-order array. However, the implementation of tensors in TensorFlow is not in the form of arrays like Numpy.It is just a reference to the operation result, which is a calculation process.Here is an example:
import tensorflow as tf

a = tf.constant([1.0, 2.0], name="a")
b = tf.constant([2.0, 3.0], name="b")
result = tf.add(a, b, name="add")
print(result)
# Tensor("add:0", shape=(2,), dtype=float32)
with tf.Session() as sess:
    print(sess.run(result)
Calculation result:Tensor("add:0", shape=(2,), dtype=float32) [3.5.] You can see that the calculation result is a structureadd represents the name (node ​​name), and 0 represents the output from the node. Shape is the structure of the array, which is a one-dimensional array with a length of 2 dtype is the data type. Note that if the types of a and b do not match, this code will not be executed.Of course, result only defines such an operation and is not actually executed. You need to run the session to see the final result.
