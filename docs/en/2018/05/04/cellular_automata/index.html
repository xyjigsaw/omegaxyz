Overview of Cellular Automata and Implementation in MATLAB

高级语言技术域MatlabMatlab元胞自动机计算机图形学

What is a cellular automaton?

Cellular automata (CA) is a grid dynamics model in which time, space, and state are discrete, and spatial interaction and temporal causality are local. It has the ability to simulate the spatiotemporal evolution process of complex systems. It can build systems where state transitions occur over time, where cells exist in a one- or multi-dimensional grid, each cell has one or more states, and each cell has neighbors (i.e. neighboring cells).

Cellular Automata Classification

Smooth type:Starting from any initial state, after a certain period of time, the cell space tends to a spatially stable configuration, where spatial stability means that each cell is in a fixed state. Does not change with time.periodic: After running for a certain period of time, the cell space tends to a series of simple fixed structures (Stable Patterns) or periodic structures (Perlodical Patterns). Since these structures can be regarded as filters, they can be applied to image processing research.Chaos type:Starting from any initial state, after running for a certain period of time, the cellular automaton exhibits chaotic non-periodic behavior, and the statistical characteristics of the generated structure no longer change, usually showing fractal dimension characteristics.Complex type:Complex local structures, or local chaos, emerge, some of which propagate continuously.

odd-even rule

The parity rule is a cellular automaton defined on a two-dimensional grid. The status of each grid is represented by 0 and 1. (1) Corresponding to each cell position (i,j), calculate the sum M(i,j) of the state values ​​St of its eight nearest neighbors at time t. (2) Determine the state St+1(i,j) at the next time t+1 based on the parity of the value of M(i,j). When M(i,j) is an even number, St+1(i,j) is equal to 0; when M(i,j) is an odd number, St+1(i,j) is equal to 1. If the sum of the eight points is an even number, it becomes 0, and if it is an odd number, it becomes 1

Realization of stationary cellular automata

% 规则，先把中间点置为1，每一时间步对每一点，如果周围
% 八个点和为偶数，则变为0，为奇数则变为 1


Map = [1 1 1; 0 0 0];
colormap(Map);
% 设置网格大小
S = 121;
L = zeros(S);
% 把中间一个数设置为 1 作为元胞种子
M = (S+1)/2;
L(M, M) = 1;
Temp = L;
imagesc(L);

% 计算层数
Layer = (S-1)/2 + 1;

for t=2:Layer
    for x=M-t+1:M+t-1
       if x==M-t+1 || x==M+t-1
           
          for y=M-t+1:M+t-1
            SUM = 0;
            for m=-1:1
               for n=-1:1
                  if x+m>0 && x+m<=S && y+n>0 && y+n<=S
                     SUM = SUM + L(x+m, y+n); 
                  end
               end
            end
            SUM = SUM - L(x, y);
            Temp(x, y) = mod(SUM, 2);
          end
          
       else
            y = M-t+1;
            SUM = 0;
            for m=-1:1
               for n=-1:1
                  if x+m>0 && x+m<=S && y+n>0 && y+n<=S
                     SUM = SUM + L(x+m, y+n); 
                  end
               end
            end
            SUM = SUM - L(x, y);
            Temp(x, y) = mod(SUM, 2);
            
            y = M+t-1;
            SUM = 0;
            for m=-1:1
               for n=-1:1
                  if x+m>0 && x+m<=S && y+n>0 && y+n<=S
                     SUM = SUM + L(x+m, y+n); 
                  end
               end
            end
            SUM = SUM - L(x, y);
            Temp(x, y) = mod(SUM, 2);
       end
    end
    L = Temp;
    imagesc(L);
    pause(0.1);
end
