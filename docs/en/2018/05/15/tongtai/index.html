What is homomorphism

Homomorphism is a concept in discrete mathematics. The last question in the midterm exam was homomorphic encryption, which sparked my interest. It was time to study it. Here is the definition of homomorphism in discrete mathematics:
Suppose (M,*) and (S,·) are two groups, σ:M→S, ∀a,b∈M, and σ(a*b)=σ(a)·σ(b), then σ is called a homomorphism or group mapping from M to S.

Homomorphic encryption

If we have an encryption function f, turn plaintext A into ciphertext A’, and turn plaintext B into ciphertext B’, that is, f(A) = A’, f(B) = B’. In addition we also have a decryption functionf−1f−1Can decrypt the encrypted ciphertext of f into the plaintext before encryption. For a general encryption function, if we add A’ and B’, we get C’. We usef−1f−1The result of decrypting C' is generally meaningless garbled code. However, if f is an encryption function that can perform homomorphic encryption, we usef−1f−1Decrypt to obtain the result C, at this time C = A + B. so,Data processing rights and data ownership can be separated, so that companies can prevent their own data from being leaked while utilizing the computing power of cloud services.

homomorphic classification

a) If it satisfiesf(A)+f(B)=f(A+B)f(A)+f(B)=f(A+B), we call this encryption function additive homomorphism b) If it satisfiesf(A)×f(B)=f(A×B)f(A)×f(B)=f(A×B), we call this encryption function multiplicative homomorphism. If an encryption function f only satisfiesadditive homomorphism, it can only be carried outAddition and subtraction operations; If an encryption function f only satisfiesmultiplicative homomorphism, it can only be carried outMultiplication and division operations; If an encryption function satisfies bothAdditive and multiplicative homomorphisms, calledFully homomorphic encryption。Then this encryption function is used to complete various encrypted operations (addition, subtraction, multiplication and division, polynomial evaluation, exponential, logarithm, trigonometric functions).The first homomorphic encryption method that satisfies additive and multiplicative homomorphism was not proposed until 2009 by Craig Gentry.

Homomorphic encryption algorithm

  1. The RSA algorithm is homomorphic for multiplication operations.
  2. Paillier's algorithm is homomorphic to addition.
  3. Gentry's algorithm is fully homomorphic.
