Python implementation of English letter information entropy and redundancy calculation

Programming LanguagesTechnologyMachine LearningPythonPython信息论
The information entropy of the 26 English letters when they appear on average is 4.7BIT, and the information entropy of the English letters after removing redundancy is 4.03BIT. Formula: H (information entropy) = -∑ Pi log2(Pi); Pi: is the probability of each letter appearing in the information; the calculation formula is not complicated. The principle of taking the logarithm with base 2 is also very simple, because if: 2n= X, then logX = n;So it can be seen that the so-called information entropy is the number of binary encoding digits of a binary character set after removing redundancy.Redundancy is obtained by counting the occurrence probability of each character. import math

def calEntropy(string):
    h = 0.0
    sumt = 0
    letter = [0] * 26
    string = string.lower()
    for i in range(len(string)):
        if string[i].isalpha():
            letter[ord(string[i]) - ord('a')] += 1
            sumt += 1
    print('\n', letter)
    for i in range(26):
        p = 1.0 * letter[i] / sumt
        if p > 0:
            h += -(p * math.log(p, 2))

    return h


test = input("输入一个英文句子：")
print('\n熵为：', calEntropy(test))
