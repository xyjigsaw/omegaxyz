Python crawls web article GUI (CSDN, Jianshu, major domestic and foreign news websites, etc.)

Programming LanguagesTechnologyPythonPythonwxpythonGUI
Python can do a lot, and crawlers are a quick way to get information. Many times we only crawl a specific website without considering the versatility of the program. The following is a pythonGUI program for crawling web articles, written in wxpython. It can capture articles from CSDN, Jianshu, WordPress-based blogs, major domestic and foreign news websites, etc. Although it can crawl articles from most websites, forcodeRich articles cannot be crawled yet, so the code is being improved.
