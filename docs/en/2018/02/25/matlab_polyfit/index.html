MATLAB curve fitting

Programming LanguagesTechnologyMATLABMATLAB
curve fitting functionPolynomial fitting function: polyfit. The result of this function will ensure that the sum of squares of the difference between the fitted value and the data value at the data point is minimized, that is, a least squares curve fit. Calling format: polyfit(X,Y,n) Executing this function will produce a polynomial P of order n such that P(X)=Y. Example: Use a fifth-order polynomial to perform least squares fitting of the sine function on [0,pi/2].
x=0:pi/20:pi/2; 
y=sin(x);
a=polyfit(x,y,5);  %用5阶多项式拟合
x1=0:pi/30:pi/2; 
y1=sin(x1);
y2=a(1)*x1.^5+a(2)*x1.^4+a(3)*x1.^3+a(4)*x1.^2+a(5)*x1+a(6);
plot(x1,y1,'b-',x1,y2,'r*')
legend('原曲线','拟合曲线')
axis([0,2,0,1.5])
interpolation functionInterpolation is divided into one-dimensional interpolation and two-dimensional interpolation. One-dimensional interpolation is to interpolate data points in the direction of the line; two-dimensional interpolation can be understood as interpolation in the direction of the surface. The one-dimensional interpolation function is the simplest and most important interpolation function. Its calling method is: Y1=interp(X,Y,X1,'interpolation method') Among them, X is the node vector value, Y is the corresponding node function value, and X1 is the interpolation point. The returned Y1 is the function value that calculates the interpolation point X1.Interpolation methods are: nearest Linear nearest term interpolation linear linear interpolation (default method) spine cubic spline interpolation cubic cubic interpolation Requirement: X may not be equally spaced, but it must be monotonic.
