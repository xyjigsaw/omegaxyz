C Language Game 4: Gambling Machine Dice

Programming LanguagesC & C++TechnologyRepostsCC语言游戏
A small game written in C language - gambling machine, suitable for people who learn C language to learn from. (A C language to write a small game- gambling machines, suitable for learning C language people learn.)
http://www.pudn.com/Download/item/id/2568776.html
The following are the rules of gambling
The banker (and casino owner) and the player (and the player) each have 1,000 yuanbao as prompt funds.
There are three ways for players to place bets:
(1) Buy big (2) Buy small (3) Buy leopard
The differences between these three methods are introduced one by one below:
Bet big: If the sum of the three dice rolled by the player is greater than 11, the player wins, otherwise the banker wins
Buy small: If the sum of the three dice rolled by the player is less than 11, the player wins, otherwise the banker wins
Buy a leopard: If the three dice thrown by the player have the same number, the player wins, otherwise the banker wins.
The odds of buying big and small are both 2, that is, the loser will pay twice the amount of gold he bet to the winner, and the odds of buying leopard are 10
The number of ingots for player and bet must be between 10 and 100 ingots
The player can press the q key at any time to end gambling, otherwise it will continue until one party loses all their ingots.

int judge(int a,int b,int c)
{
	if(a==b&&a==c)
	{
		return 3;
	}
	if(a+b+c>11)
	{
		return 1;
	}
	if(a+b+c<11)
	{
		return 2;
	}
	else
	{
		return 0;
	}
}
