python Weibo crawler GUI program

Programming LanguagesTechnologyAPP开发PythonappPython多线程wxpythonGUI
The web version of Weibo is pure HTML and calls Weibo’s own API to obtain images. The URL: https://m.weibo.cn/api/container/ is the Weibo API which contains personal information and the storage address of Weibo text and pictures. This is a pure code non-GUI blog post:https://omegaxyz.com/2018/02/13/python_weibo/GUI program functions: ① Enter the ID of the Weibo that needs to be crawled, click Run and select the saved location to crawl. ② Progress visualization interface. ③If the built-in proxy fails, you can customize the proxy settings. ④ Visualize the personal information of the Weibo users being crawled.
