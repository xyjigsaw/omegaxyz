Supervised learning vs. unsupervised learning

TechnologyMachine Learning
Machine learning can be divided into the following two common methods according to the presence or absence of training sample labels. havesupervised learning(supervised learning) andunsupervised learning(unsupervised learning). Taking classification in machine learning as an example, the input training data has features and labels. During the classification process, if all training data has labels, it is supervised learning. If the data has no labels, it is obviously unsupervised learning, that is, clustering. Supervised learning is to train an optimal model through existing training samples (that is, known data and its corresponding output) (this model belongs to a set of functions, and optimal means the best under a certain evaluation criterion). Then use this model to map all inputs to corresponding outputs, and make simple judgments on the outputs to achieve the purpose of classification. This also has the ability to classify unknown data. Typical examples are KNN and SVM. It differs from supervised learning in that we do not have any training samples in advance and need to model the data directly.
