MATLAB 2D plotting

Programming LanguagesTechnologyMATLABMATLAB
vBasic drawing commandsThe most commonly used function for drawing two-dimensional graphics is the plot function. Its calling format:plot(Y) If Y is a vector, the drawn graph uses the vector index as the abscissa value and the vector component value as the ordinate value. If Y is a matrix, plot Y's column vector against its coordinate index. If Y is a complex vector (matrix), plot(Y) is equivalent to plot(real(Y),imag(Y)). In other forms of function calls, the imaginary part of the element is ignored. For example: y=rand(100,1); plot(y)plot(X,Y) X is the abscissa and Y is the ordinate. Note: Vectors X and Y must be of the same dimension, and must also be row vectors or column vectors. When the variables X and Y are matrices of the same order, operations will be performed on the rows or columns of the matrix. In particular, the variable Y can contain multiple vectors that meet the requirements, in which case all the graphics will be drawn in the same picture.Special two-dimensional graphics functions Two-dimensional graphics functions in special coordinate systems Special coordinate systems are specifically: logarithmic coordinate system and polar coordinate system. Function semilogx: x coordinate is a logarithmic coordinate to draw a two-dimensional graph semilogy: The y coordinate is a logarithmic coordinate to draw a two-dimensional graph loglog: draw two-dimensional graphs with logarithmic coordinates Polar coordinate system drawing function polar: polar(theta,tho,s) Among them, theta is the angle vector expressed in radians; tho is the corresponding amplitude vector; s is the graphic attribute setting option, which can be omitted.For example, draw a pie graph x=[2,4,6,8]; pie(x,{'math','english','chinese','music'})For example, drawing vector graphics x=0:0.1*pi:2*pi; y=sin(x).*x; feather(x,y)
