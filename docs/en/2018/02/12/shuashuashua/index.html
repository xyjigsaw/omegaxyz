15 lines of Python code to boost website traffic

高级语言技术域PythonwebPython
Statement 1: It is your right to blog, and no one will ban you. However, we use blogs to record every detail on the road to technology. The purpose of writing a blog is purely for my own use as an online diary. If we blog just for traffic, it insults the very purpose of blogging. Statement 2: www.omegaxyz.com has added malicious robot checking code and anti-crawler plug-ins. According to actual tests, it cannot increase the number of visits to this blog.During the winter vacation, I was bored and studied Python crawlers and agents. I discovered a very simple technique to boost blog traffic. First, we set up a proxy pool, which can be imported from a database or simply an array. The proxy pool can be accessed fromhttp://www.xicidaili.com/Select (as shown above)   Since it is not appropriate to use the same IP for two visits, just randomly select an IP from the proxy pool each time. More advanced ones can inject different cookies during the request to deceive the server. Secondly, use urllib in python to obtain the web page. It can generate a lot of traffic on most websites. Personal test CSDN is effective. Code:
from urllib import request
from random import randint

url = "http://xxxxx"

n = 1000

proxy = [{'https': '110.88.10.198'}, {'https': '182.88.117.118'}, {'https': '180.158.109.60'},
         {'https': '110.88.10.198'}, {'https': '122.235.184.109'}, {'https': '221.3.39.207'}]

for _ in range(n):
    proxy_support = request.ProxyHandler(proxy[randint(0, len(proxy)-1)])
    opener = request.build_opener(proxy_support)
    opener.addheaders = [('User-Agent',
                          'Mozilla/5.0 (Windows NT 6.1; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/56.0.2924.87 Safari/537.36')]
    request.install_opener(opener)
    response = request.urlopen(url)
    print((_ + 1) / n)
