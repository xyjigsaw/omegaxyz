Random fixed group cooperative coevolution NSGA2 algorithm (CCNSGA2)

Programming LanguagesTechnologyMATLABMachine LearningEvolutionary Algorithms多目标优化算法NSGA-IIMATLAB协同进化
Cooperative Coevolution is an effective method for solving large-scale optimization algorithms. Break large-scale problems into sets of smaller sub-problems. The key to cooperative coevolution is the decomposition strategy. For cooperative coevolution algorithms, please see:https://omegaxyz.com/2017/10/14/cooperative_coevolution/NSGA2 algorithm is a multi-objective genetic algorithm. This article uses NSGA2 to optimize the cooperative co-evolution of random fixed groups. For example, there are 12 decision variables, and we fix and randomly optimize 3 decision variables, then the decision variables are divided into 4 groups. MATLAB main function code:
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
clc;
clear;
global pop
pop = 500; %种群数量
gen = 2; %迭代次数
global M
M = 2; %目标数量
Dim=22;             %搜索空间维数（未知数个数）
sub_dim= 2 ;
global min_range
global max_range
min_range = zeros(1, Dim); %下界
max_range = ones(1,Dim); %上界
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
divide_datasets();
global answer
answer=cell(M,3);
Dim_index = ones(1,1)*(1:Dim+4);
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
chromosome = initialize_variables(pop, M, Dim, min_range, max_range, Dim_index);
chromosome = non_domination_sort_mod(chromosome, M, Dim);
result = 1;
while gen ~= 0
    subgroup = rnd_divide(Dim, sub_dim);
    for i=1:length(subgroup)
        subgroup{i}(sub_dim+1)=Dim+1;
        subgroup{i}(sub_dim+2)=Dim+2;
        subgroup{i}(sub_dim+3)=Dim+3;
        subgroup{i}(sub_dim+4)=Dim+4;
        [temp_chromosome] = nsga2(chromosome(:,subgroup{i}), sub_dim, subgroup{i});
        chromosome(:,subgroup{i}(1:sub_dim)) = temp_chromosome(:,1:sub_dim);
    end
    chromosome = nsga2(chromosome, Dim, Dim_index);
    chromosome = non_domination_sort_mod(chromosome, M, Dim);
    gen =gen - 1;
    progress = 1-gen/10
end

plot(chromosome(:,Dim + 1),chromosome(:,Dim + 2),'*');
xlabel('f_1'); ylabel('f_2');
title('Pareto Optimal Front');
Random group code:
% random grouping
function group = rnd_divide(dim, subdim)
   dim_rand = randperm(dim);
   group = {};
   for i = 1:subdim:dim
      index = dim_rand(i:i+subdim-1);
      group = {group{1:end} index};
   end
end
Other algorithm dependent functions are the same as the NSGA2 algorithm of custom optimization functions.https://omegaxyz.com/2018/01/22/new_nsga2/
