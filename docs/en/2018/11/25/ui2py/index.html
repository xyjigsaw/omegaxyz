pyqt5 converts ui files generated by graphic design into py files

Programming LanguagesTechnologyAPP开发PythonPythonGUIPyQt
Qt Creator is a cross-platform Qt IDE. Qt Creator is a new lightweight integrated development environment (IDE) launched after Qt was acquired by Nokia. This IDE is cross-platform and supports Linux (32-bit and 64-bit), Mac OS X, and Windows. According to the official description, the design goal of Qt Creator is to enable developers to use Qt, the application framework, to complete development tasks more quickly and easily. Graphics drawn using the Qt Creator or designer graphical interface will generate a ui interface, and the ui file needs to be converted into a py file. The steps are given below. ①Open qt creator and select a template②Select template configuration until completed ③Enter the main interfaceThe following is the xml ui fileDraw some controlsThere will be 3 files after saving④Enter the saved folder and open the terminal (CMD) Enter the command (both MAC and Windows)

pyuic5 -o qttest.py qttest.ui
