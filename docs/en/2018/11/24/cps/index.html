Classification-based preprocessing agent model in evolutionary computation

TechnologyMachine LearningEvolutionary Algorithms

Question raised

The construction of the surrogate model is more complex, and the author hopes to construct a simpler and cheaper surrogate model to evaluate the quality of the subset. Therefore, the author proposed an algorithm called CPS (classification based preselection).

preselect(preselection）

In evolutionary algorithms, preselection is a classification problem. To be precise, it will beoffspring solutionAs an external data set, it is classified into two categories: promising and unpromissing solutions.

CPSmain steps

  • ① Update the external population based on the parent solution and add labels to the external population
  • ②Construct a pre-classifier based on KNN
  • ③According to the constructed classifier, predict the descendant candidate solution label (the core of the agent model)
  • ④Select descendant solutions based on predicted labels

Data preparation (Data Preparation）

initialization:At the beginning of the algorithm, an external population is established and left blank. The external population is updated with iterations (at the same time, the external population can be used as an archive, and the final size is less than 5*N)Iterative process:Really evaluate each individual of the current parent generation and join the external population. Non-dominated sorting of external populations. P+ contains the first half of the size of the external population as non-dominated solutions, marked +1. P-contains the second half of the size of the external population as the dominated solution, labeled -1.

Classification model (Classification Model）

K is the K of KNN, which is an odd number. Xi is the kth nearest feature vector, and C is the external population feature vector label. When a descendant solution is newly generated, we can use this classification model to label the descendant solution without making a real evaluation.

offspring selection (Offspring Selection）
