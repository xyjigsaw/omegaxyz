word2vec

word2vec is a fully connected neural network with only one hidden layer. It trains all words in the corpus and generates corresponding word vectors (Word Embedding).WIThe size isVxN, Vis the size of the word dictionary, each input is a word, and N is the set hidden layer size. The word2vec model uses a neural network language model (Neural Network Language Model)Detailed explanation of word2vec https://omegaxyz.com/2018/11/13/word2vec/

Bag of Words model

The Bag-of-words model is a simplified expression model under natural language processing and information retrieval (IR). Under this model, text such as sentences or documents can be represented by a bag containing these words, regardless of grammar and word order. Recently, the bag-of-words model has also been applied in the field of computer vision. The bag-of-words model is widely used in document classification, and the frequency of word occurrences can be used as features for training classifiers. The origin of the word "bag of words" can be traced back to Zelig Harris's 1954 article in Distributional Structure.Continuous Bag of Words Model (CBOW)
  1. Remove the nonlinear hidden layer in the forward feedback neural network and directly connect the embedding layer of the middle layer to the softmax layer of the output layer;
  2. Ignore the sequence information of the context: all input word vectors are summarized into the same embedding layer;
  3. Put future words into context

Optimization of WMD

Now calculate the WMD distance between two documents. It is very time-consuming to calculate the distance using k-NN. If the document currently to be checked is followed byCentral query documentThe WMD lower bound is large enough to determine that it is not in the k-NN list of the query document, so just throw it away without spending time finding the WMD distance of the current document.Word Centroid Distance (WCD, defined at the last equal sign, X in the formula is a word vector, and X is a matrix composed of word vectors): Note that the above formula only needs to be calculated using absolute value inequality and WMD constraints. The derived formula only requires matrix operations, which greatly reduces calculation consumption.RWMD（Relaxed word moving distance ）First remove a constraint, calculate the corresponding WMD, and finally take the maximum value. The solutions to these two relaxed optimization problems correspond to the nearest neighbor problem in the row space and column space of the word vector matrix, and are also easy to calculate. Finally, RWMD is defined as the maximum value of the two objective values ​​of these two relaxed optimization problems.
