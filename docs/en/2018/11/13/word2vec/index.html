Introduction to word2vec in NLP

TechnologyMachine LearningNatural Language Processing神经网络Natural Language Processing

Introduction

Word2vec is a group of related models used to generate word vectors. These models are shallow, two-layer neural networks trained to reconstruct linguistic word text. The network is represented by words and needs to guess the input words at adjacent positions. Under the assumption of the word bag model in word2vec, the order of the words is not important. After training is completed, the word2vec model can be used to map each word to a vector, which can be used to represent the relationship between words. This vector is the hidden layer of the neural network. WIThe size isVxN, Vis the size of the word dictionary, each input is a word, and N is the hidden layer size you set.

One-hot encoding

One hot representation is very simple to represent word vectors, but it has many problems. When the number of words is extremely large, it will cause the curse of dimensionality. In fact, except for one position of such a vector, which is 1, the other positions are all 0, and the expression efficiency is very low.one-hot encoding python implementationhttps://omegaxyz.com/2018/04/22/one-hot/?hilite=%27one-hot%27 Dristributed representation can solve the problem of One hot representation. Its idea is to map each word to a shorter word vector through training. All these word vectors form a vector space, and ordinary statistical methods can be used to study the relationship between words. What is the dimension of this shorter word vector? This generally needs to be specified by ourselves during training.

Bag-of-words model

  • Predict a text from its context
The Bag-of-words model is a simplified expression model under natural language processing and information retrieval (IR). Under this model, text such as sentences or documents can be represented by a bag containing these words, regardless of grammar and word order. Recently, the bag-of-words model has also been applied in the field of computer vision. The bag-of-words model is widely used in document classification, and the frequency of word occurrences can be used as features for training classifiers. The origin of the word "bag of words" can be traced back to Zelig Harris's 1954 article in Distributional Structure.
Continuous Bag of Words Model (CBOW)
  1. Remove the nonlinear hidden layer in the forward feedback neural network and directly connect the embedding layer of the middle layer to the softmax layer of the output layer;
  2. Ignore the sequence information of the context: all input word vectors are summarized into the same embedding layer;
  3. Put future words into context
Mathematically, the CBoW model is equivalent to multiplying the vector of a bag-of-words model by an embedding matrix to obtain a continuous embedding vector. This is also the origin of the name of the CBoW model.

Skip-gram model

  • Predict context from a text
The Skip-gram model is a simple but very practical model. In natural language processing, corpus selection is a very important issue:
First, the corpus must be sufficient. On the one hand, the word size of the dictionary must be large enough, and on the other hand, it must contain as many sentences as possible that reflect the relationship between words. For example, only if the sentence pattern "fish swims in the water" is as common as possible in the corpus, the model can learn the semantic and grammatical relationships in the sentence. This is the same as humans learning natural language. The more times they are repeated, the more they will imitate it.
Second, the corpus must be accurate. In other words, the selected corpus can correctly reflect the semantic and grammatical relationships of the language. This seems not difficult to achieve. For example, in Chinese, the corpus of "People's Daily" is relatively accurate. However, more often than not, it is not the selection of corpus that causes concerns about accuracy, but the processing method. In the n-ary model, due to the limitation of the window size, the relationship between words beyond the window range and the current word cannot be correctly reflected in the model. Simply expanding the window size will increase the complexity of training. The Skip-gram model is proposed to solve these problems well. As the name suggests, Skip-gram is to "skip certain symbols." For example, the sentence "Chinese football is really bad" has four 3-gram phrases, namely "Chinese football is good", "football is really bad", "football is really bad", and "it's really bad". However, we found that the original meaning of this sentence is "Chinese football is so bad" but the above four 3-gram phrases cannot reflect this information. The Skip-gram model allows certain words to be skipped, so the 3-gram phrase "Chinese football is terrible" can be formed. If it is allowed to skip 2 words, it is 2-Skip-gram.
