Overview of the Cohen-Sutherland algorithm

技术域计算机图形学计算机图形学

Thought

By assigning a 4-bit binary code to any endpoint (x, y) according to the area where its coordinates are located, it is judged whether the graphic element falls within the clipping window and its internal part is found through an intersection operation.

encoding method

Note: l is left, r is right, b is bottom, t is top
  • (1) If x is less than wxl, D0=1, otherwise D0=0
  • (2) If x is greater than wxr, D1=1, otherwise D1=0
  • (3) If y is less than wyb, D2=1, otherwise D2=0
  • (4) If y is greater than wyt, D3=1, otherwise D3=0
When cutting a line segment, first find the codes code1 and code2 of the endpoints p1 and p2: (1) If code1 and code2 are both 0, it means that P1 and P2 are both within the window, then all line segments are located inside the window and should be taken. (2) If the result code1&code2 after the bitwise AND operation of code1 and code2 is not equal to 0, it means that P1 and P2 are above, below, left or right of the window at the same time, then all line segments are located outside the window and should be discarded. (3) When (1) and (2) are not satisfied, ensure that p1 is outside the window: if p1 is within the window, exchange the coordinate values ​​and codes of p1 and p2. (4) Find the intersection point of the straight line segment and the window boundary, and replace the coordinate value of p1 with the coordinate value of the intersection point. That is, the line segment is divided into two at the intersection point s. Considering that p1 is a point outside the window, p1s can be removed. Go to (2). (5) Use the straight line scan conversion algorithm to draw the current straight line segment p1p2. (6) The algorithm ends.

Algorithm flow chart

defect

(1) However, for line segments within the straight line of the clipping window, the intersection process is complicated and includes addition, multiplication, division, floating point operations, etc., which is not conducive to hardware implementation. (2) Due to the adoption of coding ideas, which involve bitwise AND operations, there are special requirements for program implementation; (3) The judgment of discarding all is only suitable for line segments that are only on the same side of the window (back left, or right, or top, or bottom). This algorithm is not very convenient for line segments within the straight line of the clipping window.
