IntelliJ IDEA creates Java-Web project

Programming LanguagesTechnologyWebJavaJavaWebjsp
Both eclipse and idea can create Java web projects. The following describes the steps to create a Java web project using idea.Things to prepare
  • intellij idea (including jdk)
  • Tomcat7.0+
Optional (if you need to build a servlet server or connect to a database)
  • jdbc extension package
  • sql server or mysql

①Create project

②Select Java->Java EE->web application

③Enter the project name and click finish

Next, enter the project interface. On the left is the project file directory. src generally contains Java code, and web contains interface (view) code.

④ If you have not created a web project before, you need to import the Tomcat server

Select Tomcat Server->local

⑤Select the Tomcat address and import it to Tomcat Home and Tomcat base directory

⑥Set the browser and port that are opened by default

⑦Run index.jsp

The default browser will open, OK
