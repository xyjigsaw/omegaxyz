Python uses the stuttering module to count word frequency in "Water Margin"

Programming LanguagesTechnologyPythonPython
Chinese word segmentation is a basic task in Chinese text processing. Stuttering word segmentation is used to perform Chinese word segmentation. Its basic implementation principles have three points:
  1. based onTrie treeThe structure implements efficient word graph scanning and generates a directed acyclic graph (DAG) composed of all possible word formations of Chinese characters in the sentence.
  2. Adopteddynamic programmingFind the maximum probability path and find the maximum segmentation combination based on word frequency
  3. For unregistered words, a method based on the ability of Chinese characters to form words is used.HMM model, using the Viterbi algorithm
The following statistics are made using the word frequency of the stuttering word segmentation team Water Margin. Code:
#!/usr/bin/python    
# -*- coding:utf-8 -*-    

import sys

import jieba
import jieba.analyse
import xlwt  # 写入Excel表的库

if __name__ == "__main__":

    wbk = xlwt.Workbook(encoding='ascii')
    sheet = wbk.add_sheet("wordCount")  # Excel单元格名字
    word_lst = []
    key_list = []
    for line in open('C:\\Users\\dell\\desktop\\1.txt'):  # 1.txt是需要分词统计的文档
        try:
            item = line.strip('\n\r').split('\t')  # 制表格切分
            # print(item)
            tags = jieba.analyse.extract_tags(item[0])  # jieba分词
            for t in tags:
                word_lst.append(t)
        except:
            pass

    word_dict = {}
    with open("wordCount.txt", 'w') as wf2:  # 打开文件

        for item in word_lst:
            if item not in word_dict:  # 统计数量
                word_dict[item] = 1
            else:
                word_dict[item] += 1

        orderList = list(word_dict.values())
        orderList.sort(reverse=True)
        # print orderList  
        for i in range(len(orderList)):
            for key in word_dict:
                if word_dict[key] == orderList[i]:
                    wf2.write(key + ' ' + str(word_dict[key]) + '\n')  # 写入txt文档
                    key_list.append(key)
                    word_dict[key] = 0

    for i in range(len(key_list)):
        sheet.write(i, 1, label=orderList[i])
        sheet.write(i, 0, label=key_list[i])
    wbk.save('wordCount.xls')  # 保存为 wordCount.xls文件
