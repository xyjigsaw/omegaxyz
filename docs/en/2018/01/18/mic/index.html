Detailed explanation of maximum mutual information coefficient (MIC)

TechnologyMachine Learning信息论
maximum information coefficientmaximal information coefficient (MIC), also known as the maximum mutual information coefficient Purpose:Maximal Information Coefficient (MIC) is the maximum information coefficient, which belongs to Maximal Information-based Nonparametric Exploration (MINE). It is the largest information-based non-parametric exploration and is used to measure the linear or non-linear strength of two variables X and Y.The MIC metric is universal. It can not only discover linear functional relationships between variables, but also nonlinear functional relationships (exponential, periodic); it can not only discover functional relationships, but also non-functional relationships (such as the superposition of functional relationships, or interesting graphic patterns).The MIC metric is balanced. For functional or non-functional relationships with the same noise level, the MIC metric has approximate values. Therefore, the MIC measure can not only be used to compare the strength of the same correlation relationship vertically, but also to compare the strength of different relationships horizontally.advantage:Generality: With enough statistical samples, a wide range of relationships can be captured without being limited to specific function types (such as linear, exponential, periodic, etc.). Equitability: Give similar scores to relationships with equal levels of noise of different types.shortcoming:"The statistical ability of MIC has been questioned by some. When the null hypothesis does not hold, the statistics of MIC will be affected. This problem does not exist in some data sets, but it does exist in some data sets."Earliest source:http://science.sciencemag.org/content/334/6062/1518.full How to calculate MIC measure. A set of data points with two attributes is distributed in a two-dimensional space. Use an m times n grid to divide the data space, so that the frequency of the data points falling in the (x, y)-th grid is used as an estimate of P(x, y)., so that the frequency of the data point falling in the x-th row is used as an estimate of P(x), and similarly, the estimate of P(y) is obtained. Then calculate the mutual information of random variables X and Y. Because an m by n grid can divide the data points in more than one way, we want to obtain a grid that maximizes mutual information. Then use the normalization factor to convert the mutual information value into the (0,1) interval. Finally, the grid resolution that maximizes the normalized mutual information is found as a measure of MIC. where the resolution of the grid is limited to m x n < B,. Given i and j, grid the scatter plot composed of XY into columns i and rows j, and find the maximum mutual information value 2. Normalize the maximum mutual information value 3. Select the maximum value of mutual information at different scales as the MIC value(1) The red gridding scheme divides all data points into four areas: upper left, upper right, lower left, and lower right. The number of data points corresponding to each area is 1, 4, 4, 1. Normalize the number of data points to obtain the frequency of data points in four regions, which are 0.1, 0.4, 0.4, and 0.1 respectively. In other words, at this time, X has two values: left and right, and Y has two values: up and down. P(X=left,Y=top)=0.1, P(X=right,Y=top)=0.4, P(X=left,Y=bottom)=0.4, P(X=right,Y=bottom)=0.1. And, P(X=left)=0.5, P(X=right)=0.5, P(Y=top)=0.5, P(Y=bottom)=0.5.(2) Normalize the maximum mutual information value Divide the maximum mutual information obtained by log(min(X,Y)), which is normalization! (3) Select the maximum value of mutual information at different scales as the MIC value The above describes the calculation method of M(X,Y,D,i,j) given i and j. This step is to give many (i,j) values, calculate the value of M(X,Y,D,i,j) in each case, and use the largest value among all M(X,Y,D,i,j) as the MIC value. Note that (i, j) here is conditional and must be satisfied. n represents the data volume of data set D. Of course, you can set the value of B(n) yourself. Here is the value that others think has the best effect after experiments.
