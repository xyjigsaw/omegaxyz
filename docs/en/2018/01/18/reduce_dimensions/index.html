Feature selection and feature extraction (dimensionality reduction)

TechnologyMachine Learning特征选择
Feature selection and dimensionality reduction 1. Similarities and differences Feature selection and dimensionality reduction have some similarities. The effect achieved by the two is the same, which is to try to reduce the number of attributes (or features) in the feature data set; but the methods used by the two are different: the dimensionality reduction method mainly uses the relationship between attributes, such as combining different attributes to obtain new attributes, thus changing the original feature space; while the feature selection method is to select a subset from the original feature data set, which is an inclusive relationship and does not change the original feature space. 2. Main methods of dimensionality reduction Principal Component Analysis (principal component analysis), see "Simple and easy-to-learn machine learning algorithm-Principal Component Analysis (PCA)" for details Singular Value Decomposition (singular value decomposition), see "Simple and easy-to-learn machine learning algorithm - SVD singular value decomposition" for details Sammon's Mapping(Sammon mapping)Feature extraction and feature selection both find the most effective features (invariance of similar samples, discriminability of different samples, robustness to noise) from the original features. Feature extraction: Convert original features into a set of features with obvious physical meaning (Gabor, geometric features [corner points, invariants], texture [LBP HOG]) or statistical significance or kernel Feature selection: Select a set of features with the most statistical significance from the feature set to achieve dimensionality reduction. Both functions: 1 Reduce data storage and input data bandwidth 2 Reduce redundancy 3. Classification tends to improve at low latitudes 4. Can discover more meaningful potential variables to help develop a deeper understanding of the data.
