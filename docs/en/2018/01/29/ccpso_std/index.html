Random fixed group cooperative coevolution PSO algorithm (CCPSO)

Programming LanguagesTechnologyMATLABMachine LearningEvolutionary AlgorithmsPSO算法MATLAB协同进化
Cooperative Coevolution is an effective method for solving large-scale optimization algorithms. Break large-scale problems into sets of smaller sub-problems. The key to cooperative coevolution is the decomposition strategy. For cooperative coevolution algorithms, please see:https://omegaxyz.com/2017/10/14/cooperative_coevolution/PSO algorithm is particle swarm optimization algorithm. This article is about cooperative co-evolution of random fixed groups using PSO to optimize. For example, there are 12 decision variables, and we fix and randomly optimize 3 decision variables, then the decision variables are divided into 4 groups. MATLAB main function code:
clear;
clc;
format long;
%------给定初始化条件--------
global M
global bound
MaxDT=100;       %最大迭代次数
global Dim
Dim=22;             %搜索空间维数（未知数个数）
sub_dim= 11 ;
M=30;             %初始化群体个体数目
bound=1;
%global answer      %最后所有粒子的结果（包括特征与精确度）
x=randn(M,Dim); %随机初始化位置
v=randn(M,Dim); %随机初始化速度
result = 1;
while MaxDT ~= 0
    subgroup = rnd_divide(Dim, sub_dim);
    for i=1:length(subgroup)
        [sub_x, sub_v, temp_result] = PSO(x(:,subgroup{i}), v(:,subgroup{i}), sub_dim, subgroup{i});
        x(:,subgroup{i}) = sub_x;
        v(:,subgroup{i}) = sub_v;
        if(temp_result < result)
            result = temp_result;
        end
    end
    %可以在协同进化后进行一次全局优化
    %[x, v, temp_result] = PSO(x, v, Dim);
    %if(temp_result < result)
    %        result = temp_result;
    %end
    MaxDT =MaxDT - 1;
end
random grouping algorithm
% random grouping
function group = rnd_divide(dim, subdim)
   dim_rand = randperm(dim);
   group = {};
   for i = 1:subdim:dim
      index = dim_rand(i:i+subdim-1);
      group = {group{1:end} index};
   end
end
Other functional dependencies are the same as the PSO algorithm, see:https://omegaxyz.com/2018/01/17/matlab_pso/
