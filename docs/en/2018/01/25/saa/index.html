Simulated annealing algorithm (SAA) implemented in C language and MATLAB

Programming LanguagesC & C++TechnologyMATLABMachine LearningEvolutionary AlgorithmsCMATLAB贪心算法

mountain climbing

Before introducing the simulated annealing algorithm, let us first introducemountain climbing. Hill climbing is a greedy algorithm. The goal is to find the maximum value of the function. If the initial point is at C during initialization, the nearby local maximum point A will be found. Since point A is a local maximum point, for the mountain climbing method, the algorithm cannot jump out of the local maximum point. If the initial point is selected at D, according to the hill climbing method, all maximum value points B will be found. This also shows that whether such a greedy-based hill-climbing method can obtain the global optimal solution has a great relationship with the selection of the initial value.The idea of ​​the Simulated Annealing Algorithm (SAA) is borrowed from the annealing principle of solids. When the temperature of the solid is very high, the internal energy is relatively large, and the internal particles of the solid are in rapid and disorderly motion. When the temperature slowly decreases, the internal energy of the solid decreases, and the particles gradually become more orderly. Finally, when the solid is at normal temperature, the internal energy reaches the minimum, and at this time, the particles are the most stable. The simulated annealing algorithm is designed based on this principle. The simulated annealing algorithm starts from a certain higher temperature, which is called the initial temperature. As the temperature parameters continue to decrease, the solution in the algorithm tends to be stable. However, such a stable solution may be a local optimal solution. At this time, the simulated annealing algorithm will jump out of such a local optimal solution with a certain probability to find the global optimal solution of the objective function. As shown in the figure above, if the solution at point A is found at this time, the simulated annealing algorithm will jump out of this solution with a certain probability. For example, it jumps to point D and search again. This increases the possibility of finding the global optimal solution to a certain extent.

Simulated annealing algorithm process

(1) Randomly select a unitk, and give it a random displacement, find the resulting energy change in the systemΔEk. (2)IfΔEk⩽0, this displacement can be adopted, and the changed system state can be used as the starting point for the next change; IfΔEk>0, the probability that the shifted state can be adopted is
 
in the formulaTis the temperature, then from(0,1)Select a number from uniformly distributed random numbers in the intervalR,likeR<Pk, then the changed state will be used as the starting point for the next time; otherwise, the state before the change will be used as the starting point for the next time. (3) Go to step (1) and continue until an equilibrium state is reached.

Simulated annealing algorithm MATLAB implementation

Calculate the maximum value of -x^2-4x+3:Main function:
clc;
clear;
k = 0.1;
r = 0.9;
T = 2000;
T_min = 20;
glb = -4.0;
lub = 4.0;
gen = 100000;
best = func(rnd(glb,lub));
best_min = 9999;
while(T>T_min)
    for i=1:gen
        current = func(rnd(glb,lub));
        dE = current - best;
        if(dE>=0)
            best = current;
            best_min = current;
        else
            if(exp(dE/(T*k))>rnd(0,1))
                best = current;
            end
        end
    end
    T = r * T;
end
Random number generation function:
function y = rnd(glb, lub)
y = glb + (lub-glb)*rand();
end
Objective function:
function y = func(x)
y = -x*x-4*x+3;
end

Simulated annealing algorithm C language implementation

Calculate the maximum value of -x^2-4x+3: 
#include <stdio.h>
#include <stdlib.h>
#include <math.h>
#include <time.h>

#define num 1000000 //迭代次数

double k=0.1;
double r=0.9; //用于控制降温的快慢
double T=2000; //系统的温度，系统初始应该要处于一个高温的状态
double T_min =2;//温度的下限，若温度T达到T_min，则停止搜索
//返回指定范围内的随机浮点数
double rnd(double dbLow,double dbUpper)
{
    double dbTemp=rand()/((double)RAND_MAX+1.0);
    return dbLow+dbTemp*(dbUpper-dbLow);
}
double func(double x)//目标函数
{
    return -x*x-4*x+3;
}
int main()
{
    double best=func(rnd(-4,4));
    double dE,current;
    double gbest=0;
    int i;
    while( T > T_min )
    {
        for(i=0;i<num;i++)
        {
        //用当前时间点初始化随机种子，防止每次运行的结果都相同
        time_t tm;
        time(&tm);
        unsigned int nSeed=(unsigned int)tm;
        srand(nSeed);

        current=func(rnd(-4,4));
        dE = current - best ;

        if ( dE >= 0 ) //表达移动后得到更优解，则总是接受移动
            best = current ;
            gbest = current;

        }
        T = r * T ;//降温退火 ，0<r<1 。r越大，降温越慢；r越小，降温越快
    }
    printf("最大值是 %f\n",best);
    printf("gbest为：%f\n",gbest);
    return 0;
}
