How to install whl file in python

Programming LanguagesTechnologyPythonPython
Many third-party components of Python are whl files. How should you install such a whl file? Today I will introduce how to install a whl file. First, I recommend an informal third-party whl file package website:https://www.lfd.uci.edu/~gohlke/pythonlibs/Install belowwordcloudHere is an example of how to install the whl file ① First check your python version. The python version I use is 3.6.1 32-bit, and then select the corresponding package to download.②After the download is completed, open cmd, use pip to install wheel and execute the commandpip install wheelIf you are prompted that pip is "not an internal command", install pip first ③After installing wheel, install the whl file again. First switch to the desktop (the file I downloaded is on the desktop) and find the file to be installed. execute commandcd Desktop④Execute commandpip install wordcloud-1.3.3-cp36-cp36m-win32.whlTo install ⑤Everything will be fine once you see this.Just import wordcloud directly when writing at this time!
