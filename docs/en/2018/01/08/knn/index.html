Overview of KNN algorithm

TechnologyMachine Learning

1. Concept description of kNN

The kNN algorithm is also called the k-nearest neighbor classification algorithm. The so-called k nearest neighbor refers to the closest k neighbors (data), that is, each sample can be expressed by its K neighbors. The core idea of ​​the kNN algorithm is that in a space containing unknown samples, the data type of the sample can be determined based on the data types of the k samples closest to the sample. This algorithm involves 3 main factors: training set, distance and similarity measurement, and the size of k; Main considerations: distance and similarity;

2. Examples

In the picture on the right, which class should the green circle be assigned to? Is it a red triangle or a blue square? If K=3, since the proportion of the red triangle is 2/3, the green circle will be assigned to the red triangle class. If K=5, since the proportion of the blue square is 3/5, the green circle will be assigned to the blue square class.

3. Characteristics of kNN algorithm

The KNN algorithm can not only be used for classification, but also for transition, such as taking a transition color between two chromaticities. The KNN algorithm currently mainly uses the weighted voting method, that is, the votes of nearby neighbors are weighted according to the distance. The closer the distance, the greater the weight (the weight is the reciprocal of the square of the distance).advantage:It is easy to implement, does not require parameter estimation, does not require training, supports incremental learning, and can model complex hyperpolygonal decision spaces;shortcoming:That is, the amount of calculation is large and the analysis speed is slow;
