30 Google open source projects

TechnologyIdeas其它Developer LifegoogleTensorFlow-Pytorch

1machine learning system TensorFlow

TensorFlow is Google's second-generation machine learning system. According to Google, in some benchmark tests, TensorFlow performs 2 times faster than the first-generation DistBelief. TensorFlow has built-in extended support for deep learning, and any calculation that can be expressed using a computational flow graph can use TensorFlow. Any gradient-based machine learning algorithm can benefit from TensorFlow’s auto-differentiation. Expressing ideas in TensorFlow is also easy through the flexible Python interface.Chinese community:http://www.tensorfly.cn/

2material-design-icons

Google has open sourced 750 glyphs from the Material Design system icon pack. The system icons include commonly used icons for media playback, communication, content editing, connections, etc. In web applications, both Android and iOS designs are available.

3Front-end development tools group MDL

MDL (Material Design Lite) is a website front-end development tool set launched by Google. Material Design Lite (MDL) allows you to add a Material Design look and feel to your static content website without relying on any Java frameworks and libraries. MDL can optimize the cross-device usage experience, smoothly switch between old browsers, and provide a very fast access experience.

4Web front-end framework Angular

Angular is a very popular and easy-to-use web front-end framework currently maintained by Google. Angular 2 and the previous version Angular.js have been officially maintained separately (the GitHub addresses and project homepages of the two are different). Progressive web apps leverage the power of a modern web platform to deliver app-like experiences. High performance, offline, zero installation.

5Container cluster management system Kubernetes

Kubernetes is an open source container cluster management system from Google Cloud Platform. Build a container scheduling service based on Docker. The system can automatically select a worker container in a container cluster for use. Its core concept is Container Pod.

6data description language protobuf

Protocol Buffers (ProtocolBuffer/protobuf) is a data description language developed by Google. It is similar to XML and can serialize structured data and can be used for data storage, communication protocols, etc. At this stage, it supports three programming languages ​​including C++, JAVA, and Python. Protocol buffers have many advantages over XML for serializing structured data.

7Java Commonly used libraries Guava

This project is an open source project of Google and contains many of Google's core Java common libraries.

8Spark

Spark is a development environment based on the Chrome browser launched by Google. Provides a set of reusable UI components. Developed using Dart.

9RPC Frame GRPC

GRPC is a high-performance, open source, and general-purpose RPC framework designed for mobile and HTTP/2. Currently, C, Java and Go language versions are provided, namely: grpc, grpc-java, grpc-go. The C version supports C, C++, Node.js, Python, Ruby, Objective-C, PHP and C#. GRPC is designed based on the HTTP/2 standard and brings features such as bidirectional flow, flow control, header compression, and multiplexing requests on a single TCP connection. These features make it perform better on mobile devices and save power and space.

10High quality image compression algorithm tool Guetzli

Guetzli, a JPEG encoder for digital images and web images, enables a faster online experience by producing smaller JPEG files while maintaining compatibility with current browsers, image processing applications and JPEG standards. Google says Guetzli creates high-quality JPEG image files that are 35 percent smaller than current compression methods. The image above is a 16x16 pixel sample of a telephone line hanging against a blue sky, displaying distortion often encountered with traditional JPEG algorithms. The uncompressed original image is on the left, the smaller libjpeg in the middle, and Guetzli with less distortion on the right.

11k/vDatabase Leveldb

Leveldb is a very efficient kv database implemented by Google. The current version 1.2 can support billion-level data volume. It still has very high performance at this level, mainly due to its good design. Especially the LSM algorithm.

12Automated web audit tools Lighthouse

Lighthouse is an open source automation tool for improving the quality of web applications. You can run it as a Chrome extension, or from the command line. When you give Lighthouse a URL to review, it will run a series of tests against the page and then generate a report on the page's performance. You can refer to failed tests to see what steps you can take to improve your app.

13Gson

Gson is a Java library provided by Google for mapping between Java objects and JSON data. You can convert a JSON string into a Java object and vice versa.

14Minimal system loading tool systemjs

systemjs is a minimal system loading tool used to create plug-ins to handle alternative scene loading processes, including loading CSS scenes and images, mainly running in the browser and NodeJS. It is an extension to the ES6 browser loader that will be applied in the local browser. Usually the name of the created plug-in is the module itself. If no purpose is specifically specified, the default plug-in name is the extension name of the module.

15serialization library FlatBuffers

FlatBuffers is a Java serialization library for games and other memory-constrained applications. FlatBuffers allow you to access serialized data directly without the need for decompression and parsing. Provides strong forward and backward compatibility at the same time. FlatBuffers supports C++ and Java languages ​​without relying on third-party library support.

16Xi text editor Xi Editor

Xi Editor is a Google open source text editor written in the Rust language. It was originally built for Mac OS X and uses Cocoa as the user interface. There are plans to adapt to other platforms. All edit operations can be submitted and processed within 16ms. Text drawing uses the best technologies (like Core Text on Mac, DirectWrite on Windows, etc.) with full Unicode support.

17Grumpy

Grumpy is a Python to Go source code translation compiler and runtime designed to replace CPython 2.7. The key difference is that it compiles Python source code to Go source code and then compiles it to native code, not bytecode. This means Grumpy doesn't have a VM. Compiled Go source code is a sequence of calls to the Grumpy runtime, a Go library that serves a similar purpose to the Python C API (although the C API is not directly supported).

18Java compiler Traceur

Traceur is a Java compiler from Google, through which you can experience some new and interesting Java language features, most of which are ECMA standards or drafts that have not been implemented by current browsers, such as: array comparison, classes, modules, iterators, method parameter default values, Promise, etc.

19media player ExoPlayer

ExoPlayer is an application-level media player on Android. It provides a candidate for the Android MediaPlayer API to play local or online video and audio. ExoPlayer supports some features not available in the Android MediaPlayer API, including DASH and SmoothStreaming adaptive playback, persistent caching, and custom renderers. Unlike the MediaPlayer API, EXOPlayer is easy to customize and extend, and it can be upgraded through Play Store updates.

20Code building tools Bazel

Bazel is a reproducible code building tool from Google. It is mainly used to build Google's software and deal with the construction problems that occur in Google's development environment, such as: large-scale data construction problems, shared code base problems, and problems related to software built from source code. It supports multiple languages ​​and is cross-platform. It also supports automated testing and deployment, and features reproducibility and scale.

21Machine intelligence that creates art and composes music magenta

A group of researchers from the Google Brain team released a project, Project Magenta, whose main goal is to use machine learning to create art and compose music. Project Magenta uses the TensorFlow system, and the researchers open sourced their models and tools on GitHub.

22Automatically generate a content library for the command line interface Python Fire

Python Fire is a library open sourced by Google that can automatically generate a command line interface (CLI) from any Python code. Python Fire is a simple way to create a CLI in Python; is a useful tool for developing and debugging Python code; makes converting between Bash and Python easier; and makes using the Python REPL easier by setting up the REPL with the modules and variables you need to import and create.

23Software and hardware framework AnyPixel.js

AnyPixel.js is a software and hardware framework open sourced by Google that can be used to build various displays composed of "pixels". Each pixel can be any interactive entity object, such as LED lights, balloons, etc.

24IoT tools Physical Web

Physical Web is a project led by the Chrome team, which aims to use URLs to connect the world and facilitate users to receive data. In the Web world, various URLs can be said to be the basis of links and are decentralized. Therefore, what "The Physical Web" has to do is to let each smart device identify itself with a URL, and then users can interact with the device through the URL according to their own needs. In this way, your experience of using a smart device is almost the same as using various hyperlinks on a website.

25Docker Container performance analysis tools cAdvisorc

Advisor is a tool used by Google to analyze the resource usage and performance characteristics of running Docker containers. cAdvisor is a running daemon used to collect, aggregate, process and export information related to running containers. Each container maintains independent parameters, historical resource usage and complete resource usage data. Currently supports lmctfy containers and Docker containers.

26C++Unit testing tools

Google's open source C++ unit testing framework Google Test, referred to as gtest, is a very good unit testing framework. Supports cross-platform and some mobile operating systems including Windows CE and Symbian.

27IndexedDB relational query engine lovefield

lovefield is a relational query engine built on IndexedDB. It provides SQL-like syntax and works across browsers (currently supporting Chrome 37 and above, Firefox 31 and above, IE 10 and above).

28Java Generator source code collection Auto

Auto is a collection of Java generator source code. Java has a lot of mechanical, repetitive, untested code, and sometimes subtle bugs. Auto projects are a collection of code generators that automate these types of tasks so they can create the code you want to write without bugs.

29Python Code formatting tools

YAPF is a tool developed by Google to format Python code.

30based on TensorFlow neural network library Sonnet

The Sonnet library uses an object-oriented approach, allowing the creation of modules that define some forward conduction calculations. The module is called with some input Tensors, adds operations to the graph and returns an output Tensor. One design choice is to ensure that variable sharing is handled transparently by automatically reusing variables on subsequent calls to the same module. The library is compatible with Linux/Mac OS X and Python 2.7. TensorFlow version must be at least 1.0.1. Sonnet supports TensorFlow's virtualenv installation mode, as well as nativ pip installation.
