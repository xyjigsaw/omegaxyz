MATLAB toolbox k-means algorithm

Programming LanguagesTechnologyMATLABMachine LearningMATLABk-means
MATLAB contains the kmeans algorithm clustering toolreturn valueIdx N*1 vector, which stores the clustering label of each point The matrix of Ctrs K*P stores the K cluster centroid positions. SumD 1*K sum vector, which stores the sum of the distances between all points between classes and the centroid point of the class D N*K matrix stores the distance between each point and all centroids;parameter1. ‘Distance’ (distance measure) ‘sqEuclidean’ Euclidean distance (by default, this distance method is used) ‘cityblock’ absolute error sum, also known as: L1 ‘cosine’ for vectors ‘correlation’ for values with sequential relationships ‘Hamming’ only works with binary data 2. ‘Start’ (initial center of mass position selection method) ‘sample’ randomly selects K centroid points from X ‘uniform’ uniformly and randomly generates K centroids according to the distribution range of X ‘cluster’ randomly selects 10% of X’s subsamples in the initial clustering stage (this method initially uses the ‘sample’ method) matrix provides a K*P matrix as the initial set of centroid locations. 3. ‘Replicates’ (number of clustering repetitions) integerApplication code: 
%随机获取150个点
X = [randn(50,2)+ones(50,2);randn(50,2)-ones(50,2);randn(50,2)+[ones(50,1),-ones(50,1)]];
%X为二维下50个点的坐标
opts = statset('Display','final');

%调用Kmeans函数
%X N*P的数据矩阵
%Idx N*1的向量,存储的是每个点的聚类标号
%Ctrs K*P的矩阵,存储的是K个聚类质心位置
%SumD 1*K的和向量,存储的是类间所有点与该类质心点距离之和
%D N*K的矩阵，存储的是每个点与所有质心的距离;

[Idx,Ctrs,SumD,D] = kmeans(X,3,'Replicates',3,'Options',opts);

%画出聚类为1的点。X(Idx==1,1),为第一类的样本的第一个坐标；X(Idx==1,2)为第二类的样本的第二个坐标
plot(X(Idx==1,1),X(Idx==1,2),'r.','MarkerSize',14)
hold on
plot(X(Idx==2,1),X(Idx==2,2),'b.','MarkerSize',14)
hold on
plot(X(Idx==3,1),X(Idx==3,2),'g.','MarkerSize',14)

%绘出聚类中心点,kx表示是圆形
plot(Ctrs(:,1),Ctrs(:,2),'kx','MarkerSize',14,'LineWidth',4)
plot(Ctrs(:,1),Ctrs(:,2),'kx','MarkerSize',14,'LineWidth',4)
plot(Ctrs(:,1),Ctrs(:,2),'kx','MarkerSize',14,'LineWidth',4)

legend('Cluster 1','Cluster 2','Cluster 3','Centroids','Location','NW')
