Ant colony algorithm (ACO) MATLAB implementation

Programming LanguagesTechnologyMATLABMachine LearningEvolutionary AlgorithmsMATLAB贪心算法ACO算法

(1) The origin of ant colony algorithm

Ant colony optimization (ant colony optimization) was first proposed by Marco Dorigo and others in 1991. In the process of researching new algorithms, they discovered that when ant colonies are looking for food, they can quickly find the target by exchanging foraging information by secreting a biological hormone called pheromone. Based on this, they proposed an ant colony algorithm based on the principle of positive information feedback. The basic idea of ​​the ant colony algorithm comes from the shortest path principle of ants foraging in nature. According to the observations of insect scientists, it is found that although ants in nature have underdeveloped vision, they can find the shortest path from the food source to the nest without any prompts, and adaptively search for new best paths after the surrounding environment changes. When ants are looking for food sources, they can release a hormone called pheromone along the path they walk, so that other ants within a certain range can detect it. When more and more ants pass through some paths, there will be more and more pheromones, and the probability that the ants will choose this path will be higher. As a result, the pheromones on this path will increase, and the probability of ants taking this path will increase, and life will continue. This selection process is known as the autocatalytic behavior of ants. For a single ant, it does not want to find the shortest path, but only chooses based on probability; for the entire ant colony system, they have achieved the objective effect of finding the optimal path. This is swarm intelligence.

(2) What can the ant colony algorithm do?

Ant colony algorithm is a bionic algorithm designed based on simulating the shortest path behavior of ants looking for food. Therefore, generally speaking, ant colony algorithm is used to solve the shortest path problem, and it has really achieved better results in the Traveling Salesman Problem (TSP, a problem of finding the shortest path). At present, it has been gradually applied to other fields, including graph coloring problems, vehicle scheduling problems, integrated circuit design, communication networks, data cluster analysis, etc.

(3) Ant colony algorithm implementation

The optimized function isF（x,y）= -(x.^2+3*y.^4-0.2*cos(3*pi*x)-0.4*cos(4*pi*y)+0.6) 
clear
clc
Ant = 300;%蚂蚁数量
Times = 80;%移动次数
Rou = 0.9;%荷尔蒙发挥系数
P0 = 0.2;%转移概率
Lower_1 = -1;%搜索范围
Upper_1 = 1;
Lower_2 = -1;
Upper_2 = 1;

for i=1:Ant
    X(i,1)=(Lower_1+(Upper_1-Lower_1)*rand);
    X(i,2)=(Lower_1+(Upper_2-Lower_2)*rand);
    Tau(i)=F(X(i,1),X(i,2));
end

step=0.05;
f='-(x.^2+3*y.^4-0.2*cos(3*pi*x)-0.4*cos(4*pi*y)+0.6)';

[x,y]=meshgrid(Lower_1:step:Upper_1,Lower_2:step:Upper_2);
z=eval(f);
figure(1);
subplot(1,2,1);
mesh(x,y,z);
hold on;
plot3(X(:,1),X(:,2),Tau,'k*')
hold on;
text(0.1,0.8,-0.1,'蚂蚁的初始位置分布');
xlabel('x');ylabel('y');zlabel('f(x,y)');

for T=1:Times
    lamda=1/T;
    [Tau_Best(T),BestIndex]=max(Tau);
    for i=1:Ant
        P(T,i)=(Tau(BestIndex)-Tau(i))/Tau(BestIndex);%计算转移状态概率
    end
    for i=1:Ant
        if P(T,i)<P0%局部搜索
            temp1=X(i,1)+(2*rand-1)*lamda;
            temp2=X(i,2)+(2*rand-1)*lamda;
        else%全局搜索
            temp1=X(i,1)+(Upper_1-Lower_1)*(rand-0.5);
            temp2=X(i,2)+(Upper_2-Lower_2)*(rand-0.5);
        end
        if temp1<Lower_1%越界处理
            temp1=Lower_1;
        end
        if temp1>Upper_1
            temp1=Upper_1;
        end
        if temp2<Lower_2
            temp2=Lower_2;
        end
        if temp2>Upper_2
            temp2=Upper_2;
        end
        
        if F(temp1,temp2)>F(X(i,1),X(i,2))%更新位置
            X(i,1)=temp1;
            X(i,2)=temp2;
        end
    end
    for i=1:Ant
        Tau(i)=(1-Rou)*Tau(i)+F(X(i,1),X(i,2));%更新荷尔蒙
    end
end

subplot(1,2,2);
mesh(x,y,z);
hold on;
x=X(:,1);
y=X(:,2);
plot3(x,y,eval(f),'k*');
hold on;
text(0.1,0.8,-0.1,'蚂蚁的最终位置分布');
xlabel('x');ylabel('y');zlabel('f(x,y)');   

[max_value,max_index]=max(Tau);
maxX=X(max_index,1);
maxY=X(max_index,2);
maxValue=F(X(max_index,1),X(max_index,2));
Optimization function:
function f = F(x,y)
f = -(x.^2+3*y.^4-0.2*cos(3*pi*x)-0.4*cos(4*pi*y)+0.6);
end
