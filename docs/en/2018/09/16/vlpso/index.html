Overview of high-dimensional feature selection algorithm (VLPSO) based on variable length PSO

TechnologyMachine LearningEvolutionary AlgorithmsPSO算法特征选择

Introduction

VLPSO is a PSO algorithm that solves high-dimensional feature selection published by Binh Tran, Bing Xue and others in the top artificial intelligence magazine IEEE Transactions on Evolutionary Computation. It can solve high-dimensional feature selection problems very well, especially the calculation time can be greatly reduced.

Question raised

  • ①High-dimensional feature selection, occupying memory
  • ②High-dimensional feature selection, time-consuming

Main contributions of VLPSO (Variable Length PSO)

  • ①Proposal of variable-length single-objective PSO representation method
  • ②Adapted particle dimension value interactive learning method, borrowed from CLPSO

Representation of particles

Exemplar: The particles that need to be learned in each dimension of the current individualindex Learning Probability: Learning rate, learn when reaching the thresholdExemplarvalue, otherwise learn its ownpbest Renew Exemplar Count： rightExemplarThe reset algebra of Note that the reference here isCLPSOmethod:The only difference isPccalculation methodFor example, the 6 in the Exemplar line indicates that the 6th dimension of the 6th individual needs to be learned when updating the 4th dimension of the current particle.

Exemplargeneration

Take the following operations for the Exemplar of each dimension of each individual
  • Calculate the learning rate Pc
  • If the value of Pc is less than a random number, the Exemplar of the current dimension is the index of its own particle.
  • otherwise
  • Randomly select two particles, and the index of the particle with the larger fitness value is used as the exemplar of the current dimension.

learning ratePcCalculation method

0.05 and 0.45 are the empirical values proposed in CLPSO Rank(i) is the ranking of particles, which is calculated by the symmetric uncertainty of information theory The larger the learning rate, the greater the possibility of learning its own particles.

Population length division

PopSize=100, NbrDiv is a preset value, let it equal 5, then the population is divided into 5 groups, each group has 20 individuals.
