OpenGL numerical differentiation method (DDA) draws a straight line

高级语言C&C++技术域计算机图形学C++C语言计算机图形学openGL
The DDA algorithm is a method in computer graphics that generates straight lines based on differential equations of straight lines. Due to floating point operations and rounding, this algorithm is not conducive to hardware implementation. The main idea is to use the straight line formulay = kx + bderived. We know the two endpoints of the straight line segment P0(x0,y0) and P1(x1,y1), and we can find k and b. Under the condition that k and b are both found, as long as we know an x ​​value, we can calculate a y value. If the step of x is 1 (x adds 1 each time, that is, x = x +1), then the step of y is k+b; you can also calculate the x value by knowing a y value. At this time, the step of y is 1 and the step of x is (1-b)/k. According to the calculated x value and y value, round down to get the coordinates (x’, y’), and draw a point on the straight line segment at (x’, y’). To further simplify the calculation, b can usually be set to 0 and the starting point is regarded as (0,0). Assuming that the current point is (xi, yi), the calculation formula for solving (xi+1, yi+1) using the DDA algorithm can be summarized as: xi+1 = xi + xStep (1) yi+1 = yi + yStep (2) We generally determine xStep and yStep by calculating Δx and Δy: If Δx > Δy, it means that the maximum difference of the x-axis is greater than the maximum difference of the y-axis. The x-axis direction is the main direction of the step, xStep = 1, yStep = k; If Δy > Δx, it means that the maximum difference of the y-axis is greater than the maximum difference of the x-axis. The y-axis direction is the main direction of the step, yStep = 1, xStep = 1/k. According to this formula, (xi+1, yi+1) can be calculated iteratively through (xi, yi), and then the calculated (x, y) coordinate point can be drawn in the coordinate system.C++ code: 
#include <GL/GLUT.H>
#include <iostream>
#include <math.h>


//数值微分法
using namespace std;

float xs = 0.0;
float ys = 0.0;
float xe = 0.0;
float ye = 0.0;

void LineDDA(int x0, int y0, int x1, int y1)
{
	float x = 0.0;
	float y = 0.0;
	float m = 0.0;
	float dx = x1 - x0;
	float dy = y1 - y0;
	if (dx != 0)
	{
		m = dy / dx;
		if (m <= 1 && m >= -1)
		{
			y = y0;
			for (x = x0; x <= x1; x++)
			{
				glVertex2i(x, int(y + 0.5));
				y += m;
			}
		}
		if (m>1 || m<-1)
		{
			m = 1 / m;
			x = x0;
			for (y = y0; y <= y1; y++)
			{
				glVertex2i(int(x + 0.5), y);
				x += m;
			}
		}
	}
	else
	{
		int x = x0;
		int y = 0;
		y = (y0 <= y1) ? y0 : y1;
		int d = fabs((double)(y0 - y1));
		while (d >= 0)
		{
			glVertex2i(x, y);
			y++;
			d--;
		}
	}
}

void lineSegment()
{
	glClear(GL_COLOR_BUFFER_BIT);
	glColor3f(0.0, 0.0, 0.0);
	glBegin(GL_POINTS);
	LineDDA(xs, ys, xe, ye);
	glEnd();
	glFlush();
}

int main(int argc, char *argv[])
{
	glutInit(&argc, argv);
	glutInitDisplayMode(GLUT_RGB | GLUT_SINGLE);
	cout << "输入线段起始和终点坐标（范围为 0 - 500, 0-500，空格隔开四个数）:";
	cin >> xs >> ys >> xe >> ye;
	glutInitWindowPosition(50, 100);
	glutInitWindowSize(500, 500);
	glutCreateWindow("DDA");
	glClearColor(1.0, 1.0, 1.0, 1.0);
	glMatrixMode(GL_PROJECTION);
	gluOrtho2D(0.0, 500, 0.0, 500.0);
	glutDisplayFunc(lineSegment);
	glutMainLoop();

	return 0;
}
