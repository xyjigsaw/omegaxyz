Getting Started with OpenGL

Programming LanguagesC & C++Technology计算机图形学C++计算机图形学openGL
OpenGL (full Open Graphics Library) refers to a professional graphics program interface that defines a cross-programming language and cross-platform programming interface specifications. It is used for three-dimensional images (two-dimensional as well), and is a powerful and easy-to-call underlying graphics library. OpenGL™ is the most widely accepted 2D/3D graphics API in the industry. Since its inception, it has spawned thousands of excellent applications on various computer platforms and devices. OpenGL™ is independent of Windows or other operating systems and is network transparent. In industries including CAD, content creation, energy, entertainment, game development, manufacturing, pharmaceuticals, and virtual reality, OpenGL™ helps programmers develop high-performance, highly impactful, high-visual expressive graphics processing software on hardware devices such as PCs, workstations, and supercomputers.

Environment configuration (taking Windows 10 and Visual Studio 2013 as examples)

First download the GLUT tool (this is the OpenGL tool library) Download address:https://pan.baidu.com/s/1BoMxp-apBknXRP91m8uAvwAfter downloading, unzip it, Place the decompressed glut.dll and glut32.dll in the C:\Windows\system32 folder (32-bit system) or C:\Windows\SysWOW64 (64-bit system). Copy the decompressed glut.h to the "\VC\include\gl" folder under the installation directory (path) of Visual Studio 2013. If there is no gl folder, create a new one. Put the decompressed glut.lib and glut32.lib into the folder where the static function library is located (if it is VS2013, it is the "VC\lib" folder under its installation directory).

The first OpenGL program

C++ code:
#include <GL/glut.h>


// called draw scene
void RenderScene(void)
{
	// clear the window with current clearing color
	glClear(GL_COLOR_BUFFER_BIT);

	// set current drawing color to red
	glColor3f(1.0f, 1.0f, 0.0f);

	// draw rectangle with current color
	glRectf(-0.50f, -0.50f, 0.50f, 0.50f);

	// Flush drawing commands & swap
	glFlush();

}

// Setup the rendering state
void SetupRC(void)
{
	glClearColor(0.0f, 0.0f, 1.0f, 1.0f);
}

int main(int argc, char *argv[])
{
	glutInitDisplayMode(GLUT_SINGLE | GLUT_RGB);
	glutCreateWindow("Simple Opengl");

	glutDisplayFunc(RenderScene);

	SetupRC();
	glutMainLoop();

	return 0;
}
Effect:

Some functions in OpenGL

  • glutInit(&argc, argv) To initialize GLUT, this function must be called once before other GLUT is used.
  • glutInitDisplayMode(GLUT_RGB | GLUT_SINGLE) Set the display mode, where:GLUT_RGB means using RGB colors, and corresponds to GLUT_INDEX (meaning using index colors). GLUT_SINGLE means using single buffering, and corresponds to GLUT_DOUBLE (using double buffering).
  • glutInitWindowPosition(100, 100) Set the position of the window in the laptop screen (100,100).
  • glutInitWindowSize(400, 400) Set the size of the window (400X400).
  • glutCreateWindow("First OpenGL program") Create a window based on the information set previously. The "first OpenGL program" parameter will be used as the title of the window.Note: After the window is created, it does not appear on the screen immediately. GlutMainLoop needs to be called to see the window.
  • glutDisplayFunc(&myDisplay) Set the callback function. When drawing is required, the myDisplay function will be called. (The author’s statement is not accurate enough).
  • glutMainLoop Start program execution and perform message processing loop. (This function can display the window and wait for the window to close before returning)
