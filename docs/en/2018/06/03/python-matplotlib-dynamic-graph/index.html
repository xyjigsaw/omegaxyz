Python uses matplotlib to draw dynamic graphs

Programming LanguagesTechnologyPythonPython
Machine learning requires the use of python to implement corresponding algorithms, so I learned drawing in Matplotlib. Of course, in order to display the effect and convergence speed of each iteration in machine learning, dynamic graphics need to be drawn. Two examples are given below, which can draw dynamic bar charts and dynamic line charts respectively (using two different methods). Note that you need to use the plt.pause(time) function.

Dynamic bar chart

The basic principle is to put the data into an array, then add a number to the array each time, clear the previous image, and redraw the image.Code: 
import matplotlib.pyplot as plt
fig, ax = plt.subplots()
y1 = []
for i in range(50):
    y1.append(i)  # 每迭代一次，将i放入y1中画出来
    ax.cla()   # 清除键
    ax.bar(y1, label='test', height=y1, width=0.3)
    ax.legend()
    plt.pause(0.1)

Dynamic line chart

The basic principle is to use an array of length 2, replace the data each time and append after the original image.Code: 
import numpy as np
import matplotlib.pyplot as plt

plt.axis([0, 100, 0, 1])
plt.ion()

xs = [0, 0]
ys = [1, 1]

for i in range(100):
    y = np.random.random()
    xs[0] = xs[1]
    ys[0] = ys[1]
    xs[1] = i
    ys[1] = y
    plt.plot(xs, ys)
    plt.pause(0.1)
