Address translation in operating system paging virtual memory management

高级语言C&C++技术域操作系统C++操作系统原理
1. Request paging virtual storage management technology stores all information of the job address space on the disk. When a job is selected to run, the first few pages of the job are loaded into the main memory and started to run. For this reason, when creating a page table for a job, it should be stated which pages are already in main memory and which pages are not in main memory. 2. When the job is executed, the logical address in the instruction indicates the page number and page offset in the address of the operand (or instruction) participating in the operation. The hardware address translation mechanism looks up the page table by page number. 3. Design an "address transformation" program to simulate the hardware address change process. When the accessed page is in the main memory, after forming the absolute address, the execution of the instruction is not simulated, but the converted address is output. When the accessed page is not in the main memory, "The page is not in the main memory and a page fault interrupt occurs" is output to indicate that a page fault interrupt has occurred. 4. Perform page fault interrupt processing. After returning from the interrupt, re-execute the instruction. Assume that the length of each block of main memory is 64 bytes, and there is a job with 8 pages. The system allocates 4 main memory blocks (i.e., m=4), and can be divided into up to 4 blocks.

#include<iostream>
#include<deque>
#include<cstdio>
using namespace std;

int blockSize = 64;

struct page{
	int id;
	int flag;
	int memoryBlock;
	int changePosition;
	int outAddress;
};

deque<page> pageList;

void initPageList(int pageSize){
	struct page temp;
	for(int i=0;i<pageSize;i++){
		temp.id=i;
		if(i<4){
			temp.flag=1;
		}else{
			temp.flag=0;
		}
		pageList.push_back(temp);
	}
	pageList[0].memoryBlock=5;
	pageList[1].memoryBlock=8;
	pageList[2].memoryBlock=9;
	pageList[3].memoryBlock=10;
}

void showPage(int id,int bias){
	for(int i=0;i<pageList.size();i++){
		if(pageList[i].id==id){
			cout<<"具体信息：页面号"<<pageList[i].id<<" "<<
			"主存号： "<<pageList[i].memoryBlock<<
			"偏移量： "<<bias<<endl;
			break;
		}
	}
}

void transAddress(int logAddr){
	int Block = logAddr/64;
	int bias = logAddr%64;
	int flag=0;
	for(int i=0;i<pageList.size();i++){
		if(pageList[i].id==Block){
			if(pageList[i].flag==1){
				showPage(pageList[i].id,bias);
				int absAddr = pageList[i].memoryBlock*blockSize+bias;
				cout<<"被转换的物理地址为："<<absAddr<<endl;
				flag = 1;
				break;
			}
		}
	}
	if(flag==0){
		cout<<"该页不在主存，产生缺页中断。"<<endl;
	}
}

int main(){
	int logAddr;
	int pageSize;
	cout<<"输入页表长度：";
	cin>>pageSize;
	initPageList(pageSize);
	while(logAddr!=999){
		cout<<"\n----请输入逻辑地址(999退出)：";
		cin>>logAddr;
		transAddress(logAddr);
	}
	cout<<"ok"<<endl;
}
