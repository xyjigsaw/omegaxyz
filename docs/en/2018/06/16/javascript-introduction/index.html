Getting Started with JavaScript

Programming LanguagesTechnologyWebJavaScript&HtmlWebJavaScript

What is JavaScript

JavaScript, also known as ECMAScript, is a scripting language that is object- and event-driven, relatively safe, and widely used in client-side web development. It is also a scripting language that is widely used in client-side web development.JavaScript is the most popular scripting language in the world, because all web pages you browse on computers, mobile phones (Android, iOS), tablets, and countless mobile apps based on HTML5 have interactive logic driven by JavaScript.Simply put, JavaScript is an interpreted programming language that runs in the browser.

I already know C, C++, Java, and Python, why do I need to learn JavaScript?

Because you have no choice. In the Web world, only JavaScript can drive web pages across platforms and browsers and interact with users. Your browser (Chrome, 360, Firefox, IE, Safari, UC, QQ are all powered by JavaScript) ActionScript behind Flash has been popular for a while, but with the rise of mobile applications, no one uses Flash to develop mobile apps, so it is currently marginalized. On the contrary, as HTML5 becomes more and more popular on PC and mobile, JavaScript becomes more important. Moreover, the emerging Node.js has introduced JavaScript to the server side, and JavaScript has become an all-rounder.

The relationship between JavaScript and Java

Nothing to do with itThe reason why the two names are similar is that while Netscape was developing LiveScript, Sun was also developing the Java language. In order to benefit both parties, the two companies cooperated. Netscape renamed the LiveScript language to JavaScript. This is the origin of JavaScript.

Most web pages are based on HTML language, and JavaScript can be embedded into HTML
