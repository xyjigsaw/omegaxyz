C language determines processor big endian/little endian mode

高级语言C&C++技术域组成原理与体系结构C语言组成原理与体系结构嵌入式
Write a C program to determine the endianness mode of a processor. If the processor is in big-endian mode, output on the console: The processor is in big-endian mode; otherwise, output, the processor is in little-endian mode.

What is big endian/little endian mode

big endian mode

The so-called big-endian mode means that the high byte of data is stored in the low address of the memory, and the low byte of the data is stored in the high address of the memory. This storage mode is somewhat similar to processing data sequentially as a string: the address increases from small to large, and the data is placed from high to low;example:0000430: e684 6c4e 0100 1800 53ef 0100 0100 0000 0000440: b484 6c4e 004e ed00 0000 0000 0100 0000 In big endian mode, the first 16 bits should be read like this: e6 84 6c 4e (assuming int occupies 4 bytes)Memory method:Addresses grow in the opposite order of values.

little endian mode

The so-called little endian mode means that the high byte of data is stored in the high address of the memory, and the low byte of the data is stored in the low address of the memory. This storage mode effectively combines the high and low addresses with the bit weight of the data. The weight of the high address part is high, and the weight of the low address part is low, which is consistent with our logical method.example:0000430: e684 6c4e 0100 1800 53ef 0100 0100 0000 0000440: b484 6c4e 004e ed00 0000 0000 0100 0000 In little endian mode, the first 16 bits should be read like this: 4e 6c 84 e6 (assuming int occupies 4 bytes)Memory method:Addresses grow in the same order as values.Method 1: Use CLanguage cast 
#include <stdio.h>
int main()
{

	int a=0x1234;
	char b=*(char*)&a;
	if(b==0x12)
		printf("处理器采用大端模式\n");
	else
		printf("处理器采用小端模式\n");
	return 0;
}
Method 2: Use Union 
#include <stdio.h>

union U{
    int a;
    char c;
}u;

int main()
{
    u.a=0x1234;
    if(0x34 == u.c)
    {
        printf("处理器采用小端模式\n");
    }
    else if(0x12 == u.c)
    {
        printf("处理器采用大端模式\n");
    }
    return 0;
}
