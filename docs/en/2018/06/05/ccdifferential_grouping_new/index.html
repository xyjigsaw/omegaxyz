Detailed explanation of large-scale optimization algorithm for cooperative co-evolution of differential grouping

TechnologyMachine LearningEvolutionary Algorithms多目标优化算法协同进化
Cooperative coevolution has introduced coevolutionary algorithms with the aim of solving increasingly complex optimization problems through a divide-and-conquer paradigm. Theoretically, the idea of ​​cooperatively changing subcomponents is very suitable for solving large-scale optimization problems. In practice, however, without prior knowledge about the problem, it is unclear how the problem should be decomposed. In this paper, we propose an automatic decomposition strategy, called differential grouping, that can reveal the underlying interaction structure of decision variables and form subcomponents so that interdependencies between them are kept to a minimum. We show mathematically how such a decomposition strategy arises from the definition of partial separability. Empirical studies show that such near-optimal decompositions can significantly improve the quality of solutions to large-scale global optimization problems. Finally, we show how such an automatic decomposition yields a better approximation of the distribution of diverse subcomponents, leading to a more efficient allocation of the computational budget over diverse subcomponents.Index terms: cooperative coevolution, large-scale optimization, problem decomposition, indivisibility, numerical optimizationOverview: https://omegaxyz.com/2017/09/29/ccdifferential_grouping/ MATLAB code: https://omegaxyz.com/2017/10/25/cooperative_coevolution_matlab/

The first decision variable is checked for pairwise interactions with all other decision variables. If the algorithm detects an interaction between the first variable and any other variable, it excludes it from all decision variables. This process is repeated until all variables that interact with the first variable are detected, forming the first subcomponent. If no interaction is detected, then the variable is a separable variable.
