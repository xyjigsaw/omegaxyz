Introduction to TSPLIB data set and MATLAB reading

Programming LanguagesTechnologyMATLABMATLAB
TSPLIB is a question bank that contains TSP and related questions. The files there all have the .tsp suffix. There is a special explanatory paper on the use of these files (https://docs.google.com/file/d/0B4zUGKjaO9uERU1RZDNuRkg3TW8/edit) Data format:NAME: eil51 COMMENT: 51-city problem (Christofides/Eilon) TYPE: TSP DIMENSION: 51 EDGE_WEIGHT_TYPE : EUC_2D NODE_COORD_SECTION 1 37 52 2 49 49 3 52 64 4 20 26 5 40 30 … 50 56 37 51 30 40 EOF   NAME is the name of the file. COMMENT is an additional explanation for this question. TYPE describes the type of problem, because TSPLIB also contains some other types of problems, but here we only focus on the TSP type. DIMENSION describes the number of cities. EDGE_WEIGHT_TYPE describes the type of cost between two cities. Here is the 2D Euclidean distance we are most familiar with. NODE_COORD_SECTION describes the 2D Euclidean coordinates of each city. Each line is in the order of city number, X coordinate, and Y coordinate. However, it should be noted that EDGE_WEIGHT_TYPE is not only EUC_2D, but there are 13 types. Various types have corresponding distance calculation methods, such as Manhattan distance, geographical distance, etc. I will not list them one by one here, as they are described in detail in the paper. Here I will only mention the most common type EXPLICIT. This type is quite different from the others. The distance between cities is given explicitly and does not need to be calculated.MATLAB reading code: 
function [n_citys,city_position] = Read(filename)
fid = fopen(filename,'rt');
location=[];
A = [1 2];
tline = fgetl(fid);
while ischar(tline)
    if(strcmp(tline,'NODE_COORD_SECTION'))
        while ~isempty(A)
            A=fscanf(fid,'%f',[3,1]);
            if isempty(A)
                break;
            end
            location=[location;A(2:3)'];
        end
    end
    tline = fgetl(fid); 
    if strcmp(tline,'EOF')
        break;
    end
end
[m,n]=size(location);
n_citys = m;
city_position=location;
fclose(fid);
end
