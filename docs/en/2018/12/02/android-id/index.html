Android uses string to dynamically obtain resource ID

Programming LanguagesTechnologyAPP开发JavaJavaappAndroid
Each file in the android file has an ID, as shown in the figure below. The 0x7f060000 on the left is the file ID:If we want to obtain the ID of this file in code, we should use an efficient reflection mechanism. We can create a new Java class code as follows:
public class ResourceMan {
    public static int getResId(String variableName, Class<?> c) {
        try {
            Field idField = c.getDeclaredField(variableName);
            return idField.getInt(idField);
        } catch (Exception e) {
            e.printStackTrace();
            return -1;
        }
    }
}
Or add a method to the current class:
public static int getResId(String variableName, Class<?> c) {
        try {
            Field idField = c.getDeclaredField(variableName);
            return idField.getInt(idField);
        } catch (Exception e) {
            e.printStackTrace();
            return -1;
        }
 }
Calling method: where icon is the file name and no suffix is required.
int id = ResourceMan.getResId("icon",R.drawable.class);
//或者int id = getResId("icon",R.drawable.class);
