Detailed explanation of feature selection Filter method

TechnologyMachine Learning特征选择
Filter (filtering method) measures the classification ability of features by analyzing the internal characteristics of feature subsets, regardless of which classifier is used later. This type of method usually requires a scoring function and a threshold discriminant method to evaluate feature correlation to select the feature subset with the highest score. Through literature research, according to the different ways of selecting feature subsets, it can be further divided into two categories: feature ranking (Feature Ranking) and feature space search (Space Search).The main idea of ​​the method based on feature ranking is:1) Use the Scoring Function to score each feature and arrange all features in descending order of score; 2) Perform significance test (such as p-value, etc.) on each feature score; 3) Select the top ranked features with significant statistical significance through preset thresholds; 4) Verify the selected optimal feature subset, usually using ROC (Receiver Operating Characteristic) curve, classification accuracy, group correlation coefficient, stability, etc.The core of the feature-based ranking method is the scoring function, the following figure lists three types of popular scoring functions that appear in feature selection: based on the difference between the distributions of metric sample groups, based on information theory, and based on correlation criteria.
The feature space search method mainly uses an optimization strategy to select the feature subset that contains the most information and achieves the minimum redundancy from the entire feature set. In specific fields, there are some studies on the accurate discovery of disease-causing genes, such as Correlation-based feature selection (CFS), Maximum Relevance Minimum Redundancy (MRMR), etc. The figure below shows several mainstream methods based on the feature space search method.
Methods based on feature ranking are mostly univariate methods. They consider the influence of a single feature at a time and select the feature most relevant to the class label. This method has good computational complexity for high-dimensional small samples. However, in some application fields such as genetic microarray data, because the relationship between features is ignored, analysis is directly applied. The class accuracy is relatively average; and based on feature space search, which is a multi-variable method, this type of algorithm not only needs to consider the correlation between feature subsets and class labels, but also the correlation between feature subsets. Generally, the classification accuracy is good, but the computational complexity of finding the optimal subset under high-dimensional conditions is high.
