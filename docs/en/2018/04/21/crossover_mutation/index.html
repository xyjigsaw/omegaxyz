Detailed explanation of crossover mutation of genetic algorithm

TechnologyMachine LearningEvolutionary AlgorithmsNSGA-II

cross

Binary coded crossover

single point crossover

Single-point crossover is also called simple crossover, which means that only one crossover point is randomly set in the individual coding string, and then part of the chromosomes of the two ligand individuals are exchanged at this point. Figure 1 is a schematic diagram of a single-point crossover operation.

Two points cross

Two-point crossover refers to randomly setting two crossover points in individual coding strings, and then performing partial gene exchange. The specific operation process of two-point crossover is: ① Randomly set two crossover points in the coding strings of two paired individuals; ② Exchange part of the chromosomes of the two individuals between the two set crossover points. Figure 2 is a schematic diagram of the two-point intersection operation.

multi-point crossover

Or generalized crossover, which refers to randomly setting multiple crossover points in individual coding strings and then performing gene exchange. The operation process is similar to single-point crossover and two-point crossover.

evenly cross

Also called consistent crossover, it means that the genes at each locus of two paired individuals are exchanged with the same crossover probability, thereby forming two new individuals. The specific operation is to determine how each gene of the new individual is provided by which parent individual by setting a mask word. The main operation process is as follows: ① Randomly generate a masked word W=w1w2Lw1Lw1 with the same length as the individual coding string, where L is the length of the individual coding string; ② According to the above rules, two new offspring individuals A′ and B′ are generated from the two parent individuals A and B. If ωi=0, then the gene value of A′ at the i-th locus inherits the corresponding gene value of A, and the gene value of B′ at the i-th locus inherits the corresponding gene value of B; if ωi=1, then the gene value of A′ at the i-th locus inherits the corresponding gene value of B, and the gene value of B′ at the i-th locus inherits the corresponding gene value of A.

Intersection of two points uniformly

It refers to the random generation of two intersection points between two ligands A and B, and then gene exchange based on three randomly generated integers of 0, 1, and 2, thereby forming two new individuals [4]. When the random number is 0, the front part of the ligand crosses; when the random number is 1, the middle part of the ligand crosses; when the random number is 2, the back part of the ligand crosses.There are other crossover operators, such as shrinking agent crossover, shuffling crossover, etc.

Crossover operator suitable for floating point encoding

The floating-point encoding method means that each gene value of an individual is represented by a floating-point number within a certain range, and the encoding length of the individual is equal to the number of its decision variables. In addition to the above-mentioned crossover operators suitable for binary encoding methods that can be used in the crossover operation of floating-point encoding methods, the following main crossover operators are also used.

discrete crossover

It refers to exchanging the values ​​of variables between individuals. Each variable of the child individual can randomly select the parent individual with equal probability.

arithmetic crossover

It refers to the linear combination of two individuals to produce two new individuals. The operation object of arithmetic crossover is generally an individual represented by a floating-point number encoding. It is defined as a combination of two vectors (chromosomes): x1=λ1x1+λ2x2; x2=λ1x2+λ2x1, where λ1 and λ2 are called multipliers. Special circumstances include: ①When λ1=λ2=0.5, Davis calls it average crossover and Schwefel calls it intermediate crossover; ② When the multiplier is used as a random number on the interval [-d, 1+d], Muh-lenbein and Schlierkamp-Voosen call it an extended intermediate crossover;

Heuristic crossover

If parent individual 1 and parent individual 2, and parent individual 1 has better fitness, the following function generates child individuals: child individual = parent individual 2 + Radio3 (parent individual 1 - parent individual 2). Among them, Radio specifies how far the offspring is from the parent with better fitness, and its default value is 1.2.There are also crossover operators suitable for serial number encoding, such as partial matching crossover, sequential crossover, cyclic crossover, position-based exchange, etc.

Mutations

The basic content of the mutation operator is to change the gene values ​​at certain loci of individual strings in the population. Depending on the individual coding representation methods, the following algorithms can be used: a) Real-valued variation b) Binary mutation. Generally speaking, the basic steps of mutation operator operation are as follows: a) Determine whether to mutate based on the preset mutation probability for all individuals in the group b) Randomly select mutation bits for the mutated individuals to mutate. There are two purposes for introducing mutations in genetic algorithms: first, to enable the genetic algorithm to have local random search capabilities. When the genetic algorithm is close to the optimal solution neighborhood through the crossover operator, the local random search capability of the mutation operator can be used to accelerate convergence to the optimal solution. Obviously, the mutation probability in this case should be smaller, otherwise the building blocks close to the optimal solution will be destroyed due to mutation. The second is to enable the genetic algorithm to maintain population diversity to prevent premature convergence. At this time, the convergence probability should take a larger value. In the genetic algorithm, the crossover operator is used as the main operator because of its global search ability, and the mutation operator is used as the auxiliary operator because of its local search ability. Genetic algorithms use crossover and mutation, a pair of cooperative and competing operations, to achieve balanced search capabilities that take both global and local considerations into consideration. The so-called mutual cooperation means that when the group is stuck in a certain hyperplane in the search space during evolution and cannot get rid of it by crossover alone, the mutation operation can help to get rid of it. The so-called competition means that when the desired building blocks have been formed through crossover, the mutation operation may destroy these building blocks. How to effectively use crossover and mutation operations together is an important research content of current genetic algorithms. The basic mutation operator refers to randomly selecting one or more loci for individual code strings in the population and changing the gene values of these loci (with mutation probability P.). The basic mutation operation in (0, 1) binary code strings is as follows: Genes marked with * below the gene position are mutated. The selection of mutation rate is generally affected by factors such as population size and chromosome length, and is usually very small. The value of the genetic algorithm is generally 0.001-0.1.
