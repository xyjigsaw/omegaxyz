TensorBoard displays TensorFlow flowchart

TechnologyMachine LearningDeep LearningPythonTensorFlow-Pytorch神经网络
The operations involved in TensorFlow are usually complex and difficult-to-understand operations that occur in training huge deep neural networks. In order to make it easier to understand, debug and optimize TensorFlow programs, Google has released a set of visualization tools called TensorBoard. You can use TensorBoard to display your TensorFlow images, plot quantitative metrics generated by the images, and append data.How to display?①Add this line of code after the entire code:
writer = tf.summary.FileWriter("test", sess.graph)
Note that "test" is the file name and location. The name is used as the location here, which is the root directory of the code. ②Open the root directory folder of the code (that is, where the code is saved) and open cmd (or powershell) ③Enter a sentence like this:tensorboard --logdir=testNote that test is the location defined above.A local URL will be displayed after a while, just copy it into the browser and enter it.
