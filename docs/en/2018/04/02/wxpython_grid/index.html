wxpython control adaptive window size

高级语言技术域PythonPythonwxpythonGUI
Question: If you click on the window operated by wxpython to maximize the window, problems will occur. 1. The window controls are not maximized at the same time. 2. Some controls cannot be found after maximizing. If the window is restored, the disappeared controls cannot be found. The most complex sizer in wxPython. It brings explicit positioning of widgets. Widget items can also span rows or columns. Special Tutorial:wxpython Chinese tutorialThe constructor of wx.GridBagSizer is very simple. wx.GridBagSizer (integer vgap, integer hgap) The vertical and horizontal gaps (gaps) define the gap distance between sub-widgets. Add widget items to the grid using the Add() method. Add(self, item, tuple pos, tuple span=wx.DefaultSpan, integer flag=0, integer border=0, userData=None) "item" refers to the widget you want to insert into the grid. "pos" specifies the position of the virtual grid. The "pos" value of the upper left cell is (0, 0). "span" is a span value of the corresponding widget. For example, (3, 2) means that a widget should span 3 rows and 2 columns. The two parameters "flag" and "border" have been discussed earlier in wx.BoxSizer. These widget items in the grid can change their size as the window is scaled, or they can remain the same. If you need the part to be larger or smaller, you can use these two methods. AddGrowableRow(integer row) AddGrowableCol(integer col) Code:
import wx


class MyFrame(wx.Frame):
    def __init__(self,parent, id, title):
        wx.Frame.__init__(self, parent, id, title, wx.DefaultPosition)
        sizer = wx.GridBagSizer(9, 9)
        sizer.Add(wx.Button(self, -1, "按钮1"), (0, 0), wx.DefaultSpan,wx.ALL, 5)
        sizer.Add(wx.Button(self, -1, "按钮2随着主窗口布局变动"), (1, 1), (1, 7), wx.EXPAND)
        sizer.Add(wx.Button(self, -1, "按钮3随着主窗口布局变动"), (6, 6), (3, 3), wx.EXPAND)
        sizer.Add(wx.Button(self, -1, "按钮4"), (3, 0), (1, 1), wx.ALIGN_CENTER)
        sizer.Add(wx.Button(self, -1, "按钮5"), (4, 0), (1, 1), wx.ALIGN_LEFT)
        sizer.Add(wx.Button(self, -1, "按钮6"), (5, 0), (1, 1), wx.ALIGN_RIGHT)
        sizer.AddGrowableRow(6)
        sizer.AddGrowableCol(6)
        self.SetSizerAndFit(sizer)
        self.Centre()


class MyApp(wx.App):
    def OnInit(self):
        frame = MyFrame(None, -1, 'wxgridbagsizer.py')
        frame.Show(True)
        return True
app = MyApp(0)
app.MainLoop()
