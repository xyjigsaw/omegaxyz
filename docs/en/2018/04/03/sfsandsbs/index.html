SFS and SBS feature selection algorithms

Programming LanguagesTechnologyMATLABMachine LearningMATLAB特征选择

(1) Sequential Forward Selection (SFS, Sequential Forward Selection)

Algorithm description: The feature subset X starts from the empty set, and each time one feature x is selected to be added to the feature subset X to make the feature function J( To put it simply, each time a feature is selected to add the optimal value of the evaluation function, it is actually a simple greedy algorithm.

Algorithm evaluation: The disadvantage is that features can only be added but not removed. For example: Feature A completely depends on features B and C. It can be considered that if features B and C are added, A will be redundant. Assuming that the sequence forward selection algorithm first adds A to the feature set, and then adds B and C, then the feature subset contains redundant feature A.

Code:

(2) Sequential Backward Selection (SBS, Sequential Backward Selection)

Algorithm description: Starting from the complete set of features O, each time one feature x is eliminated from the feature set O, so that the evaluation function value reaches the optimal value after eliminating the feature x.

Algorithm evaluation: Sequential backward selection is exactly the opposite of sequence forward selection. Its disadvantage is that features can only be removed but not added.

Code:

clear;

%--------特征导入  请自行修改

    mii=ones([m n])*mi;

%%---------------SBS

    chosen=[chosen i];

for j=1:24   %%删10个,留20个
    J=zeros([1 30]);ii=0;  %J(1)是删1的结果,J(2)是删除2 的结果......
    for i=1:30  %???dele 是必要的么???????????????????????%一共30组特征 这里 请自行修改
       
    [mm nn]=size(chosen);

      end

In addition, both SFS and SBS are greedy algorithms and can easily fall into local optimal values.
