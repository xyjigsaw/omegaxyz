Operating system thread (process) synchronization Java instance

Programming LanguagesTechnologyJava操作系统Java多线程操作系统原理
To read this article, please first look at the Java thread class:https://omegaxyz.com/2018/04/09/java_threading/Java provides the Executor interface to execute tasks in the thread pool, and the ExecutorService interface to manage and control tasks.ExecutorService executor = Executors.newCachedThreadPool();The statement will create a new thread for each waiting task, and all tasks can be executed concurrently. Note that you need to use shutdown() to shut down the executor. Suppose 100 threads are created and started, each thread adds a penny to an account, and a main class that creates and starts the threads. Data corruption occurs when all threads access the same data source. Therefore, the keyword synchronized must be used to protect the deposit() thread (locking, mutual exclusion semaphore)Example code: 
import java.util.concurrent.*;

public class AccountWithoutSync {
  private static Account account = new Account();

  public static void main(String[] args) {
    ExecutorService executor = Executors.newCachedThreadPool();

    // Create and launch 100 threads
    for (int i = 0; i < 100; i++) {
      executor.execute(new AddAPennyTask());
    }

    executor.shutdown();

    // Wait until all tasks are finished
    while (!executor.isTerminated()) {
    }

    System.out.println("What is balance? " + account.getBalance());
  }

  // A thread for adding a penny to the account
  private static class AddAPennyTask implements Runnable {
    public void run() {
      account.deposit(1);
    }
  }

  // An inner class for account
  private static class Account {
    private int balance = 0;

    public int getBalance() {
      return balance;
    }

    public synchronized void deposit(int amount) {
      balance += amount;

      // This delay is deliberately added to magnify the
      // data-corruption problem and make it easy to see.
      try {
        Thread.sleep(5);
      }
      catch (InterruptedException ex) {
      }

      balance = newBalance;
    }
  }
}
Or modify run() like this:
public void run() {
    Synchronized (account){
          Account.deposit(1);
    }
}
