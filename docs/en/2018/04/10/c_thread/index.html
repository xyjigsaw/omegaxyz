Detailed explanation of multi-threaded operation in C language

Programming LanguagesC & C++Technology操作系统C++C多线程操作系统原理
pthread_t: used to define a thread type variableUsage pthread_t x1;
pthread_create: Create a thread, it has 4 parameterspthread_create(&temp, NULL, print_b, NULL); The first parameter is a pointer to the thread identifier, the second parameter is used to set the thread attributes, the third parameter is the starting address of the thread running function, and the last parameter is the parameter of the running function. Here, our function thread does not require parameters, so the last parameter is set to a null pointer. We also set the second parameter to a null pointer, which will generate a thread with default attributes.
pthread_join: used to wait for the end of a thread, which can also be understood as the start of a thread.The first parameter is the identifier of the thread being waited on, and the second parameter is a user-defined pointer, which can be used to store the return value of the thread being waited for. pthread_join(threadPool[i], &result);
pthread_exit: There are two ways to end a thread. When a function ends, the thread that called it also ends; the other way is through the function pthread_exit.The following code is an implementation of multiple (3 or more) threads (creating a thread pool) in C language:Just like the producer and consumer problem, multiple producers and consumers are required. 
#include <stdio.h>
#include <pthread.h>
#include <windows.h>


void * print_a(void *a){
    int i;
    for(i = 0;i < 10; i++){
        Sleep(1000);
        printf("aa\n");
    }
    return NULL;

}//1号进程

void * print_b(void *b){
    int i;
    for(i=0;i<20;i++){
        Sleep(1000);
        printf("bb\n");
    }
    return NULL;
}//2号进程


int main()
{
    int aNum=5;
    int bNum=3;
    pthread_t threadPool[aNum+bNum];//创建一个线程池，大小为aNum+bNum

    int i;
    for(i = 0; i < aNum; i++){
        pthread_t temp;
        if(pthread_create(&temp, NULL, print_a, NULL) == -1){
            printf("ERROR");
            exit(1);
        }
        threadPool[i] = temp;
    }//创建a进程放入线程池


    for(i = 0; i < bNum; i++){
        pthread_t temp;
        if(pthread_create(&temp, NULL, print_b, NULL) == -1){
            printf("ERROR");
            exit(1);
        }
        threadPool[i+aNum] = temp;
    }//创建b进程放入线程池


    void * result;
    for(i = 0; i < aNum+bNum; i++){
        if(pthread_join(threadPool[i], &result) == -1){
            printf("fail to recollect\n");
            exit(1);
        }
    }//运行线程池
    return 0;
}
