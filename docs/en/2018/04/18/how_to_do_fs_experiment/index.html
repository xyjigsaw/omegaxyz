How to do feature selection experiments in machine learning

TechnologyMachine Learning特征选择
Feature selection is a data preprocessing process. Excellent features can improve the performance of the classifier. In the real world, there are many categories that contain many features, such as biological DNA. After using different algorithms to select a smaller subset of features, how to evaluate the selected features is crucial.Take the following points from most feature selection papers.

①Error rates of different feature selection algorithms on different data sets

②The number of solutions selected by different feature selection algorithms on different data sets

③The number of new solutions that appear under different iteration times

illustrate:The fewer new solutions that appear, it means that the algorithm is gradually converging. Displaying the number of new solutions that appear in multiple algorithms on the same coordinate axis can effectively compare the convergence speed of the algorithms.
