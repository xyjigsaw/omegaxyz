Detailed explanation of C++ static members

Programming LanguagesC & C++TechnologyC++
Static members solve the problem of data and function sharing between different objects of the same class.
  • Instance properties: All objects of a class have the same attributes, and the attribute values РђІРђІcan be different.
  • class attributes: Describes the common characteristics of all objects of a class, and its attribute values РђІРђІare the same for any object.
static data members
  • Declared with keyword static
  • All objects of this class maintain the same copy of this member
  • It must be defined and initialized outside the class, and use (::) to indicate the class it belongs to.
classname::identifierstatic member function
  • Code outside the class can call static member functions using the class name and the scope operator.
  • You can also call static member functions directly using the object name.
Static member functions can directly access the static data and function members of the class. Access to non-static members must be through the object name. Example: Point class with static member functions and static data members
