MATLAB parallel operation program

高级语言技术域MatlabMatlab多线程
When matlab calculates big data memory and large matrix operations, single-core operations obviously cannot meet the high-speed operation requirements. In fact, matlab provides solutions for multi-core computing. Here we first introduce the two simplest ones.①parfor ②distributed

parfor

The for loop is generally the most time-consuming. The solution is to use parfor instead of for. parfor can automatically allocate loop tasks to multiple cores. Now, run parpool(4) in the command line window of matlab to check whether parallel computing can be performed. If no error is reported, parallelism is turned on. When parfor is running, there will be multiple matlab processes calculating. Computational efficiency is significantly improved.

Code without using parfor:

answer = 0;
for i=1:1000000
    answer = answer + i;
end
time:

Code using parfor:

answer = 0;
parfor i=1:1000000
    answer = answer + i;
end
time: Obviously adding the parfor statement is faster! Notice:① Iteration or associative assignment statements cannot be used in the parfor loop, because data cannot be exchanged when multiple cores calculate. ② parfor only supports one level of looping, so the following writing method is non-parallel and may cause errors. ③ Before running the program, MATLAB will prompt you where parfor cannot be used (displayed with red wavy lines)

distributed

For the storage and calculation problems of large matrices, in addition to dividing the blocks yourself, you can use matlab's own memory allocation solution, but you need to ensure that your computer has enough memory, and the calculation efficiency will be greatly improved. Otherwise there will be memory overflow problems. The specific method is
tic;
X = randn(10000)；
x_dis = distributed(X);
clear(X);
whos
a = svd(x_dis);
a_out = gather(a);%distrituted数据类型变为double
toc;
You can then use x_dis instead of X for calculation. The function of the gather function is to change the distributed data type into double. In addition, most of matlab's toolboxes have begun to support parallel computing, so if the problem can be solved with the built-in toolbox, you can use the parallel computing method that comes with the toolbox. Matlab's Parallel Computing Toolbox toolbox supports parallelism.
