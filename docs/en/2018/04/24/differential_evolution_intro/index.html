Introduction to DE algorithm

Differential Evolution (DE) was proposed by Storn et al. in 1995. Like other evolutionary algorithms, DE is a stochastic model that simulates biological evolution. Through repeated iterations, individuals that adapt to the environment are preserved. However, compared with evolutionary algorithms, DE retains the global search strategy based on the population, and uses real number encoding, simple mutation operations based on differences, and one-to-one competition survival strategies to reduce the complexity of genetic operations. At the same time, DE's unique memory ability allows it to dynamically track the current search situation to adjust its search strategy. It has strong global convergence capabilities and robustness, and does not require the use of characteristic information of the problem. It is suitable for solving some optimization problems in complex environments that cannot be solved using conventional mathematical programming methods. • DE Algorithm-Author’s website:http://www1.icsi.berkeley.edu/~storn/code.html• Wikipedia database :https://en.wikipedia.org/wiki/Differential_evolutionThe DE algorithm is mainly used to solve global optimization problems of continuous variables. Its main working steps are basically the same as other evolutionary algorithms, including three operations: mutation, crossover, and selection. The basic idea of ​​the algorithm is to start from a randomly generated initial group.The difference vector of two individuals randomly selected from the population is used as the source of random variation for the third individual,The difference vector is weighted and then summed with the third individual according to certain rules to generate a mutated individual. This operation is called mutation. Then, the mutant individual is parameter-mixed with a predetermined target individual to generate a test individual. This process is called crossover. If the fitness value of the test individual is better than the fitness value of the target individual, the test individual will replace the target individual in the next generation, otherwise the target individual will still be preserved. This operation is called selection. In the evolution process of each generation, each individual vector is used as a target individual once. Through continuous iterative calculations, the algorithm retains excellent individuals, eliminates inferior individuals, and guides the search process to approach the global optimal solution.

The relationship between DE and GA

Compared with the genetic algorithm, the difference between the differential evolution algorithm and the genetic algorithm is that the initial population is randomly generated, and the fitness value of each individual in the population is used as the selection criterion. The main process also includes three steps: mutation, crossover and selection. The difference is that the genetic algorithm controls the hybridization of the parent generation based on the fitness value. The offspring generated after mutation have a probability value of being selected. In the maximization problem, individuals with a large fitness value have a correspondingly greater probability of being selected. The mutation vector of the differential evolution algorithm is generated by the parent difference vector, and crosses with the parent individual vector to generate a new individual vector, which is directly selected with its parent individual. Obviously, the differential evolution algorithm has a more significant approximation effect than the genetic algorithm.
