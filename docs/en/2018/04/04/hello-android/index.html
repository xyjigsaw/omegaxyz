Hello, Android

TechnologyAPP开发appAndroid
Android Studio is an Android integrated development tool based on IntelliJ IDEA. Similar to Eclipse ADT, Android Studio provides integrated Android development tools for development and debugging. Based on IDEA, Android Studio provides: ①Gradle-based build support ②Android-specific reconstruction and quick repair ③Prompt tools to capture issues such as performance, availability, version compatibility, etc. ④Supports ProGuard and application signature ⑤Template-based wizard to generate commonly used Android application designs and components ⑥Powerful layout editor that allows you to drag and drop UI controls and preview the effects How to generate a Hello, world program using Android? ①Click New Project, enter the application name and company domain name and click next② Select SDK. Here I choose API24 from 2017, Android7.0 mobile phone and tablet mode (although Oreo is out)③Select Empty Activity④Enter the activity name HelloAndriodActivity⑤Code automatically generatedFinally, click to start the virtual machine and you will see the following pictureCopyright of the image is reproduced.
