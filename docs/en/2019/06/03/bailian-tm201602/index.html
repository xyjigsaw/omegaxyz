A:rock paper scissors

Global question number 4973, number of submissions: 222, number of attempts: 100, number of passes: 95
Total time limit:
1000ms
Memory limit:
65536kB
describe
Rock, paper, scissors is a common guessing game. Rock beats scissors, scissors beats paper, and paper beats rock. If two people throw the same punch, there will be no winner. One day, Little A and Little B were playing rock, paper, scissors. It is known that their punches follow a cyclical pattern, such as: "Rock-Paper-Rock-Scissors-Rock-Paper-Rock-Scissors...", which is a cycle of "Rock-Paper-Rock-Scissors". Excuse me, after Little A and Little B competed for N rounds, who won more rounds?
enter
The input contains three lines. The first line contains three integers: N, NA, NB, which respectively represent the cycle length of Little A's punch and the cycle length of Little B's punch after N rounds. 0 < N,NA,NB < 100. The second line contains NA integers, indicating the rules of Little A's punches. The third line contains NB integers, indicating the rules of Little B's punches. Among them, 0 means "rock", 2 means "scissors", and 5 means "paper". Separate two adjacent integers with a single space.
output
Output a line. If Little A wins more rounds, output A; if Little B wins more rounds, output B; if the two are tied, output draw.
Sample input
10 3 4
0 2 5
0 5 0 2
Sample output
A
hint
For test data, the guessing process is: A: 0 2 5 0 2 5 0 2 5 0 B：0 5 0 2 0 5 0 2 0 5 A won 4 rounds, B won 2 rounds, and both sides tied for 4 rounds, so A won more rounds.
C++ code
#include<iostream>
#include<vector>
#include<cstdio>
using namespace std;

int judge(int x, int y){
    if(x == y)return 0;
    if((x == 0 && y == 2) || (x == 2 && y == 5)|| (x == 5 && y == 0))return 1;
    if((x == 2 && y == 0) || (x == 5 && y == 2)|| (x == 0 && y == 5))return -1;
    return -1;
}

int main(){
    int n;
    cin>>n;
    int a, b;
    cin>>a>>b;
    int awin = 0, bwin = 0;
    vector<int> apool(a);
    vector<int> bpool(b);
    for(int i = 0; i < a; i++)cin>>apool[i];
    for(int i = 0; i < b; i++)cin>>bpool[i];
    for(int i = 0; i < n; i++){
        if(judge(apool[i % a], bpool[i % b]) == 1)awin++;
        else if(judge(apool[i % a], bpool[i % b]) == -1)bwin++;
    }
    if(awin == bwin)printf("draw\n");
    else{
        printf("%s\n", awin > bwin ? "A" : "B");
    }
    return 0;
}

B: String equality

Global question number 1745, number of submissions 246, number of attempts 97, number of passes 90
Total time limit:
1000ms
Memory limit:
65536kB
describe
Determine whether two strings consisting of uppercase and lowercase letters and spaces are equal after ignoring case and ignoring spaces.
enter
Two lines, each containing a string.
output
If the two strings are equal, output YES, otherwise output NO.
Sample input
a A bb BB ccc CCC
Aa BBbb CCCccc
Sample output
YES
C++ implementation
#include<iostream>
#include<vector>
using namespace std;

int main(){
    string a, tempa = "";
    string b, tempb = "";
    getline(cin, a);
    getline(cin, b);
    for(int i = 0; i < a.length(); i++){
        if(a[i] == ' ')continue;
        else tempa += toupper(a[i]);
    }
    for(int i = 0; i < b.length(); i++){
        if(b[i] == ' ')continue;
        else tempb += toupper(b[i]);
    }
    printf("%s\n", tempa == tempb ? "YES" : "NO");
    return 0;
}

C: Image rotation

Total time limit:
1000ms
Memory limit:
65536kB
describe
Input a black and white image with n rows and m columns, rotate it 90 degrees clockwise and output it.
enter
The first line contains two integers n and m, indicating the number of rows and columns of pixels in the image. 1 <= n <= 100, 1 <= m <= 100. The next n lines, each line contains m integers, represent the grayscale of each pixel in the image. Two adjacent integers are separated by a single space, and each element is between 0 and 255.
output
M lines, each line containing n integers, are images rotated 90 degrees clockwise. Separate two adjacent integers with a single space.
Sample input
3 3
1 2 3
4 5 6
7 8 9
Sample output
7 4 1
8 5 2
9 6 3
C++ implementation
#include<iostream>
#include<vector>
using namespace std;

int a[101][101];
int b[101][101];

int main(){
    int m, n;
    cin>>m>>n;
    for(int i = 0; i < m; i++){
        for(int j = 0; j < n; j++){
            cin>>a[i][j];
            b[j][n - i - 1] = a[i][j];
        }
    }
    for(int i = 0; i < n; i++){
        for(int j = 0; j < m; j++){
            printf("%d%s", b[i][j], j == m - 1 ? "\n" : " ");
        }
    }
    return 0;
}

D:Chessboard problem

Global question number 323, number of submissions 131, number of attempts 52, number of passes 26
Total time limit:
1000ms
Memory limit:
65536kB
describe
Placing chess pieces on a chessboard of a given shape (the shape may be irregular) makes no difference between the chess pieces. It is required that any two chess pieces cannot be placed in the same row or column on the chessboard. Please program to solve all feasible placement plans C for k chess pieces on a chessboard with a given shape and size.
enter
The input contains multiple sets of test data. The first line of each set of data is two positive integers, n k, separated by a space, indicating that the chessboard will be described in an n*n matrix and the number of chess pieces placed. n <= 8 , k <= n When -1 -1 indicates the end of input. The subsequent n lines describe the shape of the chessboard: each line has n characters, where # represents the chessboard area and . represents the blank area (the data is guaranteed not to have redundant blank rows or columns).
output
For each set of data, a line of output is given, outputting the number of arranged options C (data guarantee C<2^31).
Sample input
2 1
#.
.#
4 4
...#
..#.
.#..
#...
-1 -1
Sample output
2
1
C++ implementation
#include<iostream>
#include<algorithm>
#include<cstdio>
#include<cstring>
using namespace std;

bool visit[10]; //用来标记每一列是否有棋子
char matrix[10][10];

int n, k, ans;

void DFS(int x, int cnum){
    if(cnum >= k){ //如果棋子已经用完，记录方案数加1
        ans++;
        return;
    }
    for(int i = x; i < n; i++){ //从x行开始进行，因此下面只需要对列进行递归
        for(int j = 0; j < n; j++){
            if(!visit[j] && matrix[i][j] == '#'){ //如果这一列没有被访问，且在棋盘上
                visit[j] = true;
                DFS(i + 1, cnum + 1); //搜索下一列
                visit[j] = false;
            }
        }
    }
}

int main(){
    while(cin>>n>>k){
        if(n == -1 && k == -1)break;
        ans = 0;
        fill(visit, visit + n, false);
        fill(matrix[0], matrix[0] + n * n, 0);
        for(int i = 0; i < n; i++){
            scanf("%s", matrix[i]);
        }
        DFS(0, 0);
        cout<<ans<<endl;
    }
}
 

G:Tangled in Cables

Global question number 1077 Number of submissions 126 Number of attempts 56 Number of passes 41
Total time limit:
1000ms
Memory limit:
65536kB
describe
You are the owner of SmallCableCo and have purchased the franchise rights for a small town. Unfortunately, you lack enough funds to start your business properly and are relying on parts you have found in an old warehouse you bought. Among your finds is a single spool of cable and a lot of connectors. You want to figure out whether you have enough cable to connect every house in town. You have a map of town with the distances for all the paths you may use to run your cable between the houses. You want to calculate the shortest length of cable you must have to connect all of the houses together.
enter
Only one town will be given in an input.
  • The first line gives the length of cable on the spool as a real number.
  • The second line contains the number of houses, N
  • The next N lines give the name of each house's owner. Each name consists of up to 20 characters {a–z,A–Z,0–9} and contains no whitespace or punctuation.
  • Next line: M, number of paths between houses
  • next M lines in the form
< house name A > < house name B > < distance > Where the two house names match two different names in the list above and the distance is a positive real number. There will not be two paths between the same pair of houses.
output
The output will consist of a single line. If there is not enough cable to connect all of the houses in the town, output Not enough cable If there is enough cable, then output Need < X > miles of cable Print X to the nearest tenth of a mile (0.1).
Sample input
100.0
4
Jones
Smiths
Howards
Wangs
5
Jones Smiths 2.0
Jones Howards 4.2
Jones Wangs 6.7
Howards Wangs 4.0
Smiths Wangs 10.0
Sample output
Need 10.2 miles of cable
C++ implementation
#include<iostream>
#include<vector>
#include<unordered_map>
#include<algorithm>
#include<cstdio>
using namespace std;

double cableLen = 0;
int n, m;
int father[501];

struct node{
    string src;
    string dst;
    double val;
};

int findFather(int x){
    while(x != father[x])x = father[x];
    return x;
}

bool unionSet(int x, int y){
    int fx = findFather(x);
    int fy = findFather(y);
    if(fx != fy){
        father[fy] = fx;
        return true;
    }
    return false;
}

bool cmp(node a, node b){return a.val < b.val;}

int main(){
    double anslen = 0;
    int k = 0, i = 0;
    scanf("%lf%d", &cableLen, &n);
    unordered_map<string, int> name;
    for(int i = 0; i < n; i++){
        string str;
        cin>>str;
        father[i] = i;
        name[str] = i;
    }
    scanf("%d", &m);
    vector<node> pool(m);
    for(int i = 0; i < m; i++){
        cin>>pool[i].src>>pool[i].dst>>pool[i].val;
    }
    sort(pool.begin(), pool.end(), cmp);
    while(k < n - 1){
        if(unionSet(name[pool[i].src], name[pool[i].dst])){
            anslen += pool[i].val;
            k++;
        }
        i++;
    }
    if(anslen <= cableLen)
        printf("Need %.1f miles of cable\n", anslen);
    else printf("%s\n", "Not enough cable");
    return 0;
}
