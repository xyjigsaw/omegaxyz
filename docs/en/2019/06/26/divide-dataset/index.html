.mat data set production

If the entire data set is a .mat file and the last column of data is a label, separate the decision variables and labels separately, one as ins and the other as lab.

.mat data set description

Dataset name GLIOMA GIOMA contains two matrices, one is the instance matrix or decision variable (ins), and the other is the label matrix (lab)The Ins matrix size is 50*4434, which means that the GLIOMA data set has 50 instances (samples) and 4434 features. Each of these 50 instances (samples) has a corresponding label lab, and the label is the category. Open the Ins matrix. There are 50 rows indicating that there are 50 instances (samples), and 4434 columns indicating that there are 4434 features (too many to display). Any value (scalar) here is called a feature value, any column is a feature vector (column vector), and any row is an instance vector (row vector).Open lab matrixThere are 50 labels, and the labels are categories (for example, 1 represents young children, 2 represents youth, and so on). You can see that this is a data set with 4 categories.

The data set is divided into training set and test set code

10% off divided

illustrate:
  • ①In the code directory, create a new folder dataset and put the .mat data set into it
  • ②The input dataName is a string. If the data set name is GLIOMA.mat, then the input dataName is ‘GLIOMA’ (do not add .mat)
  • ③iter is the number of times the algorithm is run. The first time the algorithm is run, the first randomly divided value is called, and the i-th time the i-th run is called, the i-th randomly divided value is called.
  • ④Indices is the index for randomly dividing the data set, iter = 1 is generated, and the subsequent files are read from the first generated file.
function [train_F,train_L,test_F,test_L] = DIVDATA10fold(dataName,iter)
file = ['dataset/',dataName,'.mat'];
load(file)
fold = 10
dataMat=ins;
len=size(dataMat,1);
maxV = max(dataMat);
minV = min(dataMat);
range = maxV-minV;
newdataMat = (dataMat-repmat(minV,[len,1]))./(repmat(range,[len,1]));
if mod(iter, 10) == 1
    Indices   =  crossvalind('Kfold', length(lab), fold);
    save Indices;
else
    load Indices;
end
site = find(Indices==iter);
test_F = newdataMat(site,:);
test_L = lab(site);
site2 = find(Indices~=iter);
train_F = newdataMat(site2,:);
train_L =lab(site2);
end

Thirty-seven division

The three-seven division is similar to the ten-fold division, except that there are fewer input parameters and global variables.
function [train_F,train_L,test_F,test_L] = DIVDATA37(dataName)
file = ['dataset/',dataName,'.mat'];
load(file)

dataMat=ins;
len=size(dataMat,1);
%归一化
maxV = max(dataMat);
minV = min(dataMat);
range = maxV-minV;
newdataMat = (dataMat-repmat(minV,[len,1]))./(repmat(range,[len,1]));

Indices   =  crossvalind('Kfold', length(lab), 10);
site = find(Indices==1|Indices==2|Indices==3);
test_F = newdataMat(site,:);
test_L = lab(site);
site2 = find(Indices~=1&Indices~=2&Indices~=3);
train_F = newdataMat(site2,:);
train_L =lab(site2);
end
