A:Scholarship

Global question number 7908, number of submissions 207, number of attempts 40, number of passes 32
Total time limit:
1000ms
Memory limit:
65536kB
describe
A primary school recently received a donation and plans to use part of it to provide scholarships to the top five students with outstanding academic performance. At the end of the semester, each student has scores in three subjects: Chinese, mathematics, and English. First, sort by total score from high to low. If the total scores of two students are the same, then sort by Chinese scores from high to low. If the total scores and Chinese scores of two students are the same, then the student with the smaller student number will be ranked first. In this way, the ranking of each student is uniquely determined. Task: First calculate the total score based on the input scores of the three courses, then sort according to the above rules, and finally output the student IDs and total scores of the top five students in ranking order. Note that among the top 5 students, each person’s scholarship is different, so you must sort strictly according to the above rules. For example, in a correct answer, if the output data of the first two lines (each line outputs two numbers: student number, total score) is: 7 279 5 279 The meaning of these two rows of data is: the student numbers of the two students with the highest total scores are No. 7 and No. 5. The total score of these two students is 279 (the total score is equal to the sum of the input scores of Chinese, mathematics, and English), but the student with student number 7 has a higher Chinese score. If your output data for the top two are: 5 279 7 279 Then it will be handled as an output error and no points will be scored.
enter
Contains n+1 lines: The first line is a positive integer n, indicating the number of students in the school participating in the selection. Lines 2 to n+1, each line has 3 numbers separated by spaces, each number is between 0 and 100. The three numbers in row j represent the Chinese, mathematics, and English scores of the student with student number j-1. The student number of each student is numbered from 1 to n according to the input sequence (exactly the row number of the input data minus 1). The data given are correct and do not need to be verified. 50% of the data meets: the total scores of each student are different; 100% of the data satisfies: 6<=n<=300.
output
There are 5 lines in total, each line is two positive integers separated by spaces, which in turn represent the student IDs and total scores of the top 5 students.

C++ implementation
#include<iostream>
#include<vector>
#include<algorithm>
using namespace std;

struct node{
    int id;
    int chinese;
    int math;
    int eng;
    int all;
};

bool cmp(node a, node b){
    if(a.all != b.all)return a.all > b.all;
    else return a.chinese > b. chinese;
}

int main(){
    int n;
    cin>>n;
    vector<node> pool(n);
    for(int i = 0; i < n; i++){
        pool[i].id = i + 1;
        cin>>pool[i].chinese>>pool[i].math>>pool[i].eng;
        pool[i].all = pool[i].chinese + pool[i].math + pool[i].eng;
    }
    sort(pool.begin(), pool.end(), cmp);
    for(int i = 0; i < 5; i++){
        cout<<pool[i].id<<" "<<pool[i].all<<endl;
    }
    return 0;
}

You are now invited to solve a simple higher-order congruence equation, xk=1(mod n). (Note: xk=1(mod n) means xkThe remainder after dividing by n is 1) Data range: n and k are integers, and 0 < k < n < 1000 Enter two integers, k and n. Output all integer solutions x of the equation that satisfy 0 < x < n from small to large, one per line. Sample input
3 7
Sample output
1
2
4
Tip: Pay attention to the possible range of intermediate calculation results. (a * b) % p = ((a % p) * (b % p)) % p Sample input: 398 702 Sample output 1 53 649 701 C++ implementation
#include<stdio.h>


int res(int x, int  k, int n){
    if(k == 1)return x % n;
    else return ((x % n) * res(x, k - 1, n)) % n;
}

int main(){
    int k, n, i;
    scanf("%d%d", &k, &n);

    for(i = 1; i < n; i++){
        if(res(i, k, n) == 1)printf("%d\n", i);
    }
    return 0;
}

H: Topological sorting

Global question number 4865, number of submissions: 30, number of attempts: 17, number of passes: 16
Total time limit:
10000ms
Memory limit:
1000kB
describe
Given the structure of a graph, output its topological sorting sequence. It is required that under the same conditions, the vertices with smaller numbers come first.
enter
Several rows of integers. The first row has 2 numbers, which are the number of vertices v and the number of arcs a. The next row has a row. Each row has 2 numbers, which are the two vertex numbers associated with the arc.
output
A sequence of several space-separated vertices (in lowercase letters)
Sample input
6 8
1 2
1 3
1 4
3 2
3 5
4 5
6 4
6 5
Sample output
v1 v3 v2 v6 v4 v5
C++ implementation
#include<iostream>
#include<vector>
#include<queue>
#include<cstdio>
using namespace std;


int main(){
    int n, m;
    cin>>n>>m;
    vector<int> vec[n + 1];
    vector<int> cnt(n + 1);
    vector<bool> visit(n + 1);
    int a, b;
    for(int i = 0; i < m; i++){
        cin>>a>>b;
        vec[a].push_back(b);
        cnt[b]++;
    }
    queue<int> q;
    for(int i = 1; i <= n; i++){
        if(cnt[i] == 0){
            q.push(i);
            visit[i] = true;
            break;
        }
    }
    int flag = 0;
    while(!q.empty()){
        int cur = q.front();
        q.pop();
        for(int i = 0; i < vec[cur].size(); i++)cnt[vec[cur][i]]--;
        for(int i = 1; i <= n; i++){
            if(cnt[i] == 0 && visit[i] == false){
                q.push(i);
                visit[i] = true;
                break;
            }
        }
        printf("v%d%s", cur, ++flag == n ? "\n" : " ");
    }
    return 0;
}
