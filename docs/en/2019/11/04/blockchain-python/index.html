Simple blockchain Python implementation

What is blockchain

Blockchain is a data structure and a distributed database.
From a technical point of view:A block is a data structure that records transactions and reflects the flow of funds for a transaction. The blocks of transactions that have been reached in the system are connected together to form a main chain, and all nodes participating in the calculation record the main chain or part of the main chain.

composition

block header

The three main sets of metadata contained in the block header are: (1) data used to connect the previous block and indexed from the hash value of the parent block; (2) mining difficulty, timestamp, and Nonce (random number, a counter used in the proof-of-work algorithm, which can also be understood as a value that records the answer to the math problem related to decrypting the block); (3) Merkle tree root data that can summarize and quickly summarize and verify all transaction data in the block. Of course, the block header not only contains these metadata, but also includes other metadata such as version number, difficulty value, etc.
From this structure, most functions of the blockchain are implemented by the block header.

block body

The transaction information recorded in the block body is the task data carried by the block, including the private keys of both parties to the transaction, the number of transactions, the digital signature of electronic currency, etc.

chain

The Bitcoin system creates a block approximately every 10 minutes. This block contains all transactions that occurred on the entire network during this period. Each block saves the hash value of the previous block, so that each block can find its previous block, thus connecting these blocks to form a chain structure.

block field

  • Difficulty The difficulty level of this block
  • ExtraData is additional data related to this block. It was used more in the early days of the blockchain, but is rarely used now.
  • gasLimit is the maximum gas allowed to be used in the current block. Gas represents a calculation amount and the computing power unit used.
  • gasUsed The gas used by the current block
  • Hash The hash value of the current block. If the block has not been confirmed, this field will be null
  • LogsBloom is a Bloom filter (data structure) composed of log information. If the block is not confirmed, the value is null.
  • Miner The miner who obtained the accounting rights of the block
  • mixHash A Hash value that, when combined with a nonce, proves that sufficient calculations have been performed on this block
  • nonce A Hash value that, when combined with mixHash, proves that sufficient calculations have been performed on this block
  • Number The count of the current block (the block sequence number of the genesis block is 0, and for each subsequent block, the block sequence number increases by 1)
  • parentHash The Hash value of the parent block header (this is also what makes the block a blockchain)
  • receiptsRoot contains the hash value of the root node of the tree containing all transaction receipts listed in this block
  • Sha3Uncles Hash value of data block
  • Size The byte size of the current block
  • stateRoot The root hash of the block state tree
  • Timestamp unix timestamp when the block was packaged
  • totalDifficulty The total difficulty from the blockchain to the current block
  • Transactions transaction object
  • transactionsRoot The root hash of the transaction tree of the block
  • Uncles uncle hash array tree block details, please see https://blog.csdn.net/weixin_42874184/article/details/81735695

Python implementation

Implemented proof of work, difficulty calculation, hashing part
import hashlib
import datetime
import random

class Block:
    def __init__(self, index, transaction, pre_hash, difficulty, nonce):
        self.index = index
        self._timestamp = datetime.datetime.now()
        self._transaction = transaction
        self._pre_hash = pre_hash
        self._nonce = nonce
        self._difficulty = difficulty
        self.hash = self.gen_hash()

    @property
    def get_hash(self):
        return self.hash

    @property
    def get_index(self):
        return self.index

    @property
    def get_transaction(self):
        return self._transaction

    @property
    def get_difficulty(self):
        return self._difficulty

    @property
    def get_timestamp(self):
        return self._timestamp

    @property
    def get_pre_hash(self):
        return self._pre_hash

    @property
    def get_nonce(self):
        return self._nonce

    def gen_hash(self):
        sha = hashlib.sha256()
        data = str(self.index) + str(self._timestamp) + str(self._transaction) + str(self._pre_hash) + str(self._nonce) + str(self._difficulty)
        sha.update(data.encode("utf8"))
        return sha.hexdigest()


class Blockchain:
    def __init__(self, max_blocks):
        self._chain_list = [Block(0, "The First Block", '0', '1', random.randrange(0, 99999))]
        self._previous_block = self._chain_list[0]
        self._current_block = 1
        self.max_blocks = max_blocks

    @staticmethod
    def create_next_block(self):
        last_block = self._chain_list[self._current_block - 1]
        last_difficulty = last_block.get_difficulty
        this_index = last_block.index + 1
        this_transaction = "New Block " + str(this_index) + " has been generated successfully."
        nonce = random.randrange(0, 99999)
        this_difficulty = self.proof_of_work(nonce)
        pre_hash = last_block.hash
        self._chain_list.append(Block(this_index, this_transaction, pre_hash, this_difficulty, nonce))
        self._current_block += 1

    def last_block(self):
        return self._chain_list[-1]

    def proof_of_work(self, nonce):
        proof = 0
        while self.valid_proof(self, proof, nonce) is False:
            proof += 1
        return proof

    @staticmethod
    def valid_proof(self, proof, nonce):
        guess = f'{proof}{nonce}'.encode()
        guess_hash = hashlib.sha256(guess).hexdigest()
        pattern = "0000"
        return guess_hash[:len(pattern)] == pattern

    def run(self):
        for i in range(self.max_blocks):
            self.create_next_block(self)

    def show_block(self, index):
        print("----------------------------------------")
        print("ID: ", self._chain_list[index].get_index)
        print("Time", self._chain_list[index].get_transaction)
        print("Cur Hash", self._chain_list[index].get_hash)
        print("Pre Hash", self._chain_list[index].get_pre_hash)
        print("Difficulty", self._chain_list[index].get_difficulty)
        print("Trans", self._chain_list[index].get_transaction)
        print("Nonce", self._chain_list[index].get_nonce)
        print("----------------------------------------")


myBlockChain = Blockchain(100)
myBlockChain.run()
for i in range(myBlockChain.max_blocks):
    myBlockChain.show_block(i)
