C++ finds the intersection and union of sets

Programming LanguagesC & C++TechnologyC++PAT
It is very convenient to find the intersection and union of sets in Python, but it is still difficult in C++. There are functions in set such as set_intersection (to take the intersection of sets), set_union (to take the union of sets), set_difference (to take the difference of sets), and set_symmetric_difference (to take the symmetric difference of sets). There are 5 parameters in total. The first 4 parameters are the same sets a.begin(), end and b.begin, end. The fifth parameter is the saved location or direct output.
  • Save to set or vector: Save the combined result of sets A and B into set C
set_union(A.begin(),A.end(),B.begin(),B.end(),inserter(C1 , C1.begin()));
  • Output: Directly output the result of combining A and B. (cout, " ") Inside the double quotes is the symbol or space you want to use to separate the elements of the set.
set_union(A.begin(),A.end(),B.begin(),B.end(),ostream_iterator<int>(cout," “));
Note that some compilers require importing
#include<bits/stdc++.h>
to run, not just <set>

Example question PAT A1063 Set Similarity (25 points)

Given two sets of integers, the similarity of the sets is defined to be N​c​​/N​t​​×100%, where N​c​​ is the number of distinct common numbers shared by the two sets, and N​t​​ is the total number of distinct numbers in the two sets. Your job is to calculate the similarity of any given pair of sets.

Input Specification:

Each input file contains one test case. Each case first gives a positive integer N (≤50) which is the total number of sets. Then N lines follow, each gives a set with a positive M (≤10​4​​) and followed by M integers in the range [0,10​9​​]. After the input of sets, a positive integer K (≤2000) is given, followed by K lines of queries. Each query gives a pair of set numbers (the sets are numbered from 1 to N). All the numbers in a line are separated by a space.

Output Specification:

For each query, print in one line the similarity of the sets, in the percentage form accurate up to 1 decimal place.

Sample Input:

3
3 99 87 101
4 87 101 5 87
7 99 101 18 5 135 18 99
2
1 2
1 3

Sample Output:

50.0%
33.3%

#include<bits/stdc++.h>

        cin>>t;
        while(t--){
            cin>>x;
            pool[i].insert(x);
        }
    }
    int check;
    cin>>check;
    while(check--){
        int a, b;
        cin>>a>>b;
        vector<int> ANS1;
        vector<int> ANS2;
        set_union(pool[a-1].begin(),pool[a-1].end(),pool[b-1].begin(),pool[b-1].end(), inserter(ANS1,ANS1.begin()));
        set_intersection(pool[a-1].begin(),pool[a-1].end(),pool[b-1].begin(),pool[b-1].end(), inserter(ANS2,ANS2.begin()));
        printf("%.1f%%\n", ANS2.size()*1.0/ANS1.size()*100);
    }
    return 0;
}
