LaTeX scientific research literature citation compilation and style adjustment

Technology其它LaTeX

Summary

Citing literature is an important step in standard writing. The loss of literature citations directly leads to the suspicion of plagiarism in the work. Next, the author gives the steps to cite the literature in Google Scholar and Baidu Scholar, and generateBibTeXTo the specific operation method in LaTeX.

Literature search

Log in to Google Scholar or Baidu Scholar and select the corresponding article, as shown in the figure below: Google QuotesBaidu citationClick to select the format you need (It is very important. It depends on the requirements of the journal you are submitting to. If it is not standardized, it may be rejected by the editorial department for reformat.) In this way, you can add citations after the article you need (just copy and paste)

BibTex import bib file

Of course, many times we will use the more professional article writing tool LaTeX to write papers. At this time, automatic management and numbering of references becomes very important, because when revising, we may add a reference or delete a reference.First select BibTex (lower left corner of the picture below) BibTex style is as follows (like json)When importing LaTeX, first create a .bib fileWrite the references that need to be cited in .bib format, such as referencet.bib file, as follows:Create a new txt file and change the suffix to .bibThe content is as shown in the picture, which contains two papers.

Write LaTeX code, automatic numbering and management

The part quoted by the bib file: (This part is best placed before \end)
\bibliographystyle{splncs04} %这是一个样式文件
\bibliography{reference} %其中reference为reference.bib文件。
The part written in the document: (just use \cite{})
This is a test\cite{Akundi2018Information} of reference. 
Machine learning and artificial\cite{Dong2018Packet} intelligence are worse than mathematics.
Note that the \cite{} braces should be filled with the content after @ in BibTex 
\documentclass{article}  % 选择模版，这里是使用Latex自带的article模版
    \author{Reacubeth}
    \title{First Test}
\begin{document} 
\maketitle

This is a test\cite{Akundi2018Information} of reference. 
Machine learning and artificial\cite{Dong2018Packet} intelligence are worse than mathematics.

\bibliographystyle{splncs04} %这是一个样式文件
\bibliography{reference} %其中reference为reference.bib文件

\end{document}

style adjustment

Style files are generally of .cls or .bst type. Specific style files can be obtained from the journal submission official website. The above example references are in splncs04 style, which is suitable for conference papers including LNCS. If you need to adjust it to IEEEtran style, replace it in the first line below.
\bibliographystyle{IEEEtran} %换风格
\bibliography{reference} %其中reference为reference.bib文件。
Note that if other styles of pdf have been generated previously, it will not work when compiling directly after changing the code. Because of the cache files .aux and .bbl, you need to delete all previous cache files and follow the above steps again.As a result, you can see that the style of the reference has been adjusted.
