Overview of HyperVolume multi-objective evaluation indicators

TechnologyMachine LearningEvolutionary Algorithms多目标优化算法特征选择

propose

The Hypervolume index evaluation method was first proposed by Zitzler et al. It represents the volume of the hypercube surrounded by the individuals in the solution set and the reference point in the target space.

Evaluation criteria

The Hypervolume index evaluation method is a Pareto-compliant evaluation method, that is to say, if a solution set S is better than another solution set S’, then the Hypervolume index of the solution set S will also be greater than the Hypervolume index of the solution set S’.

Evaluation object

HyperVolume can simultaneously evaluate the convergence and distribution of solution sets.

reference point

The accuracy of calculating the Hypervolume index depends on the selection of reference points. That is, when evaluating the same solution set, selecting different reference points will result in different calculation results.

Some insights in feature selection

In the target domain of feature selection (number of features and error rate), if the true frontier is not known, the reference point (1, 1) can be selected. If the true frontier is known, we can select the point closest to the origin, or select the point with the lowest error rate as the reference point.
