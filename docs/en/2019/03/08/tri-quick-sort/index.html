Implementation and application of three-way fast queuing in C++

高级语言C&C++数据结构技术域C++数据结构C语言LeetCode
Three-way quick sort is an upgraded version of the quick sort algorithm, used to process arrays with a large amount of duplicate data. The main idea is to select a key, throw it to the left if it is smaller than the key, throw it to the right if it is larger than the key, and implement it recursively. Specific operation process reference: https://blog.csdn.net/k_koris/article/details/80585979C++ code: 
#include<iostream>

using namespace std;

void swap(int &a, int &b){
    int t = a;
    a = b;
    b = t;
}

void triSort(int *a, int low, int high){
    if(low >= high)return;
    int key = a[low];
    int i = low, j = low;
    int k = high;
    while( i <= k){
        if(a[i] < key)swap(a[i++], a[j++]);
        else if(a[i] > key)swap(a[i], a[k--]);
        else i++;
    }
    triSort(a, low, j);
    triSort(a, k + 1, high);
}


int main(){
    int n[12] = {5, 9, 0, 1, 6, 3, 8, 7, 2, 4, 4, 4};
    triSort(n, 0, 11);
    for(int i = 0; i < 12; i++){
        cout<<n[i]<<" ";
    }
    cout<<endl;
    return 0;
}

Given a color containing red, white and blue, a total ofn an array of elements,In situSort them so that elements of the same color are adjacent and in the order red, white, blue. In this question, we use the integers 0, 1, and 2 to represent red, white, and blue respectively.Notice:This problem cannot be solved using the sorting functions in the code base.Example: 
enter: [2,0,2,1,1,0]
Output: [0,0,1,1,2,2]
Advanced:
  • An intuitive solution is to use a counting sorted two-pass scan algorithm. First, iteratively calculate the number of elements 0, 1, and 2, and then rewrite the current array in order of 0, 1, and 2.
  • Can you come up with a one-pass scan algorithm using only constant space?

class Solution {
public:
    void swapx(int &a, int &b){
        int t = a;
        a = b;
        b = t;
    }
    void sortColors(vector<int>& nums) {
        int i = 0, j = 0, k = nums.size()-1;
        int key = 1;
        while(i <= k){
            if(nums[i] < key)swapx(nums[i++], nums[j++]);
            else if(nums[i] > key)swapx(nums[i], nums[k--]);
            else i++;
        }
    }
};
