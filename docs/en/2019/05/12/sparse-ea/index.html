Introduction to multi-objective evolutionary algorithms based on sparse large-scale matrices

TechnologyMachine LearningEvolutionary Algorithms多目标优化算法特征选择神经网络

Introduction

You can see that the featured image in this article is an extremely sparsely connected neural network, which is obtained by the algorithm SparseEA we are about to introduce in the paper. This paper was published by BIMK's Tian Tian, ​​Zhang Xingyi and others in the journal IEEE Transactions on Evolutionary Computation (SCI Area 1) on April 2019.
The paper proposes a multi-objective algorithm to solve the problem of large-scale sparsity, which exists in many fields: machine learning, data mining, and neural networks.
The author mainly discusses four specific issues
  • ①Feature selection
  • ②Pattern mining
  • ③Key node detection
  • ④Neural network training
Although the above four problems exist in different fields, they are all multi-objective problems, and the solution sets of their pareto surfaces are all sparse. For example, for large-scale feature selection problems, less than 100 features can be selected out of 10,000 dimensions, and the compression rate reaches 99%, which is a typical sparse problem.

specific questions

Algorithmic Contribution

  • ① Designed a new population initialization strategy (according to the sparse large-scale characteristics, a good frontier can be obtained)
  • ② Designed a new genetic operator based on the sparsity of pareto solution set

Specific algorithm

Algorithm framework

A framework similar to NSGA2

Initialization strategy

In order to integrate the two encodings, two vectors need to be introduced, one is the decision variable vector dec (actually the evolutionary solution, for 01 encoding, it can be set to all 1), and the other is the mask vector mask (actually a 01 vector, used to record the quality of each dimension, and the good ones are set to 1). The final decision variable is the inner product of the two.
The result after initialization:It can be seen that through this initialization strategy, a good distribution that approximates the pareto surface is obtained.

crossover mutation operator

This cross mutation is the core of the algorithm. It flips one element in the 0 element or one element in the non-0 element with the same probability each time in the binary vector mask. The flipping is performed based on the fitness value of the decision variable. Therefore, the generated children will not have the same number of 0s and 1s, and the sparsity of the children can be maintained.
The result after using crossover mutation:It can be seen that through this strategy, the sparsity is improved, and fewer and fewer dimensions are set to 1.
