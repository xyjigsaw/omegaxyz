A: Simplest true fraction

Global question number 3526, number of submissions 67, number of attempts 39, number of passes 38
Total time limit:
1000ms
Memory limit:
65536kB
describe
Given n positive integers, take any two numbers as the numerator and denominator to form the simplest true fraction. Program to find how many such combinations there are.
enter
The first line is a positive integer n (n<=600). The second line contains n different integers, separated by a single space between two adjacent integers. The integer is greater than 1 and less than or equal to 1000.
output
An integer, that is, the number of the simplest proper fraction combinations.
Sample input
7
3 5 7 9 11 13 15
Sample output
17
C++ implementation
#include<iostream>
#include<vector>
using namespace std;

int gcd(int x, int y){
    if(y == 0)return x;
    else return gcd(y, x % y);
}

int main(){
    int n;
    cin>>n;
    vector<int> all(n);
    for(int i = 0; i < n; i++){
        cin>>all[i];
    }
    int ans = 0;
    for(int i = 0; i < n - 1; i++){
        for(int j = i + 1; j < n; j++){
            if(gcd(all[i], all[j]) == 1)ans++;
        }
    }
    cout<<ans<<endl;
    return 0;
}

B: n-gram cross frequency statistics

Global question number 7604, number of submissions: 128, number of attempts: 34, number of passes: 31
Total time limit:
1000ms
Memory limit:
65536kB
describe
The n-gram frequency statistical method is commonly used in text analysis, that is, counting the frequency of occurrence of n adjacent units (such as words, Chinese characters, or characters) in the entire text. Assume there is a string. Please use the n-gram method to count the frequency of each substring of length n in character units, and output the highest frequency and the most frequent substring. The given string contains only uppercase and lowercase letters, is no longer than 500 characters, and 1 < n < 5. If there are multiple substrings with the highest frequency, they will be output in order according to the order in which they first appear in the sequence, one for each line. If the highest frequency is not greater than 1, NO will be output.
enter
The first line is n; The second line is a string.
output
Output the highest frequency and all substrings with the highest frequency. If the highest frequency is not greater than 1, only one line of NO is output.
Sample input
3
abcdefabcd
Sample output
2
abc
bcd
hint
In the example, all 3-grams are: abc, bcd, cde, def, efa, fab, abc, bcd. The last cd is not enough to form a 3-gram, so it will not be considered. In this way, abc and bcd both appear twice, and the rest only appear once.
C++ implementation
#include<iostream>
#include<vector>
#include<unordered_map>
#include<algorithm>
using namespace std;

struct node{
    string s;
    int pos;
};

bool cmp(node a, node b){return a.pos < b.pos;}

int main(){
    int n, max = 0;
    string str;
    cin>>n>>str;
    unordered_map<string, int> cnt;
    for(int i = 0; i <= str.length() - n; i++){
        cnt[str.substr(i, n)]++;
    }
    for(auto it = cnt.begin(); it != cnt.end(); it++){
        if(max < it->second)max = it->second;
    }
    cout<<max<<endl;
    vector<node> ans;
    for(auto it = cnt.begin(); it != cnt.end(); it++){
        if(max == it->second){
            node temp;
            temp.s = it->first;
            temp.pos = str.find(it->first);
            ans.push_back(temp);
        }
    }
    sort(ans.begin(), ans.end(), cmp);
    for(int i = 0; i < ans.size(); i++){
        cout<<ans[i].s<<endl;
    }
    
    return 0;
}

#include<iostream>
#include<vector>

    string one, two, three, four, all = "";

    getline(cin, two);
    getline(cin, three);
    getline(cin, four);

    all += two;
    all += three;
    all += four;

        if(all[i] >= 'A' && all[i] <= 'Z'){
            cnt[all[i]]++;
        }
    }

    for(char c = 'A'; c <= 'Z'; c++){
        if(maxc < cnt[c])maxc = cnt[c];
    }

        for(char j = 'A'; j <= 'Z'; j++){
            if(cnt[j] == maxc){
                printf("%c%s", '*', j == 'Z' ? "\n" : " ");
                cnt[j]--;
            }
            else printf("%c%s", ' ', j == 'Z' ? "\n" : " ");
        }
        maxc--; 
    }
    for(char c = 'A'; c <= 'Z'; c++){
        printf("%c%s", c, c == 'Z' ? "\n" : " ");
    }
    return 0;

F:Game Prediction

Global question number 325, number of submissions 27, number of attempts 11, number of passes 8
Total time limit:
1000ms
Memory limit:
65536kB
describe
Suppose there are M people, including you, playing a special card game. At the beginning, each player receives N cards. The pip of a card is a positive integer which is at most N*M. And there are no two cards with the same pip. During a round, each player chooses one card to compare with others. The player whose card with the biggest pip wins the round, and then the next round begins. After N rounds, when all the cards of each player have been chosen, the player who has won the most rounds is the winner of the game. Given your cards received at the beginning, write a program to tell the maximal number of rounds that you may at least win during the whole game.
enter
The input consists of several test cases. The first line of each case contains two integers m (2 <= m <= 20) and n (1<= n <= 50), representing the number of players and the number of cards each player receives at the beginning of the game, respectively. This followed by a line with n positive integers, representing the pips of cards you received at the beginning. Then a blank line follows to separate the cases. The input is terminated by a line with two zeros.
output
For each test case, output a line consisting of the test case number followed by the number of rounds you will at least win during the game.
Sample input
2 5
1 7 2 10 9

6 11
62 63 54 66 65 61 57 56 50 53 48

0 0
Sample output
Case 1: 2
Case 2: 4
C++ implementation
#include<iostream>
#include<vector>
using namespace std;

bool cards[1001];

//贪心策略

int main(){
    int m, n, t = 0;
    while(cin>>m>>n){
        t++;
        if(m == 0 && n == 0)break;
        int ans = 0, others_large = 0;
        fill(cards, cards + 1000, false);
        for(int i = 1; i <= n; i++){
            int temp;
            cin>>temp;
            cards[temp] = true;
        }
        for(int i = n * m; i >= 1; i--){
            if(cards[i]){
                if(others_large == 0)ans++;
                else others_large--;
            }else{
                others_large++;
            }
        }
        cout<<"Case "<<t<<": "<<ans<<endl;
    }
    return 0;
}

You are assigned to design network connections between certain points in a wide area. You are given a set of points in the area, and a set of possible routes for the cables that may connect pairs of points. For each possible route between two points, you are given the length of the cable that is needed to connect the points over that route. Note that there may exist many possible routes between two given points. It is assumed that the given possible routes connect (directly or indirectly) each two points in the area. Your task is to design the network for the area, so that there is a connection (direct or indirect) between every two points (i.e., all the points are interconnected, but not necessarily by a direct cable), and that the total length of the used cable is minimal.
enter

The first line of the set contains two integers: the first defines the number P of the given points, and the second the number R of given routes between the points. The following R lines define the given routes between the points, each giving three integer numbers: the first two numbers identify the points, and the third gives the length of the route. The numbers are separated with white spaces. A data set giving only one number P=0 denotes the end of the input. The data sets are separated with an empty line. The maximal number of points is 50. The routes between two points i and j may be given as i j or as j i.

For each data set, print one number on a separate line that gives the total length of the cable used for the entire designed network.

C++ implementation
#include<iostream>
#include<algorithm>
#include<vector>
using namespace std;

//Kruskal最小生成树模板题
int n, m, father[100000];

struct node{
    int src, dst;
    int val;
};

bool cmp(node a, node b){return a.val < b.val;}

int findFather(int a){
    int t = a;
    while(a != father[a])a = father[a]; //找到根节点
    return a;
}

void init(){
    for(int i = 1; i <= n; i++)father[i] = i;
}

int unionSet(int x, int y){ //和并查集一样，如果将根合并
    int fx = findFather(x);
    int fy = findFather(y);
    if(fx != fy){
        father[fx] = fy;
        return  1;
    }return 0;
}

int main(){
    while(cin>>n){
        if(n == 0)break;
        init(); //初始化不能丢
        cin>>m;
        int ans = 0;
        vector<node> pool(m);
        for(int i = 0; i < m; i++){
            cin>>pool[i].src>>pool[i].dst>>pool[i].val;
        }
        
        sort(pool.begin(), pool.end(), cmp);
        int k = 0, i = 0;
        while(k < n - 1){ 
            if(unionSet(pool[i].src, pool[i].dst)){ //如果两个节点的根不一样则合并，本质上将两个预给定的边选择出来
                //也不会成环
                k++;
                ans += pool[i].val;
            }
            i++;
        }
        cout<<ans<<endl;
    }
    return 0;
}
