C++ reading and writing files (fstream, etc.)

Programming LanguagesC & C++TechnologyC++C
July 2019 has been very busy, so this is probably the first post of the month. A computer test in a university needs to read data from a file and write the data to the file. To complete this operation, you need to use the fstream module. There is a lot of information on the Internet, but it is a mess and cannot meet some simple needs. The following is the C++ code for reading from the file (reading all characters line by line) and writing data to the file (append writing).

Read from file (read all characters line by line)

ifstream type adoptedBe careful to use getline
#include<iostream>
#include<fstream>

using namespace std;

int main(){
    ifstream inFile; //读取文件使用ifstream
    inFile.open("/Users/reacubeth/Desktop/test.txt");
    string str;
    while(getline(inFile, str)){
        cout<<str<<endl;
    }
    inFile.close();
    return 0;
}

Write to file (append)

The ofstream type is used, and when reading the file, you must select ios::app to open the file so that you can append. 
#include<iostream>
#include<fstream>

using namespace std;

int main(){
    ofstream outFile;
    outFile.open("/Users/reacubeth/Desktop/test.txt", ios::app);
    string str = "cdscs\n dvfsvcdfsvdsfv❤️dfvdfsvdf\n";
    if(!outFile.fail()){
        outFile<<str;
    }
    outFile.close();
    return 0;
}
Don’t forget to close the file when you’re done!
