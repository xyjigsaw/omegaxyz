Conversion C++ implementation of binary tree traversal

Programming LanguagesC & C++Data StructuresTechnologyC++Data Structures
There are three types of binary tree traversal:
  • Preorder traversal: The traversal order rule is [around the root]
  • In-order traversal: The traversal order rule is [left root right]
  • Postorder traversal: The traversal order rule is [left and right roots]
What is [root left and right] is to traverse the root first, then traverse the left node, and finally traverse the right node. For example:
  • Preorder traversal: ABCDEFGHK
  • In-order traversal: BDCAEHGKF
  • Postorder traversal: DCBHKGFEA
Please note during the conversion process:
  • Preorder traverse the root node at the first item
  • In-order traversal with the root node in the middle
  • Post-order traversal with the root node at the end
Therefore, it is divided into two parts according to the in-order root node during the recursive process.C++ code: 
#include <iostream>
using namespace std;

void ToPost(char *inOrder, char *preOrder, int length){
    if(length<=0)return;
    int root;
    for(root=0;root<length;root++){
        if(inOrder[root]==preOrder[0])break;
    }
    ToPost(inOrder,preOrder+1,root);
    ToPost(inOrder+root+1,preOrder+root+1,length-root-1);
    cout<<preOrder[0];
}

void ToPre(char *inOrder, char *postOrder, int length){
    if(length<=0)return;
    int root;
    for(root=0;root<length;root++){
        if(inOrder[root]==postOrder[length-1])break;
    }
    cout<<postOrder[length-1];
    ToPre(inOrder,postOrder,root);
    ToPre(inOrder+root+1,postOrder+root,length-root-1);
}

int main()
{
    char pre[]="ABDECFG";
    char in[]="DBEAFCG";
    char post[]="DEBFGCA";
    cout<<"原始序列:\n";
    cout<<"Pre:"<<pre<<endl
        <<"In:"<<in<<endl
        <<"Post:"<<post<<endl;

    cout<<"后序遍历:\n";
    ToPost(in,pre,8);
    cout<<endl;
    cout<<"前序遍历:\n";
    ToPre(in,post,8);
    cout<<endl;
    return 0;
}
