C language command line Sudoku game

Programming LanguagesC & C++TechnologyC++CC语言游戏HCI

Features and functions

  1. Design a command line interface game. After logging in to the game interface, there should be descriptions of game rules and button descriptions.
  2. There should be a difficulty selection before entering the game: 1 means easy, 2 means medium, and 3 means difficult.
  3. In the main interface of the game, you can input Sudoku values ​​cyclically, play several rounds in a row until you exit, check the results, view the answers, and start a new round.
  4. Information prompt function: there should be prompts when input errors are made, and there should be celebration interfaces when the game fails or wins, etc.

int dfs(int x,int y)
{
    if(!check) return 0;
    if(x==0&&y==9)
    {
     //   print();
        check=false;
        return 0;
    }
    if(a[x][y]==0)
    for(int j=1;j<=9;j++)
    {
    	int i=c[j];
        if(square[ge(x,y)][i]&&lie[y][i]&&hang[x][i])
        {
            square[ge(x,y)][i]=false;
            lie[y][i]=false;
            hang[x][i]=false;
            ans[x][y]=i;

            if(y==1) dfs(x-1,9);else dfs(x,y-1);

            square[ge(x,y)][i]=true;
            lie[y][i]=true;
            hang[x][i]=true;
            if(!check) return 0;
        }
    }
    if(!check) return 0;
    if(a[x][y]!=0) if(y==1) dfs(x-1,9);else dfs(x,y-1);
}

int work(int x){
	if(x==10)
	{
		return 0;
	}
	int i=1;
	int y=rand()%9+1;
	while(!f[y]) y=rand()%9+1;
	f[y]=false;
	c[x]=y;
	work(x+1);
}

Effect

Log in to the game interface

You will see that there are four options: a, b, c, e;If you press "b":If you press "c":If you press "e", you will exit the game; If you press "a", the game difficulty selection interface will appear:

Game difficulty selection

For the picture above, if you press "0", it will return to the previous directory; If you press 1~3, you will enter the main interface of the game. When you press "1", the Sudoku has 20 spaces, when you press "2", the Sudoku has 40 spaces, and when you press "3", the Sudoku has 60 spaces.

Enter the game main interface

If you press "a", this starts to enter the horizontal and vertical coordinates of the number to be filled in and its value. The entered value is marked in red to distinguish the characters that were automatically generated at the beginning; if there are numbers in the original question at the entered position, there will be a prompt of "This has been filled in". If you win, Figure 3.9 will appear and the total answering time will be generated.If you press "c", the Sudoku answer is output according to the DFS algorithm: Note: Human-computer interaction experiment 1
