HTML jumps to a certain position on the page

Programming LanguagesTechnologyWebJavaScript&HtmlWebJavaScriptHTML
When designing a web page, sometimes you need to jump to a certain position on the page. Here are two methods of jumping to a certain position on the page. There is no sliding effect in the following two methodsMethod 1: Control using JavaScriptCode:
<body>
<a href="javascript:void(0);" onclick="javascript:document.getElementsByTagName('BODY')[0].scrollTop=document.getElementsByTagName('BODY')[0].scrollHeight;">到页底</a>
<div style="height:3000px;"></div>
<a href="javascript:void(0);" onclick="javascript:document.getElementsByTagName('BODY')[0].scrollTop=0;">回页顶</a>
</body>
Method 2: Use html point controlNote the ID used here to find
<body>
<a href="../../../../index.html#here" >bottom</a>//跳到id为here的锚点
// 内容
<a id='here' ></a>//锚点ID
</body>
Copy the above code and enter the link below to test the effect: https://omegaxyz.com/onlinetools/web-running/
