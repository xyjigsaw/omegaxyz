HTML send email code (start email client)

Programming LanguagesTechnologyWebJavaScript&HtmlWebHTML
When optimizing your website, you need some social plug-ins. In order to improve website speed, it is better to write it yourself. Below is the code for clicking a link to send an email. The principle is to start the local mail client. Actual test supports windows, mac, Android and iOS. The code is very simple, just one line of a tag. Where mailto is followed by your own email address. subject is the email subject, and after &body is the initialization of the email subject content. In addition, the content in the a tag can be an image or text.Code: 
<a href="mailto:xyjigsaw@aliyun.com?subject=OmegaXYZ的访客&amp;body=访客"><p>测试发送邮件</p></a>
