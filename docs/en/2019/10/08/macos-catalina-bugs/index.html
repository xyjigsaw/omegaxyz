Summary of frequently asked questions about macOS Catalina

技术域操作系统mac
As a pseudo-programmer, let me talk about some software compatibility and precautions of macOS Catalina.Apps that cannot be used in the official version of macOS Catalina
  • Youdao dictionary crashes
  • Brew version is too low loading error
  • The VScode header file has wavy lines and needs to be reconfigured
Reason for error:xcrun: error: invalid active developer path (/Library/Developer/CommandLineTools), missing Update Xcode command line tools Just enter the following statement on the command line
xcode-select --install
After that, the following interface will pop up, just upgrade.
  • The App Store's LaTeX helper Daum Equation Editor cannot be used because it is 32-bit.
  • Synchronous push 32-bit cannot be used
  • Since the Python that comes with the system is 2.7, it will not support future versions (official diss)
  • Console Python3 loads extremely slowly for the first time
  • Console interactive shell will change from bash to zsh
Enter the following statement to upgrade your account to zsh
chsh -s /bin/zsh
zsh is compatible with Bash to a certain extent. However, scripts that comply with the bash specification are not guaranteed to be executed correctly by the Zsh interpreter.
