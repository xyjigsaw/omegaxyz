LeetCode 1027 The longest arithmetic sequence

Given an array of integersA,returnAof the longest arithmetic subsequence inlength. Think back,AA subsequence of is a listA[i_1], A[i_2], ..., A[i_k]in0 <= i_1 < i_2 < ... < i_k <= A.length - 1. and ifB[i+1] - B[i]( 0 <= i < B.length - 1) have the same values, then the sequenceBIt's equal.Example 1: 
输入：[3,6,9,12]
输出：4
解释： 
整个数组是公差为 3 的等差数列。
Example 2: 
输入：[9,4,7,2,10]
输出：3
解释：
最长的等差子序列是 [4,7,10]。
Example 3: 
输入：[20,1,15,3,10,5,8]
输出：4
解释：
最长的等差子序列是 [20,15,10,5]。
hint:
  1. 2 <= A.length <= 2000
  2. 0 <= A[i] <= 10000
analyze:Construct a map type array of dp, where dp[d][x] represents the length of the arithmetic sequence with a tolerance of d before the x-th position. Obviously the initialization answer is 2. First, a normal loop is performed, and then all arrays before the current number i are judged. The tolerance d is first found, and then whether the dp array of tolerance d contains the number j. If it does, then dp[d][i] = dp[d][j] + 1, otherwise it is set to 2.C++ code: 
class Solution {
public:
    int longestArithSeqLength(vector<int>& A) {
        map<int, map<int, int> > dp;  //dp[i][j]代表在第j位之前，公差d的等差数列长度
        int ans = 2;
        for(int i = 1; i < A.size(); i++){
            for(int j = 0; j < i; j++){
                int d = A[i] - A[j];
                if(dp[d].count(j) > 0)dp[d][i] = dp[d][j] + 1; //如果存在i之前相同的d，则加1
                else dp[d][i] = 2;
                ans = max(ans, dp[d][i]);
            }
        }
        return ans;
    }
};

analyze:The dp array dp[i][j] represents the length of the Fibonacci array from i to j. If there is tmp = A[i] + A[j], then there is dp[i][j] = dp[j][M[tmp]] + 1;C++ code: 
class Solution {
public:
    int lenLongestFibSubseq(vector<int>& A) {
	    vector<vector<int>>dp(A.size(), vector<int>(A.size(), 0));
	    map<int, int> M;
	    for (int i = 0; i < A.size(); i++)M[A[i]] = i;
	    int ans = 0;
	    for (int i = A.size() - 1; i >= 0; i--) {
		    for (int j = i + 1; j < A.size(); j++) {
			    dp[i][j] = 2;//默认为2
			    int tmp = A[i] + A[j];
			    if (M.count(tmp)) {
				    dp[i][j] = dp[j][M[tmp]] + 1;
				    if (dp[i][j] > ans) ans = dp[i][j];
			    }
		    }
	    }
	    return ans;
    }
};
