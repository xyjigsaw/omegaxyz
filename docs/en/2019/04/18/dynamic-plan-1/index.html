Introduction

dynamic programmingIt is a branch of operations research and a mathematical method for optimizing the decision process. In the early 1950s, when American mathematician R.E. Bellman and others were studying the optimization problem of multistep decision process, they proposed the famous principle of optimality, which transformed the multi-stage process into a series of single-stage problems, and used the relationship between each stage to solve them one by one. This created a new method for solving such process optimization problems - dynamic programming.

LeetCode 5longest palindrome substring

Given a strings,turn upsThe longest palindrome substring in . you can assumesThe maximum length is 1000.Example 1: 
输入: "babad"
输出: "bab"
注意: "aba" 也是一个有效答案。
Example 2: 
输入: "cbbd"
输出: "bb"
analyze:Set an array of dp boolean type with the size of dp[s,size()][s.size()], where dp[i][j] represents that the string s from i to j is a palindrome. The following situations exist in the recursion:
  • Obviously a single character from self to self is dp[i][i] = true
  • When the * part in the middle of i******j is a palindrome, if s[i]==s[j], then i to j are also palindromes. If the middle part is not a palindrome, then i to j are not palindromes, and dp[i][j] = dp[i+1][j-1]
  • If s[i] == s[i - 1] then dp[i][i-1] = dp[i-1][i] = true
C++ code: 
class Solution {
public:
    string longestPalindrome(string s){
        if (s.size() < 2) return s;
        int dp[10000][10000] = {0};
        int start = 0, len = 1; //start表示开始下标，len表示字串长度
        for(int i = s.size() - 1; i >= 0; --i){
            dp[i][i] = 1;
            if(i >= 1 && s[i] == s[i - 1])dp[i][i-1] = dp[i-1][i] = 1;
            for(int j = i + 1; j < s.size(); ++j){
                if(s[i] == s[j]){
                    dp[i][j] = dp[i+1][j-1];
                    if(dp[i][j] && j - i + 1 > len){// 记录位置
                        start = i;
                        len = j - i + 1;
                    }
                }
            }
        }
        return s.substr(start, len);
    }
};

LeetCode 152 productmaximumsubsequence

Given an array of integersnums, find the continuous subsequence with the largest product in a sequence (the sequence contains at least one number).Example 1: 
输入: [2,3,-2,4]
输出: 6 解释: 子数组 [2,3] 有最大乘积 6。
Example 2: 
输入: [-2,0,-1]
输出: 0
解释: 结果不能为 2, 因为 [-2,-1] 不是子数组。
analyze:
This is different from LIS because it is continuous, so you only need to judge the adjacent ones.
In addition, since the product is negative, it is positive, so two values ​​need to be saved, the current maximum value and the current minimum value.
C++ code: 
class Solution {
public:
    int maxProduct(vector<int>& nums) {
        vector<int> posi(nums.size(), 0);
        vector<int> negt(nums.size(), 0);
        posi[0] = nums[0];
        negt[0] = nums[0];
        int ans = nums[0];
        for(int i = 1; i < nums.size(); i++){
            posi[i] = max(nums[i], max(nums[i] * posi[i-1], nums[i] * negt[i-1]));
            negt[i] = min(nums[i], min(nums[i] * posi[i-1], nums[i] * negt[i-1]));
            ans = max(posi[i], ans);
        }
        return ans;
    }
};

LeetCode 279 perfect square number

Given a positive integern, find several perfect square numbers (such as1, 4, 9, 16, ...) such that their sum equals n. You need to minimize the number of perfect squares that make up the sum.Example 1: 
输入: n = 12 输出: 3 解释: 12 = 4 + 4 + 4.
Example 2: 
输入: n = 13 输出: 2 解释: 13 = 4 + 9.
analyze:This problem is the least coin problem: coins correspond to perfect square numbers. Therefore, we first calculate the perfect square number less than n, and then calculate it through the dynamic programming array dp[n], where dp[i] represents the minimum number of positive integers i. When the current perfect square number x is less than the integer i, we determine the size of dp[i-x]+1 and dp[i]. If dp[i-x] is smaller, replace it. In addition, dp[0]=0.C++ code: 
class Solution {
public:
    int numSquares(int n) {
        vector<int> pool;
        for(int i = 1; i*i <= n; i++){
            pool.push_back(i*i);
        }

        int dp[10000] = {0};
        dp[0] = 0;
        for(int i = 1; i <= n; i++){
            dp[i] = i;
            for(int j = 0; j < pool.size(); j++){
                if(pool[j] <= i){
                    dp[i] = dp[i] < dp[i - pool[j]] + 1 ? dp[i] : dp[i - pool[j]] + 1;
                }
            }
        }
        return dp[n];
    }
};
