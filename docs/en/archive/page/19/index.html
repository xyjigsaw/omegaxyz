2017-11-22

Naive Bayes classifier

Naive Bayes classification is a very simple classification algorithm. It is called Naive Bayes classification because the idea of ​​​​this method is really simple. The basic idea of ​​Naive Bayes is as follows: for a given item to be classified, find the probability of each category appearing under the condition that this item appears. Whichever is the largest is considered to be the category to which the item to be classified belongs. The word "naive" corresponds to the "independence assumption". We will choose the category with the largest conditional probability. This is the ideological basis of naive Bayes.
